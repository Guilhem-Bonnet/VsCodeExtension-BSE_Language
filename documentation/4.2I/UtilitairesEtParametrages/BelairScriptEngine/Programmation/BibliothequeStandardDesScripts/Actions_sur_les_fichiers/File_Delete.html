Bibliothèque standard des scripts

File_Delete

Effacement d'un fichier archivé

paramètres

  • id (int, obligatoire) : numéro du fichier archivé

retour

Réponse obtenue lorsque le fichier demandé a été trouvé et supprimé :

<?xml version="1.0"?>
<response>
    <affectedrows>1</affectedrows>
    <warning></warning>
</response>

Réponse obtenue lorsque le fichier demandé est trouvé dans la base mais n'est pas trouvé dans son répertoire de stockage :

<?xml version="1.0"?>
<response>
    <affectedrows>1</affectedrows>
    <warning>File unknown : D:\BELAIR4\APPCRM\SCANNER\201802\10580516597834.doc</warning>
</response>

Réponse obtenue lorsque le fichier demandé n'est pas trouvé dans la base :

<?xml version="1.0"?>
<response>
    <affectedrows>0</affectedrows>
    <warning>File_id 189 not found in database</warning>
</response>
