File_ConvertCsvToXml

Cette action permet de convertir un fichier CSV en un flux XML

Paramètres

  • inputfilename (string, obligatoire) : fichier CSV à convertir
  • encoding (string, optionnel) : encodage parmi les valeurs ci-dessous (par défaut "ANSI")
    • ANSI
    • UTF8 ou UTF-8
    • UTF16 ou UTF-16
  • ReadcolumnNames (booléen, optionnel) : si True, les noms des colonnes sont extraites de la 1ere ligne d'entête du fichier (suppression des accents, ...)
  • SkipFirstLine (booléen, optionnel) : si True, la 1ere ligne est ignorée (ligne d'en-tête). Forcé à True si ReadcolumnNames = True
  • Columns (string, optionnel) : liste des nom des colonnes à obtenir (noms séparés par ";")
    • Le nombre de noms indiqué (N) peut être inférieur au nombre de colonnes du fichier CSV : dans ce cas, seuls les Neres colonnes sont extraites
  • Rowname (string, optionnel) : nom de la balise pour chaque ligne extraite dans le XML (par défaut "row")
  • outputfileName (string, optionnel) : nom du fichier XML à enregistrer
    • Si le nom ne contient pas de répertoire, le fichier est créé dans le même répertoire que le fichier d'entrée
