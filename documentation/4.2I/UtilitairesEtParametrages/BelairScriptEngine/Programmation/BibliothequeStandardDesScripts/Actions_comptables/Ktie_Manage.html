Ktie_Manage

Création / Modification de comptes de tiers

Cette fonction permet de créer ou modifier un tiers en comptabilité, ainisi que l'ensemble des comptes d'un groupe de tiers, pour ce tiers

Parametres

  • Codek (string, obligatoire) : code comptabilité
  • Tiers (string, obligatoire) : code Tiers
  • Name (string, optionnel) : nom du tiers (par défaut, constante 'TIERS xxxxx')
  • Group (string, optionnel) : nom du groupe de tiers (si non renseigné, seul le record de KTIE sera géré

Opérations

  • Création ou modification dans KTIE
  • Si Group est renseigné, création ou modification des comptes(KCTE) listés dans le groupe de comptes (table KGT)

Resultat

Flux XML indiquant le(s) compte(s) créé(s)

<?xml version="1.0"?>
<kctes codek="1" tiers="AB1234" name="Assurance Blanchi" group="APPORTEURS">
  <kcte>
    <compte>468900</compte>
    <tiers>AB1234</tiers>
    <intitule>Assurance Blanchi commissions </intitule>
  </kcte>
  <kcte>
    <compte>469000</compte>
    <tiers>AB1234</tiers>
    <intitule>Assurance Blanchi commissions </intitule>
  </kcte>
</kctes>
