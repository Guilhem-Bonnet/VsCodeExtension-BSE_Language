Societe_Create

Création d'une ligne dans KCO

Paramètres :

  • data : flux XML suivant
    <data>
<entite>1</entite>
<codek>1</codek>
<libk>Comptabilité Modèle n. 1 Bel Air</libk>
<siren>357465454546541</siren>
<orias>123456789</orias>
<tiers>50</tiers>
</data>

Consignes concernant les paramètres :

  • si codek est indéfini, codek doit être déterminé automatiquement par l'action Societe_ListKcoItems (avec summary=true) pour récupérer le 1er code non utilisé (balise <firstavailable> )
  • l'action doit elle même déterminer le n. de l'entité créée et le retourner dans le résultat de l'action : donné par le SQL 'SELECT MAX from [KCO]'
  • tiers : n. de tiers associé (ou 0)

Consignes concernant le retour :

  • entite, codek et numéro du tiers doivent être retournés en cas de succès
  • le formatage de la réponse en cas de problème doit être respecté (ex. action User_Update)
