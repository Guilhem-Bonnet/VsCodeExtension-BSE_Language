Quittance_Details

Quittance Details

Paramètres

  • quittance (int, obligatoire) : numéro interne quittance
  • details (booléen, optionnel) : si True, toues infos sur quittance, sinon, seulements les id
  • garanties (booléen, optionnel) : si True, détail primes par garantie
  • addinfospqg (booléen, optionnel) : si True, ajout de 4 champs "externes" sur chaque ligne pqg:
    • external_cie_nomcie" (i.e. CIE_B_NOMCIE)
    • external_opt_affiche (i.e. OPT_B_AFFICHE)
    • external_opt_codegar" (i.e. OPT_B_CODEGAR)
    • external_opt_libelle" (i.e. OPT_B_LIBELLEE)
  • intervenants (booléen, optionnel) : si True, détail des intervenants (QINT)
  • addinfosqint (booléen, optionnel) : si True, ajout d'un champ "externe" sur chaque ligne qint
    • external_qint_rsociale" (i.e. TIERS.B_RSOCIALE)

Retour
