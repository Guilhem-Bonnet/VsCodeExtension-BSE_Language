Qbor_Create

Création d'un bordereau à partir d'un modèle

Parametres

  • modele (integer, obligatoire) : n. du bordereau modèle
  • nature (string, optionnel) : nature du bordereau ('C' par défaut)
  • annee (integer, optionnel) : annee du bordereau, par défaut année en cours
  • mois (integer, optionnel) : mois du bordereau, par défaut mois en cours
  • cie (integer, optionnel) : n. de compagnie (par défaut 0 = toutes cies)
  • nom (string, optionnel) : nom du bordereau
  • affecte (booléen, optionnel) : bordereau affecté (False par défaut)
  • XmlResult (booléen, optionnel) : résultat sous forme XML (False par défaut)

Retour

Si XmlResult = True, Flux Xml donnant le numéro du bordereau créé

<qbor_create>
   <numero>nnnn</numero>
</qbor_create>
