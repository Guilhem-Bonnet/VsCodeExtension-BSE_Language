DocHolder_AddFile

Ajout d'un fichier dans le porte-document d'un Tiers/Contrat/Sinistre

Cette fonction ajoute dans un porte-documents un fichier déjà enregistré dans FICH (pas d'upload)

Paramètres

  • TargetKind (string, obligatoire): T,C,S => indique la table concernée (TIERS, CONTRAT, SINISTRE)
  • TargetId (integer, obligatoire): id dans la table concernée
  • FileId (integer, obligatoire) : Id du fichier dans FICH
  • Path (string, optionnel) : si renseigné, contient le chemin vers un sous répertoire 'cible' du porte-documents
  • Title (string, optionnel) : titre du fichier dans le porte-documents

Retour

Flux XML

<DocHolder_AddFile>
    <id>id dans la table xxxx_FILES</id>
</DocHolder_AddFile>
