DocHolder_GetLegacyContent

Cette fonction doit convertir le contenu d'un porte-document (au nouveau format XML ou non) dans l'ancien format.(si le contenu en entrée n'est pas au format XML, on le rend tel quel) Cette action appelle la fonction DocHolder_InternalLegacyContent(Data)

Paramètres

  • Data : string, obligatoire : le contenu à convertir

Retour

Le texte converti au format initial des porte-documents (avant la migration en XML)
