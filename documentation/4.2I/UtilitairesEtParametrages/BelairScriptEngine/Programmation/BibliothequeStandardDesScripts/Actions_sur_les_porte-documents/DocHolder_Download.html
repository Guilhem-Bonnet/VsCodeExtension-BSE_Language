Bibliothèque standard des scripts

DocHolder_FileDownload

Cette fonction télécharge un des fichiers d'un porte-documents

Paramètres

  • TargetKind (string, obligatoire): T,C,S => indique la table concernée (TIERS, CONTRAT, SINISTRE)
  • TargetId (integer, obligatoire): id dans la table concernée
  • FileId(integer, optionnel) : id du fichier dans le porte-documents
  • OU Title (string, optionnel) : Titre du document dans le porte-documents

Retour

Flux XML

<file>
  <id>(FICH.B_ID)</filename>
  <name>(FICH.B_FICHIER)</name>
  <data>(contenu du fichier encodé)</data>
</file>
