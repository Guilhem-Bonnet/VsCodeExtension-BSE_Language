Risk_ListItems

Liste des risques sur un contrat/piece

Parametres

  • contrat (int, obligatoire) : numéro de contrat
  • piece (int, obligatoire) : numéro de pièce
  • all (booléen, default False) : si True, affichage de tous les risques, même ceux sortis du contrat

retour

Flux Xml contenant toutes les informations du risque (table RISA) auxquelles sont ajoutées 3 champs de la table ADH : entrée / sortie / motif sortie
