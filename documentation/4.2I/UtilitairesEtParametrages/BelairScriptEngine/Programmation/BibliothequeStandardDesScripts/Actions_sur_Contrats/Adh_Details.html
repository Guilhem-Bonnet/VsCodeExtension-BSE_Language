Adh_Details

Donne le détail d'une adhésion

Cette action donne les mêmes informations que la balise <adhesion> de l'action Cont_Details
Elle doit donc surtout être utilisée pour les contrats multi-adhésion (flottes, ...)

Parametres:

  • adhesion (int, obligatoire) ; numéro de l'adhésion
  • extensions (booléen, false par défaut) : si True, détail des extensions liées à l'adhésion
  • garanties (booléen, false par défaut) : si True, détail des garanties acquises
