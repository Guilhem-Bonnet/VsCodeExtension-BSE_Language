Bibliothèque standard des scripts

Event_ChangeStatus

Changement du status d'un événement de la GRC

Parametrres

  • folder (int, obligatoire) : numéro du dossier tiers
  • event (int, obligatoire) : numéro d'ordre de l'événement dans le dossier du tiers
  • status (string, obligatoire) : nouveau code statut de l'événement

Retour

Retour: aucun si pas d'erreur
