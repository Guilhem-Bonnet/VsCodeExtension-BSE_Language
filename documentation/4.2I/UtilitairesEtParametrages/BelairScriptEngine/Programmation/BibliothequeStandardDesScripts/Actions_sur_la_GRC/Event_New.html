Event_New

Event_New

Paramètres

  • dossier (int, obligatoire) : numéro du dossier tiers
  • data : flux XML

Le paramètre data donne le flux XML des autres infos
(on peut ainsi préciser contrat, sinistre, recla)

<data>
    <object typename="CHRO">
      <param name="Codeclas" type="ptUnknown" is_null="true"/>
      <param name="Codelet" type="ptUnknown" is_null="true"/>
      ...
    </object>
</data>

retour

<chro>
  <numtiers>1002</numtiers>
  <ordre>29</ordre>
</chro>

Remarque :
si des pièces sont à joindre, il faudra ensuite utiliser l'action File_Upload en utilisant les identifiants numtiers et ordre retournés par cette action
