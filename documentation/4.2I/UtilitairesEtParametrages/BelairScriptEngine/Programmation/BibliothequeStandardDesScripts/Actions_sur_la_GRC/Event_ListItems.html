Event_ListItems

Récupère la liste des événements d'un tiers, d'un contrat ou d'un sinistre. On peut aussi indiquer l'ordre (l'id événement) pour n'avoir qu'un seul événement.

Paramètres

  • tiers (int, optionnel) : numéro du dossier tiers
  • contrat (int, optionnel) : numéro interne du contrat
  • sinistre (int, optionnel) : numéro interne du sinistre
  • recla (int, optionnel) : numéro interne de la réclamation
  • event (int, optionnel) : numéro d'ordre de l'événement dans le dossier du tiers
  • datefrom (date, optionnel) : date de départ
  • dateto (date, optionnel) : date de fin
  • user (string, optionnel) : utilisateur concerné

traitement

REequête SQL sur CHRO selon les paramètres donnés

retour

<events>
  <objects>
    <object typename="CHRO">
      <param name="Codeclas" type="ptUnknown" is_null="true"/>
      <param name="Codelet" type="ptUnknown" is_null="true"/>
      ...
    </object>
    <object typename="CHRO">...</object>
    ...
  </objects>
</events>
