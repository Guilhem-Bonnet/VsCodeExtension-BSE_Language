Bibliothèque standard des scripts

Event_LinkedFiles

Liste des pièces joints d'un évènement de la GRC

Parametrres

  • folder (int, obligatoire) : numéro du dossier tiers
  • event (int, obligatoire) : numéro d'ordre de l'événement dans le dossier du tiers

Retour

<files>
  <file rank=1>
    <id>1051</id>
    <filename>rib.jpg</filename>
  </file>
  <file rank=2>
    <id>1060</id>
    <filename>rib.jpg</filename>
  </file>
</files>
