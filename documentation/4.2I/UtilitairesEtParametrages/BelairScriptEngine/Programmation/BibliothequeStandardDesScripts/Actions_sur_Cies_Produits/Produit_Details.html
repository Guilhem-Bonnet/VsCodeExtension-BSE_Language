Bibliothèque standard des scripts

Produit_Details

Informations d'un produit

Paramètres

  • code : string, obligatoire : code produit
  • options : booleen, optionnel : détail des garanties
  • basecouvs : boolean, optionnel : détail des niveaux de couverture du produit
  • clauses : boolean, optionnel : détail des clauses du produit

Retour

Flux XML, contenu variable selon paramètres choisis
