Produit_ListItems

Produit_ListItems

Paramètres :

  • typeecran : string, optionnel : type d'écran (A1, A2, etc)
  • branche : string, optionnel : branche métier (PL, SA, etc...)
  • cie : integer, optionnel : filtre sur compagnie principale
  • entite : integer, optionnel : filtre sur l'entité comptable
  • disponible : boolean, optionnel : si True, on ignore les produits 'non disponibles'

Résultat :

<prod-rows>
  <objects>
    <object typename="prod">
      <param name="Branc" type="ptString">EM</param>
      <param name="Branche" type="ptString">AP</param>
      <param name="Catalog" type="ptUnknown" is_null="true"/>
      <param name="Cieprin" type="ptInt" int_val="17"/>
      ...
    <object>
    ...
  </objects>
</prod-rows>
