Mandat_Create

Création d'un Mandat

Paramètres

  • dossier (int, obligatoire) : numéro de tiers
  • data : flux XML de toutes les informations du mandat à créer
    <?xml version="1.0"?>
<mandat>
  <object typename="mandat">
    <param name="Creation" type="ptDateTime" date_val="2013-12-20T00:00:00.000"/>
    <param name="Deriban" type="ptInt" int_val="-1"/>
    <param name="Derics" type="ptUnknown" is_null="true"/>
    <param name="Iban" type="ptInt" int_val="47"/>
    <param name="Numtiers" type="ptInt" int_val="800"/>
    <param name="Revoc" type="ptUnknown" is_null="true"/>
    <param name="Rum" type="ptString">20131220QUERAB41</param>
    <param name="Signatur" type="ptDateTime" date_val="2013-12-20T00:00:00.000"/>
    <param name="Type" type="ptString">M</param>
  </object>
  <contrats>
    <contrat>7</contrat>
    <contrat>53</contrat>
  </contrats>
</mandat>

Retour : n. du mandat créé

<?xml version="1.0"?>
<mandat><id>53</id></mandat>

Contrôles

  • Le tiers dossier doit exister
  • Le champ Iban: doit être renseigné
  • L'iban correspondant doit exister dans la table IBAN et doit être autorisé pour les prélèvements

Remarques:

  • Numtiers est forcé à dossier
  • Type est forcé à "M"
  • DerIban est forcé à -1
  • SI Creation est vide, mise de la date du jour
  • SI Rum est vide , calcul à partir de la fonction MandatRumLib.Mandat_GetRumNumber(mandat)
    • L'algorithme est donc personnalisable
    • Si la formule de calcul SEPANUMRUM existe appel d'un script via BaiimpPool pour utiliser cette formule
  • Tous les contrats indiqués dans la balise optionnelle <contrats> sont associés au mandat créé
