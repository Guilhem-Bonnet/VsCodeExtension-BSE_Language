Iban_Create

Création d'un Iban

Paramètres

  • dossier (int, obligatoire) : numéro de tiers
  • data : flux XML de toutes les informations de l'Iban à créer
    <?xml version="1.0"?>
<iban>
<object typename="Iban">
<param name="Domicil" type="ptString">MASSY</param>
<param name="Iban" type="ptString">FR5110011000201111111111999</param>
<param name="Idbic" type="ptInt" int_val="15"/>
<param name="Prelev" type="ptBool" bool_val="true"/>
<param name="Titul" type="ptString">MARTIN Pierre</param>
<param name="Virement" type="ptBool" bool_val="true"/>
</object>
</iban>

Retour : n. de l'Iban créé

<iban>
<numero>63</numero>
</iban>

Remarques :

  • Iban.Numtiers est forcé à la valeur du paramètre dossier
  • Pas de contrôle de la validité de l'IBAN pour le moment
