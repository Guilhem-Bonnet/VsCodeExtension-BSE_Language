User_EMailAccountListItems

Liste des comptes mail associés à un utilisateur

Parametres

  • username (string, optionnel) : code utilisateur (UTIL.B_UTIL)
    • si absent, récupérer les parametres de l'utilisateur connecté (fonction interne username)
  • default (booléen, optionnel) : si true, limiter au compte par défaut (b_default='T')

retour

<mailing-rows>
  <mailing>
    <codeuser>DANIEL</codeuser>
    <rank>0</rank>
    ...
    <!-- tous les champs rendus par le SQL -->
  </mailing>
  <mailing>
    <codeuser>DANIEL</codeuser>
    <rank>1</rank>
    ...
    <!-- tous les champs rendus par le SQL -->
  </mailing>
</mailing-rows>
