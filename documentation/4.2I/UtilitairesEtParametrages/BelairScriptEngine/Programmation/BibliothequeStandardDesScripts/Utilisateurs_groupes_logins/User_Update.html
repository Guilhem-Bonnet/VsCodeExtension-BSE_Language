User_Update

Modification d'un utilisateur

Parametres

  • login (obligatoire) : Login utilisateur BELAIR
  • data (string, obligatoire) : flux XML contenant les données à modifier pour l'utilisateur 'login'
remarque sur la balise mdp
  • L'action User_details retourne le mot de passe crypté dans la balise <mdp>
  • pour User_update :
    • si la balise <mdp> contient la même valeur, la valeur précédente n'est pas modifiée
    • si elle est différente, alors on considère que <mdp> est le nouveau mot de passe, EN CLAIR, qui sera donc crypté comme pour User_Create
<data>
  <passwordhash>(optionnel) : password</passwordhash>
  <credentialtype>(optionnel) : type Credential </credentialtype>
  <fullname>(optionnel) : Nom Complet </fullname> //peut être null
  <style>(optionnel) : Style </style> //peut être null
  <palette>(optionnel) : Palette </palette> //peut être null
  <language>(optionnel) : language  ex "fr" </language>
  <workphone>(optionnel) : Work Phone </workphone> //peut être null
  <cellphone>(optionnel) : Cell Phone </cellphone> //peut être null
  <disabled>(optionnel) : disabled </disabled>
  <email>(optionnel) : Mail default </email> 
</data>
