Parametres

  • username (string, optionnel) : code utilisateur (UTIL.B_UTIL)
    si absent, récupérer les parametres de l'utilisateur connecté (fonction interne username)
  • profilcrm (true/false), optionnel :
    Si true, il faut trouver quel est le profil CRM_xxxx associé à l'utilisateur et ajouter la balise <profilcrm>CRM_XXXXXX</profilcrm> (c.f. #13609 )
