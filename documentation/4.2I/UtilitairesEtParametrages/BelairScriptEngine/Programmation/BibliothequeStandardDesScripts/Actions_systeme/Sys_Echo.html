Sys_Echo

L'action teste la disponibilité de service des actions et renvoie une chaîne de caractères passée en paramètres.

Paramètres

  • Message (string, obligatoire) : une chaîne de caractères à retourné comme l'écho

Résultats

  • ScriptResult : copie de la chaîne de caractères envoyée

Exemples

<?php
  $actionClient = new BasActionClient($BASE_URL, array("trace" => true, "exceptions" => true)); 
  $params = new BasParams();
  $params->AddString("Message", "Hello from PHP client!");
  $result = BasParams::CreateCopy($actionClient->RunAction($sc, "Sys_Echo", $params->ToSoapVar("params")));
  foreach($result->Item as $param)
  {
    $param->DebugPrint();
  }
?>
