Tab_Save

Cette action permet de créer et de modifier une table de codes et son contenu

Parametres

  • tabcode (String, obligatoire) : nom de la table de codes
  • action (String, obligatoire) : "UPDATE" ou "CREATE"
  • data : flux XML
<?xml version="1.0" encoding="utf-8"?>
<tab tabcode="TITRE" type="C" align="G" size="4">
  <items>
    <objects>
      <object typename="tab"><param name="Tabaff" type="ptString">Auto Ecole</param><param name="Tabcode" type="ptString">TITRE</param><param name="Tabref" type="ptString">AE</param><param name="Tabval" type="ptString">Auto Ecole</param><param name="Valref" type="ptString">AE:Auto Ecole</param></object>
      <object typename="tab"><param name="Tabaff" type="ptString">Aréo-Club</param><param name="Tabcode" type="ptString">TITRE</param><param name="Tabref" type="ptString">AERO</param><param name="Tabval" type="ptString">Aréo-Club</param><param name="Valref" type="ptString">AERO:Aréo-Club</param></object>
      ...
    </objects>
  </items>
</tab>

Ce flux correspond au retour de l'action Tab_ListValues (#13807)

Règles sur les attributs type, align et size

Ces attributs sont optionnels et en cas d'absence, les valeurs suivantes sont prises en compte
  • CREATE
    • type = "C"
    • align = "G"
    • size = "16"
  • UPDATE
    • type = tab.type
    • align = tab.align
    • size = tab.size
