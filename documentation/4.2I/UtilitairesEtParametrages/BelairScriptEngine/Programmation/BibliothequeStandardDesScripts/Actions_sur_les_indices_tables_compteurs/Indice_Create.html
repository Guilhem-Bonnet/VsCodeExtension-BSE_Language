Indice_Create

Création d'un indice

Entité concernée : indices

Paramètres

  • indice (string, obligatoire) : nom de l'indice
  • data : flux XML
  <data>
    <libelle>
    <format>
  </data>

Contrôles

  • vérification parametres indice et data
  • data/libelle : obligatoire
  • data/format : obligatoire . Valeurs acceptées : ('0', '0.0', '0.00', '0.000' , '0.0000' )

Retour

loginfos : "Indice %s créé"
