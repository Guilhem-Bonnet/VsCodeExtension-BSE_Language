Tab_ListValues

Tab_ListValues

Parametres

  • tabcode (string, obligatoire) : nom de la table

retour

<?xml version="1.0"?>
<tab tabcode="TITRE" type="C" align="G" size="4">
  <items>
    <objects>
      <object typename="tab">
        <param name="Tabaff" type="ptString">Auto Ecole</param>
        <param name="Tabcode" type="ptString">TITRE</param>
        <param name="Tabref" type="ptString">AE</param>
        <param name="Tabval" type="ptString">Auto Ecole</param>
        <param name="Valref" type="ptString">AE:Auto Ecole</param>
      </object>
      <object typename="tab">
        <param name="Tabaff" type="ptString">Aréo-Club</param>
        <param name="Tabcode" type="ptString">TITRE</param>
        <param name="Tabref" type="ptString">AERO</param>
        <param name="Tabval" type="ptString">Aréo-Club</param>
        <param name="Valref" type="ptString">AERO:Aréo-Club</param>
      </object>
      ....    
      </objects>
  </items>
</tab>
  • Les attributs type, align et size sont extraites du 'header' (ligne '.' dans TAB)
  • les balises <items> sont les lignes autres que '.'
