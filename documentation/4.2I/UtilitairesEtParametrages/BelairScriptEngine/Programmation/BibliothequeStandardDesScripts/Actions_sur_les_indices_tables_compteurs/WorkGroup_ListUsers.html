Bibliothèque standard des scripts

WorkGroup_ListUsers

Parametres

group (string, obl): code du groupe (param group pour rester cohérent avec les autres actions mais on devrait utiliser partout le nom de l'entité qui est workgroup)
userdetails (boolean, opt) : si true alors on retourne tous les autres champs de l'entité user sauf passwordhash

Retour si OK

<user-rows group="[code du groupe]">
  <user>
    <id></id>
    <login></login>
    <fullname></fullname>
    <disabled></disabled>
    <!-- si userdetails = true alors tous les autres champs de l'entité user -->
  </user>
  ...
</user-rows>

Retour si pas OK

  1. Code du groupe non défini dans la base : warning = "Le groupe [code du groupe] n'existe pas"
