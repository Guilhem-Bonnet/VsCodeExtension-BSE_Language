Xtlog_Updatepwd

Mise à jour du mot de passe d'un login de connexion à une plate forme Bel Air.
Cette fonction peut être utilisée dans un système de réinitialisation de mot de passe (mot de passe perdu, ....)

Paramètres

  • dossier (entier, obligatoire) : dossier dans Belair (xtlog.b_numtiers)
  • ordreext (entier, obligatoire) : numéro d'ordre dans le dossier dans Belair (xtlog.b_ordreext)
  • newpassword (string, obligatoire) : nouveau mot de passe (en MD5)
le parametre dossier peut être obtenu de 2 manieres
  • par l'action Xtlog_login : attribut dossier du noeud tiers
  • par l'action Xtlog_search : parametre numtiers du noeud xtlog
le parametre ordreext peut être obtenu de 2 manieres
  • par l'action Xtlog_login : noeud tiers/ordreext (ajout le 25/09/2019)
  • par l'action Xtlog_search : parametre ordreext du noeud xtlog

Retour

vide

  • si succès, message dans log "mise à jour effectuée"
  • si échec (enregistrement xtlog non trouvé, message dans logError "echec de la mise à jour"
