Bibliothèque standard des scripts

Xtlog_Login

Connexion d'un utilisateur externe à une plate forme externe (extranet, etc...)
Cet utilisateur doit être associé à un dossier Tiers de Belair

Paramètres

  • login (string, obligatoire) : login utilisateur
  • password (string, obligatoire) : mot de passe
  • domain (string, obligatoire) : domaine concerné (ex. 'ECF', 'CMI', 'INTRANET'...)

Retour

Flux Xml:

<?xml version="1.0"?>
<tiers dossier="1000">
  <role>C</role>
  <rsociale>MARTIN Pierre</rsociale>
  <portef/>
  <email/>
</tiers>

En cas d'erreur, i.e. nombre de résultat(s) 1 :

  • retour vide : 
    <?xml version="1.0"?>
<tiers/>
  • et message dans le logError "login ou password incorrect"
