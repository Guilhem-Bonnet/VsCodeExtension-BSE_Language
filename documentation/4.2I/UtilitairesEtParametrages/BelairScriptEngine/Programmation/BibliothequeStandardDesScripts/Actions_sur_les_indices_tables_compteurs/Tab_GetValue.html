Tab_GetValue

Valeur d'une référence pour une table de codes

Entité concernée : tab

Parametres

  • tabcode (string, obligatoire) : nom de la table
  • tabref (string, obligatoire) : référence

Retour

<tab tabcode="aaaa" tabref="aaaa"><valeur>aaaaaa</valeur></tab>
