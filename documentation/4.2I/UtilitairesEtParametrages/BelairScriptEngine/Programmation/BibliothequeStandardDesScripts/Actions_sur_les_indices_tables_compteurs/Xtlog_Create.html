Bibliothèque standard des scripts

Xtlog_Create

Création d'un login de connexion à une plate forme Bel Air.

Paramètres

  • dossier (obligatoire) : numéro de tiers de rattachement
  • domain (obligatoire) : doit être une entrée de la table de codes XTLOGSITE
  • login (obligatoire) : login
  • mot de passe (un des deux est obligatoire):
    • password : mot de passe en clair (sera encodé en MD5)
    • OU passwordMD5 (optionnel) : mot de passe encodé en MD5 (préférable)
  • role (optionnel, "C" par défaut): doit être une entrée de la table de codes XTLOGROLE
  • user (optionnel) : nom utilisateur Belair associé au compte - dans ce cas, role est forcé à "U"
  • email (optionnel) : email associé au compte

Retour

  • Si succès: "OK" avec message dans log "login xxxxxx du domaine xxxxxx créé avec succès"
  • Si échec: "KO" avec message d'erreur dans logerror "ensemble login/password déjà existant", "tiers %s inexistant", ...
