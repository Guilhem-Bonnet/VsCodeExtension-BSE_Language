Indice_ListValues

Liste des valeurs d'un indice

Entité concernée : indval

Parametres

  • indice (string, obligatoire) : nom de l'indice

Contrôles:

  • Vérifier que l'indice existe => erreur
  • Rechercher les valeurs
select b_date as date, b_valeur as valeur from [indval] where b_indice = :indice order by b_date
  • Appliquer à valeur le format de l'indice (c.a.d. le nombre de décimales)

Retour

<indval-rows indice="aaaa" format="xxx">
  <indval date="xxxxx" valeur="xxxxxx">
</indval-rows>
