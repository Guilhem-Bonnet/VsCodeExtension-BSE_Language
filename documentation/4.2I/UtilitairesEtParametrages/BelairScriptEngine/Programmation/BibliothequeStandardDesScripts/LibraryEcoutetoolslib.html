Bibliothèque standard des scripts


Library EcouteToolsLib

Cette librairie permet de traiter des fichiers par un script BSI, via l'automate d'import.
L'intégration de fichier peut être utilisée par exemple pour enregistrer des dossiers saisis par internet.

Cette librairie contient les fonctions utilisées par l'action BSE Ecoute.bs

Utilisation en ligne de commande (ou dans un plannificateur de tâches)

Exemple:

BSEExec.exe /D /:I:D:\belair\appdei\BASScripts\Std\Ecoute.bs /:IP:exececoute.xml.xml /:O:EcouteExecResult.out

exececoute.xml - exemple de contenu du fichier de paramètres d'entrée :

<?xml version="1.0" encoding="utf-8"?>
<params>
  <param name="filename" type="ptString">C:\temp\ecoute\ecouteconf.xml</param>
  <param name="returnlog" type="ptBool" bool_val="true"/>
</params>
