Tiers_RBEListItems

Récupération des bénéficiaires effectifs d'une entreprise

Paramètres:

  • dossier (int, obligatoire) : numéro de tiers . Le tiers doit être une personne morale
  • resultfmt (string, facultatif) : type de retour "xml" pour récupérer un flux XML (par défaut), "" pour récupérer la liste sous forme de tableau d'objets)

Retour

Les données récupérées de l'api opendata-rncs.inpi.fr, selon le format voulu
