Tiers_Search

Recherche alphabétique d'un TIERS

paramètres

  • reference (string, optionnel) : référence à rechercher
  • dppname (string, optionnel) : nom DPP à rechercher (clause b_nom = 'xxx')

referene est obligatoire SAUF si dppname est passé en parametre

  • typetiers (char(1), optionnel) : filtre sur UN type de tiers
  • codp (string, optionnel) : filtre sur codepostal
  • datenais : filtre sur date de naissance du DPP
  • detailorigine (booléen, optionnel) : information sur la table et le champ concerné

Retour
