function Xml_ExtractContent(Source, Path, RootName, CreateFirstNode, IgnoreNull)

Retourne un nouveau document XML représentant le contenu intégral d'un noeud d'un document XML

Paramètres:

  • Source : document XML contenant le noeud à extraire
  • Path : chemin Xpath vers le noeud à extraire
  • RootName : nom du noeud principal du document XML à créer
  • CreateFirstNode (True/False) : création de la balise correspondant au noeud à extraire (si false, on extrait juste son contenu)
  • IgnoreNull (Trus/False) : si True, on ignore les balises de type
