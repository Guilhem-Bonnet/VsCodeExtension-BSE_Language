SQL : résupération des valeurs des champs

SQL:FieldValue(Query, Fieldname)

Fonction générique retournant la valeur d'un champ d'un ensemble de données SQL


Paramètres d'entrée :

  • Query : l'objet de type SQL
  • Fieldname : le nom du champ

SQL:FieldAsBool(Query, Fieldname)

Fonction retournant la valeur d'un champ d'un ensemble de données SQL sous forme True/False


Paramètres d'entrée :

  • Query : l'objet de type SQL
  • Fieldname : le nom du champ

Rôle de cette fonction :
Selon le SGBD utilisé, les champs booléens de la base Belair sont soit des champs natifs booléens, soit des champs de type VarChar(1).
Un champ booléen de type varchar(1) accepte les valeurs 'T' et 'V' pour True et 'F' pour False:
cette fonction permet de traiter de manière uniforme les champs booléens de la base Belair en rendant toujours True ou False, quelque soit le SGBD.
