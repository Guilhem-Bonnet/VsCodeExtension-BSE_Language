Fonctions de sécurité

EncodePwd(password, casse)

Attention :

Cette fonction est dépréciée !
Pour encoder le mot de passe d'un utilisateur : utiliser MakePasswordHash.


La fonction renvoie le texte encodé, selon l'algorithme interne Belair de hachage (basé sur une clé).


Paramètres d'entrée :

  • password : la chaîne de caractères à encoder.
  • casse (optionnel, booléen) pour transformer les caractères à la casse majuscule, "True" par défaut.

Résultat :
La valeur string encodée en minuscule.


Exemples :

Print(EncodePwd("Password"))
Print(EncodePwd("Password", false))

Résultat :

03081e0304001c03
03283e2324203c23
