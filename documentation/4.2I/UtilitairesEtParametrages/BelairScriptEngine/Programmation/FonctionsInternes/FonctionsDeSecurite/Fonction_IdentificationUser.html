Identification de l'utilisateur

UserName()

La fonction renvoie le nom de l'utilisateur.

UserFullName()

La fonction renvoie le nom complet de l'utilisateur, si renseigné.

SiteName()

La fonction renvoie le nom du site éventuellement associé à l'utilisateur.

IsUserinWorkGroup(WorkGroup)

La fonction renvoie True si l'utilisateur appartient à un groupe d'utilisateurs, False sinon.


Paramètres d'entrée :

  • workgroup : le nom du groupe d'utilisateurs.
