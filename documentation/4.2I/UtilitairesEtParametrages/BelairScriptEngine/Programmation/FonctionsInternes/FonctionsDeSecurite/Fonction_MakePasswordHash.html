Fonctions de sécurité

MakePasswordHash

La fonction renvoie le texte encodé, selon l'algorithme interne Belair de hachage (SHA2-512/256 with salt).


Paramètres d'entrée :

  • 1. L'identifiant d'utilisateur (integer).
  • 2. Le mot de passe à encoder (string).

Résultat :
La valeur string encodée (en minuscule).


Exemples :

Print(MakePasswordHash(123, "Password"))

Résultat :

7aa4ea38500430085350f8b9ef6b4e8e9c659857069bb9ecc53572e48f2ac3f8
