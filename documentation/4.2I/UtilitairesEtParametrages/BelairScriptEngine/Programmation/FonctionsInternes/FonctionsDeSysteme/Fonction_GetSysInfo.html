Fonctions de système

GetSysInfo()

La fonction renvoie les informations système.


Paramètres d'entrée :

Aucun.



Résultat :

L'objet concernant les informations suivantes :

  • Version : une chaîne contenant une version de Belair soit :
    • versionMajor ;
    • versionMinor ;
    • versionRelease ;
    • versionRevision.
  • ApplicationName : le nom d'application.
  • OSVersionName : la chaîne contenant la version du système d'exploitation.


Exemples :

local sysinfo
sysinfo := GetSysInfo()
Print(sysinfo.Version)
Print(sysinfo.VersionMajor)
Print(sysinfo.VersionMinor)
Print(sysinfo.VersionRelease)
Print(sysinfo.VersionRevision)
Print(sysinfo.ApplicationName)
Print(sysinfo.OSVersionName)
