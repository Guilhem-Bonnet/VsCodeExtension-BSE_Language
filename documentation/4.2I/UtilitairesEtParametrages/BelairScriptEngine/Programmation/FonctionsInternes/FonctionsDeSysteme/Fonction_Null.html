Fonctions de système

Null()

La fonction renvoie la valeur vide.
Pour comparer utiliser IsNull().


Paramètres d'entrée :

Aucun.



Résultat :

La valeur vide.


Exemples :

local o1
o1 := Obj:New()
o1.PropInt := 123
o1.EmptyProp := Null()
Print(DumpVar(o1))
Print("IsNull(EmptyProp): %s", IsNull(o1.EmptyProp))

Résultat :

Object (Object):
  EmptyProp (Empty): 
  PropInt (Integer): 123
IsNull(EmptyProp): True
