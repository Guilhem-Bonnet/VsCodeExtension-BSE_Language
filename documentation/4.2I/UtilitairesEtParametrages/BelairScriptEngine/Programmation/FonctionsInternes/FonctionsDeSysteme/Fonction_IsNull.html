Fonctions de système

IsNull(valeur)

La fonction renvoie "true" si l'argument est vide, ou "false" si non.


Paramètres d'entrée :

valeur : une valeur (variable, constante, retour de fonction, etc).



Résultat :

Valeur booléenne.


Exemples :

local o1
o1 := Obj:New()
o1.PropInt := 123
o1.EmptyProp := Null()
Print("IsNull(PropInt): %s", IsNull(o1.PropInt))
Print("IsNull(EmptyProp): %s", IsNull(o1.EmptyProp))

Résultat :

IsNull(PropInt): False
IsNull(EmptyProp): True
