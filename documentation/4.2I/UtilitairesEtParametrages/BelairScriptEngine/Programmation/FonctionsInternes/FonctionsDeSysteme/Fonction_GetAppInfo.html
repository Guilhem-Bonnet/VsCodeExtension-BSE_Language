Fonctions de système

GetAppInfo()

La fonction renvoie les informations de l'application Belair.
La connexion à Belair est nécessaire, sinon une erreur est générée.


Paramètres d'entrée :

Aucun.


Résultat :

L'objet concernant les informations suivantes :

  • ConfFileName : le fichier de configuration utilisé.
  • AppName : le nom de l'application (3 caractères alphanumériques).
  • AppTitle : l'intitulé de l'application.


Exemples :

local appinfo
appinfo := GetAppInfo()
Print(appinfo.ConfFileName)
Print(appinfo.AppName)
Print(appinfo.AppTitle)

Résultat :

D:\belair\APPDEI\appdei.mssql.CONF
DEI
PRESENTATION BELAIR
