Fonctions de système

Assert(condition, message)


La fonction vérifie l'expression de type booléen et génère une erreur si la valeur est "false".



Paramètres d'entrée :

  • condition : l'expression, ou variable, de type booléen ;
  • message : le message d'erreur (optionnel).


Résultat :

Aucun


Exemples :

local test
test := 0
Assert(test = 0) // OK
Assert(not (test < 0), "Negative value") // OK
Assert(test > 0, "Natural number is expected") // Error
