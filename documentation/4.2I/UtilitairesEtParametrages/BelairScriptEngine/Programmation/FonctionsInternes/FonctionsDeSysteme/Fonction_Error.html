Fonctions de système

Error(code, message, arguments)


Génère une exception interne et interrompt l'exécution de script.
Si une transaction est en cours, elle sera annulée.

En contexte BAS, l'exception standard SOAP sera générée et envoyée au client. Voir "Gestion des exception BAS".


Paramètres d'entrée :

  • Code (int, optionnel) - le code d'erreur.
  • Message (string) - le message d'erreur ou la chaîne de format comme dans la fonction Print.
  • Arguments (optionnel) - les arguments à formater.

Résultat :
Aucun.


Exemples :

Error("General protection error")
Error(123, "File not found")

local msg
msg := "User not found" 
Error(msg)

Error("Format avec %s", param1)
Error(111, "Format avec %s", param1)

local fmt
fmt := "User not found %s" 
Error(fmt, param1)

Error(456, "Format avec %s, %s, %s", param1, param2, param3)
