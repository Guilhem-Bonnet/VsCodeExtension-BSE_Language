Fonctions de fichiers

Fonctions de manipulation de fichiers texte

File:ReadLineCSV(FileDescriptor)

La fonction lit une ligne en format CSV depuis le fichier texte ouvert pour la lecture préalable.


Paramètres d'entrée :

  • FileDescriptor : l'objet de type FileDescriptor.


Résultat :
Le tableau de valeurs de type "string".


Exemples :

local f1, values, lineCSV
f1 := File:Create("test.txt")
values := Obj:NewArray()
values[0] := 123
values[1] := "String1" 
values[2] := "String 2 with spaces" 
values[3] := "String ""3"" with quotes" 
values[4] := "String 4" + CRLF() + "with CRLF" 
File:WriteLineCSV(f1, values)
values[0] := 456
File:WriteLineCSV(f1, values)
File:Close(f1)
// Read first line of CSV file
f1 := File:OpenRead("test.txt")
lineCSV := File:ReadLineCSV(f1)
Print(DumpVar(lineCSV))

Résultat :

Array length: 5
  [0]: 123
  [1]: String1
  [2]: String 2 with spaces
  [3]: String "3" with quotes
  [4]: String 4
with CRLF
