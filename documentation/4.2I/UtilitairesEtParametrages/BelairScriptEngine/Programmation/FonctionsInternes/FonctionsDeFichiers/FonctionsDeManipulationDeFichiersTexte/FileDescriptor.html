Type FileDescriptor

FileDescriptor est un type d'objet utilisé pour les manipulations de fichiers texte.

Propriétés du type FileDescriptor

CSVSeparator (string) : le caractère de séparation des valeurs dans un fichier CSV. ";" est par défaut.
Encoding (string) : l'encodage de fichier ouvert pour lecture ou l'écriture.
EOF (bool) : indique si le fin de fichier est atteint lors de la lecture par ReadLine(), etc...
Path (string) : le nom de fichier avec le chemin éventuel

Exemple

local f1
f1 := File:Create("test.txt")
File:WriteLine(f1, "First line")
File:WriteLine(f1, "Second line")
File:Close(f1)
f1 := File:OpenRead("test.txt")
Print(DumpVar(f1))
while not f1.EOF begin
  Print(File:ReadLine(f1))
end

Résultat

Object (FileDescriptor):
  CSVSeparator (UnicodeString): ;
  Encoding (UnicodeString): UTF8
  EOF (Boolean): False
  Path (UnicodeString): test.txt

Second line
