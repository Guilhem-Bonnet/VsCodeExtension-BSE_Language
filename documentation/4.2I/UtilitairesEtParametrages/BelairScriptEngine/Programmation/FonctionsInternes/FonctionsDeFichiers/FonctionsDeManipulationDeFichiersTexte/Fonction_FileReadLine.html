Fonctions de fichiers

Fonctions de manipulation de fichiers texte

File:ReadLine(FileDescriptor)

La fonction lit une ligne de fichier texte ouvert pour la lecture préalable.


Paramètres d'entrée :

  • FileDescriptor : l'objet de type FileDescriptor.


Résultat :
La chaîne de caractères lue.


Exemples :

local f1
f1 := File:Create("test.txt")
File:WriteLine(f1, "First line")
File:WriteLine(f1, "Second line")
File:Close(f1)
// Read the file
f1 := File:OpenRead("test.txt")
while not f1.EOF begin
  Print(File:ReadLine(f1))
end

Résultat :

First line
Second line
