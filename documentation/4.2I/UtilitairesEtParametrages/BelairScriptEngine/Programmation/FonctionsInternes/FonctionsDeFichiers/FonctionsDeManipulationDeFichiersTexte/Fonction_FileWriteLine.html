Fonctions de fichiers

Fonctions de manipulation de fichiers texte

File:WriteLine(FileDescriptor, String)

La fonction ajoute une ligne dans le fichier texte créé ou ouvert pour l'écriture préalable.


Paramètres d'entrée :

  • FileDescriptor : l'objet de type FileDescriptor.
  • String : la chaîne de caractères.


Résultat :
Aucun.


Exemples :

local f1
f1 := File:Create("test.txt")
File:WriteLine(f1, "First line")
File:WriteLine(f1, "Second line")
File:Close(f1)
