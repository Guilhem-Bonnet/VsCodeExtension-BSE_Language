Fonctions de fichiers

Fonctions de manipulation de fichiers texte

File:OpenWrite(FileName, Encodage)

La fonction ouvre un ficher texte pour l'écriture en mode exclusive.


Paramètres d'entrée :

  • FileName : le nom de fichier (string).
  • Encodage : l'encodage (optionnel, string) :
    • ANSI
    • UTF8 ou UTF-8 (valeur par défaut)
    • UTF16 ou UTF-16


Résultat :
Un objet de type FileDescriptor.


Exemples :

local f1
f1 := File:Create("test.txt")
File:WriteLine(f1, "First line")
File:Close(f1)
// Append second line
f1 := File:OpenWrite("test.txt")
File:WriteLine(f1, "Second line")
File:Close(f1)
//
f1 := File:OpenRead("test.txt")
while not f1.EOF begin
  Print(File:ReadLine(f1))
end

Résultat :

First line
Second line
