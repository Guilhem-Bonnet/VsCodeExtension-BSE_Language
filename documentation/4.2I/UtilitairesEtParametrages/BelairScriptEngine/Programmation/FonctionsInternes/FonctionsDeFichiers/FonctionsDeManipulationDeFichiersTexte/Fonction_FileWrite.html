Fonctions de fichiers

Fonctions de manipulation de fichiers texte

File:Write(FileDescriptor, String)

La fonction ajoute une chaîne de caractères dans le fichier texte créé ou ouvert pour l'écriture préalable.


Paramètres d'entrée :

  • FileDescriptor : l'objet de type FileDescriptor.
  • String : la chaîne de caractères.


Résultat :
Aucun.


Exemples :

local f1, i, c
f1 := File:Create("test.txt")
File:Write(f1, "ABC")
File:Close(f1)
f1 := File:OpenRead("test.txt")
i := 1
while not f1.EOF begin
  c := File:ReadChar(f1)
  Print("Char[%s]: %s (%s)", i, c, Ord(c))
  i := i + 1
end

Résultat :

Char[1]: A (65)
Char[2]: B (66)
Char[3]: C (67)
