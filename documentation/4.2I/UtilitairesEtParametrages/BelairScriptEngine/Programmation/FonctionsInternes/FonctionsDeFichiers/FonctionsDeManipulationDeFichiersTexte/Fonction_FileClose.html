Fonctions de fichiers

Fonctions de manipulation de fichiers texte

File:Close(FileDescriptor)

La fonction ferme le fichier ouvert pour la lecture ou l'écriture.


Paramètres d'entrée :

  • FileDescriptor : l'objet FileDescriptor.


Résultat :
Aucun.


Exemples :

local f1
f1 := File:Create("test.txt")
// Write to file here
File:Close(f1)
