Librairie de Fonctions Complémentaires de manipulation de fichiers


Library FileSystem

Cette librairie donne des fonctions complémentaires de manipulation de fichier

Function GetFileList(Path, Filter, Recurse, AddPathName)


Retourne un tableau contenant une liste de fichiers


Path = répertoire
Filter = filtre (exemple "*.TXT")
Recurse = si TRUE, recherche récursive dans les sous-répertoires
AddPathName = si TRUE, le tableau contient le chemin complet des fichiers trouvés ; sinon, il ne contient que les noms de fichier
