Fonctions de manipulation de dates

IsInLeapYear(Date)

Indique si la valeur DateTime spécifiée appartient à une année bissextile.


Paramètres d'entrée :

  • Date : la date.


Résultat :
La valeur de type booléan.


Exemples : 

local date
date := EncodeDate(2016, 10, 11)
Print(IsInLeapYear(date))
date := EncodeDate(2018, 10, 11)
Print(IsInLeapYear(date))

Résultat :

True
False
