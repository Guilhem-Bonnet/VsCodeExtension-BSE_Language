Fonctions de manipulation de dates

DateDiff(DiffUnit, ToValue, FromValue)

Renvoie la différence entre deux valeurs DateTime.


Paramètres d'entrée :

  • DiffUnit : la part de DateTime (l'unité de différence) :
    • "year", "yy" pour les années ;
    • "month", "mm" pour les mois ;
    • "week", "ww" pour les semaines ;
    • "day", "dd" pour les jours ;
    • "hour", "hh" pour les heures ;
    • "minute", "mi" pour les minutes ;
    • "second", "ss" pour les seconds ;
    • "millisecond", "ms" pour les milliseconds.
  • ToValue : la valeur "jusqu'à".
  • FromValue : la valeur "a partir".


Résultat :
Le nombre entier (signé) de la différence, dans l'unité choisie.


Exemples : 

local date1, date2
date1 := EncodeDateTime(2015, 10, 11, 21, 35, 51, 123)
date2 := EncodeDateTime(2017, 8, 25, 7, 44, 31, 456)
Print(DateDiff("year", date2, date1))
Print(DateDiff("month", date2, date1)) 
Print(DateDiff("week", date2, date1))
Print(DateDiff("day", date2, date1))
Print(DateDiff("hour", date2, date1))
Print(DateDiff("minute", date2, date1))
Print(DateDiff("second", date2, date1))
Print(DateDiff("ms", date2, date1))

Résultat :

Years: 1
Months: 22
Weeks: 97
Days: 684
Hours: 16402
Minutes: 984128
Seconds: 59047720
Milliseconds: 59047720333
