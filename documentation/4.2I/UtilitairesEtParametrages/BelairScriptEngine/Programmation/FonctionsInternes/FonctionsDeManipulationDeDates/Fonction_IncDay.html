Fonctions de manipulation de dates

IncDay(Date, NumberOfDay)

Renvoie une date décalée d'un certain nombre de jours.


Paramètres d'entrée :

  • Date : la date et le temps.
  • NumberOfDay : nombre entier de jours à décaler, peut être négatif pour renvoyer à une date de n jours antérieure.


Résultat :
La valeur de type DateTime décalée.


Exemples : 

local date
date := EncodeDateTime(2016, 10, 11, 21, 35, 51)
Print(FormatDateTime("yyyy-mm-dd hh:nn:ss", date))
Print(FormatDateTime("yyyy-mm-dd hh:nn:ss", IncDay(date, 10)))
Print(FormatDateTime("yyyy-mm-dd hh:nn:ss", IncDay(date, -10)))

Résultat :

2016-10-11 21:35:51
2016-10-21 21:35:51
2016-10-01 21:35:51
