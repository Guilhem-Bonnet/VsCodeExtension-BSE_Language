Fonctions de manipulation de dates

IncYear(Date, NumberOfYear)

Renvoie une date décalée d'un certain nombre d'années.


Paramètres d'entrée :

  • Date : la date et le temps.
  • NumberOfYear : nombre entier d'années à décaler, peut être négatif pour renvoyer à une date de n années antérieure.


Résultat :
La valeur de type DateTime décalée.


Exemples : 

local date
date := EncodeDateTime(2016, 10, 11, 21, 35, 51)
Print(FormatDateTime("yyyy-mm-dd hh:nn:ss", date))
Print(FormatDateTime("yyyy-mm-dd hh:nn:ss", IncYear(date, 3)))
Print(FormatDateTime("yyyy-mm-dd hh:nn:ss", IncYear(date, -3)))

Résultat :

2016-10-11 21:35:51
2019-10-11 21:35:51
2013-10-11 21:35:51
