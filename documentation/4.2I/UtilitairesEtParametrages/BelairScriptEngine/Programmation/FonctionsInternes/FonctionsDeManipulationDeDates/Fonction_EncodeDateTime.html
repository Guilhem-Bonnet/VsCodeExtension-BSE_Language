Fonctions de manipulation de dates

EncodeDateTime(YearValue, MonthValue, DayValue, HourValue, MinValue, SecValue)

La fonction encode la date et le temps spécifiés par les valeurs entières de l'année, du mois etc.


Paramètres d'entrée :

  • YearValue : l'année ("yyyy").
  • MonthValue : le mois ("mm").
  • DayValue : le jour ("nn").
  • HourValue : l'heure ("hh").
  • MinValue : les minutes ("nn").
  • SecValue : les secondes ("ss").


Résultat :
La valeur de type DateTime.


Exemples : 

local date1
date1 := EncodeDateTime(2017, 2, 22, 10, 55, 32) 
Print(date1)
Print(FormatDateTime("yyyy-mm-nn hh:nn:ss", date1))

Résultat :

22/02/2017 10:55:32
2017-02-55 10:55:32
