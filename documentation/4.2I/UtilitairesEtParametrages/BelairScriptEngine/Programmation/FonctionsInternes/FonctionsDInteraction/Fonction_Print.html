Fonctions d'interaction

Print(text,variables)

La fonction renvoie le texte vers la sortie abstraite, qui est définie par le programme d'exécution des scripts, par exemple :

  • la console Windows, en cas de BSEExec ;
  • la fenêtre de sortie, en cas de BSEDev ;
  • etc.


Paramètres d'entrée :

  • texte : le texte ou la variable de type string,
ou :
  • texte : Une chaine de format compatible à celui de la fonction Format de Delphi ;
    (Uniquement l'indication de format %s est acceptée !)
  • variables : liste des variables à formater (de 1 à 99 au maximum)


Résultat :
Aucun.


Exemples : 

print("Message")
print(SomeStringVariable)
print("Format avec %s", param1)
print(FormatVariable, param1)
print("Format avec %s, %s, %s", param1, param2, param3)
