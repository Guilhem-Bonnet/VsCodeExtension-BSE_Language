Fonctions de XML

XML:FirstChild(doc)

La fonction rend courant le premier nœud enfant de celui courant.


Paramètres d'entrée :

  • doc : le document XML.

Résultat :

  • "True" - l'opération est réussie.
  • "False" - le nœud courant n'a pas d'enfant ; le nœud courant n'est pas changé.

Exemples :

local doc
// Create document
doc := XML:New()
XML:StartNode(doc, "items")
  XML:StartNode(doc, "item1")
    XML:AppendNode(doc, "item11")
  XML:EndNode(doc)
  XML:StartNode(doc, "item2")
// Navigation
XML:First(doc)
Print(XML:GetNodeName(doc))
XML:FirstChild(doc)
Print(XML:GetNodeName(doc))
XML:FirstChild(doc)
Print(XML:GetNodeName(doc))
if XML:Next(doc) then Print("Exists") else Print("No more children")
XML:Parent(doc)
XML:Next(doc)
Print(XML:GetNodeName(doc))
Print(XML:ToString(doc))

Résultat :

items
item1
item11
No more children
item2
<?xml version="1.0"?>
<items><item1><item11/></item1><item2/></items>
