Fonctions de XML

XML:SetNodeText(doc, texte)

La fonction affecte le texte du nœud courant.


Paramètres d'entrée :

  • doc : le document XML.
  • texte : l'expression ou la variable (tous les types convertibles au string sont acceptables), ou le texte.

Résultat :
Aucun.


Exemples :
local doc
doc := XML:New()
XML:StartNode(doc, "car")
  XML:StartNode(doc, "manufacturer")
    XML:SetNodeText(doc, "Toyota")
  XML:EndNode(doc)
  XML:StartNode(doc, "on_stock")
    XML:SetNodeText(doc, true)
  XML:EndNode(doc)
  XML:StartNode(doc, "last_seen_at")
    XML:SetNodeText(doc, EncodeDateTime(2017, 11, 9, 12, 15, 35))
  XML:EndNode(doc)
  XML:StartNode(doc, "weight")
    XML:SetNodeText(doc, 1234.5678)
  XML:EndNode(doc)
  XML:StartNode(doc, "count")
    XML:SetNodeText(doc, 4321)
  XML:EndNode(doc)
XML:EndNode(doc)
Print(XML:PrettyFormat(doc))

Résultat :

<?xml version="1.0"?>
<car>
  <manufacturer>Toyota</manufacturer>
  <on_stock>true</on_stock>
  <last_seen_at>2017-11-09T12:15:35.000</last_seen_at>
  <weight>1.23456780000000E+0003</weight>
  <count>4321</count>
</car>
