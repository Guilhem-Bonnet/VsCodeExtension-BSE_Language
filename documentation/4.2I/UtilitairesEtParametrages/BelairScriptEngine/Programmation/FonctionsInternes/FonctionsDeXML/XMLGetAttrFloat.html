Fonctions de XML

XML:GetAttrFloat(doc, name, default)

La fonction renvoie la valeur d'attribut du nœud courant convertie en type réel (double precision).
Si l'attribut n'existe pas, et si la valeur par défaut n'est pas spécifiée : alors, une erreur est générée.


Paramètres d'entrée :

  • doc : le document XML.
  • name : le nom d'attribut.
  • default : la valeur par défaut (optionnel).

Résultat :
La valeur (double precision).


Exemples :

local doc, weight, height
doc := XML:New()
XML:StartNode(doc, "order")
  XML:SetAttr(doc, "weight", 123.456)
  weight := XML:GetAttrFloat(doc, "weight")
  height := XML:GetAttrFloat(doc, "height", 23.4)
XML:EndNode(doc)
Print(weight)
Print(height)
Print(XML:ToString(doc))

Résultat :

123,456
23,4
