Fonctions de XML

XML:PrettyFormat(doc)

La fonction formate le texte de document XML en format pouvant être interprété par l'homme.


Paramètres d'entrée :

  • doc : le document XML.

Résultat :

  • Le texte (string) contenant l'XML formaté.

Exemples :
local doc
doc := XML:New()
XML:StartNode(doc, "car")
  XML:AppendNode(doc, "manufacturer", "Toyota")
  XML:AppendNode(doc, "on_stock", true)
  XML:AppendNode(doc, "last_seen_at", EncodeDateTime(2017, 11, 9, 12, 15, 35))
  XML:AppendNode(doc, "weight", 1234.5678)
  XML:AppendNode(doc, "count", 4321)
XML:EndNode(doc)
Print(XML:PrettyFormat(doc))

Résultat :

<?xml version="1.0"?>
<car>
  <manufacturer>Toyota</manufacturer>
  <on_stock>true</on_stock>
  <last_seen_at>2017-11-09T12:15:35.000</last_seen_at>
  <weight>1.23456780000000E+0003</weight>
  <count>4321</count>
</car>
