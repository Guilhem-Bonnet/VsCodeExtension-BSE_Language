Fonctions de XML

XML:GetNodeBool(doc, default)

La fonction renvoie le texte du nœud courant convertie en type booléen.


Paramètres d'entrée :

  • doc : le document XML.
  • default : la valeur par défaut.

Résultat :
La valeur de type booléen convertie à partir du texte du nœud courant.
Si le texte est vide ou absent : alors, la valeur par défaut est retournée.

Génère une erreur si la conversion n'est pas possible.


Les valeurs/formats acceptées/és :

  • "True".
  • "False".

Les valeurs ne sont pas sensitives à la casse de caractères.


Exemples :

local doc
doc := XML:New()
XML:StartNode(doc, "items")
  XML:AppendNode(doc, "item1", true)
  XML:AppendNode(doc, "item2", "true")
  XML:AppendNode(doc, "item3")
XML:EndNode(doc)
XML:First(doc)
XML:FirstChild(doc)
Print(XML:GetNodeBool(doc))
XML:Next(doc)
Print(XML:GetNodeBool(doc))
XML:Next(doc)
Print(XML:GetNodeBool(doc, false))
XML:First(doc)
Print(XML:PrettyFormat(doc))

Résultat :

True
True
False
