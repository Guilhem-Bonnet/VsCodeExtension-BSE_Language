Fonctions de XML

XML:EndNode(doc)

La fonction revient vers le nœud courant comme il était avant le dernier appel de XML:StartNode.


Paramètres d'entrée :

  • doc : le document XML.

Résultat :
Aucun.


Exemples :

local doc
doc := XML:New()
XML:StartNode(doc, "cars")
  Print(XML:GetNodeName(doc))
  XML:StartNode(doc, "toyota")
    Print(XML:GetNodeName(doc))
  XML:EndNode(doc)
  Print(XML:GetNodeName(doc))
XML:EndNode(doc)

Résultat :

cars
toyota
cars
