Fonctions de XML

XML:AreSame(doc1, doc2, ignore)

La fonction compare deux documents XML.
La fonction n'est pas sensitive à la casse de caractères de valeurs ou de balises. Pour la comparaison sensitive à la casse : utiliser XML:AreEqual.


Paramètres d'entrée :

  • doc1 : le premier document XML.
  • doc2 : le deuxième document XML.
  • ignore (True/False): inorer la casse de caractères de noms de balises/attributs (optionnel, False par défaut).

Résultat :

  • "True" - les documents sont égaux.
  • "False" - il y a une différence entre ces documents.

Exemples :

local doc1, doc2
doc1 := XML:New()
XML:StartNode(doc1, "items")
  XML:StartNode(doc1, "item1")
    XML:SetAttr(doc1, "attr1", "Attr1 Text")
    XML:SetNodeText(doc1, "Item1 Text")
  XML:EndNode(doc1)
  XML:AppendNode(doc1, "item2", "Item2 Text")
XML:EndNode(doc1)
doc2 := XML:Open("<items><item1 attr1=""attr1 text"">item1 text</item1><item2>item2 text</item2></items>")
Print(XML:AreSame(doc1, doc2))
doc2 := XML:Open("<Items><Item1 Attr1=""Attr1 Text"">Item1 Text</Item1><Item2>Item2 Text</Item2></Items>")
Print(XML:AreSame(doc1, doc2, true))

Résultat :

True
True
