Fonctions de XML

XML:GetAttrInt(doc, name, default)

La fonction renvoie la valeur d'attribut du nœud courant convertie en type entier (int32).
Si l'attribut n'existe pas, et si la valeur par défaut n'est pas spécifiée : alors, une erreur est générée.


Paramètres d'entrée :

  • doc : le document XML.
  • name : le nom d'attribut.
  • default : la valeur par défaut (optionnel).

Résultat :
La valeur (int32).


Exemples :

local doc, item_count, total_qty
doc := XML:New()
XML:StartNode(doc, "order")
  XML:SetAttr(doc, "item_count", 12)
  item_count := XML:GetAttrFloat(doc, "item_count")
  total_qty := XML:GetAttrFloat(doc, "total_qty", -1)
XML:EndNode(doc)
Print(item_count)
Print(total_qty)
Print(XML:ToString(doc))

Résultat :

12
-1
