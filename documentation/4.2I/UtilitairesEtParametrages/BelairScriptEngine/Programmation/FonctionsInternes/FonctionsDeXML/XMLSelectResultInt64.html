Fonctions de XML

XML:SelectResultInt64(doc, Xpath, default)

La fonction sélectionne le nœud comme XML:SelectResult, puis le convertie en type numérique entier comme XML:GetNodeInt64.

Paramètres d'entrée :

  • doc : le document XML.
  • Xpath : la requête XPath.
  • default : la valeur par défaut, si la requête ne trouve rien (optionnel).

Exemples :
local doc
doc := XML:New()
XML:StartNode(doc, "items")
  XML:AppendNode(doc, "item1", 1234567812345678)
  XML:AppendNode(doc, "item2", "1234567812345678")
  XML:AppendNode(doc, "item3")
XML:EndNode(doc)
Print(XML:SelectResultInt64(doc, "/items/item1"))
Print(XML:SelectResultInt64(doc, "/items/item2"))
Print(XML:SelectResultInt64(doc, "/items/item3", 8765432187654321))
Print(XML:PrettyFormat(doc))

Résultat :

1234567812345678
1234567812345678
8765432187654321
<?xml version="1.0"?>
<items>
  <item1>1234567812345678</item1>
  <item2>1234567812345678</item2>
  <item3/>
</items>
