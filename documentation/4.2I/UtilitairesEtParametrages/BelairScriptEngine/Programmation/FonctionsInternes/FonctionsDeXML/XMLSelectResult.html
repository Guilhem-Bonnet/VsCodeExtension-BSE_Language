Fonctions de XML

XML:SelectResult(doc, Xpath, default)

La fonction recherche le nœud spécifié par la requête XPath comme la fonction XML:Select, puis renvoie le texte du nœud trouvé.
Le texte peut être affecté par la fonction XML:SetNodeText.


Paramètres d'entrée :

  • doc : le document XML.
  • Xpath : la requête XPath.
  • default : la valeur par défaut, si la requête ne trouve rien (optionnel).

Résultat :

  • Le texte du nœud trouvé.

Si la requête XPath ne trouve rien, et si la valeur par défaut n'est pas spécifiée, une erreur est générée.


Exemples :
local doc
doc := XML:New()
XML:StartNode(doc, "cars")
  XML:AppendNode(doc, "manufacturer", "Toyota")
  XML:StartNode(doc, "manufacturer")
    XML:SetNodeText(doc, "Renault")
    XML:SetAttr(doc, "country", "France")
  XML:EndNode(doc)
  XML:AppendNode(doc, "manufacturer", "Nissan")
XML:EndNode(doc)
Print(XML:SelectResult(doc, "/cars/manufacturer[1]"))
Print(XML:SelectResult(doc, "//manufacturer[@country='France']"))
Print(XML:SelectResult(doc, "//*[text()='Renault']"))
Print(XML:SelectResult(doc, "//*[text()='Ford']", "No Ford found"))
Print(XML:PrettyFormat(doc))

Résultat :

Toyota
Renault
Renault
No Ford found
<?xml version="1.0"?>
<cars>
  <manufacturer>Toyota</manufacturer>
  <manufacturer country="France">Renault</manufacturer>
  <manufacturer>Nissan</manufacturer>
</cars>
