Fonctions de XML

XML:GetNodeInt64(doc, default)

La fonction renvoie le texte du nœud courant converti en type numérique entier (64 bits).


Paramètres d'entrée :

  • doc : le document XML.
  • default : la valeur par défaut.

Résultat :
La valeur de type numérique entier (int64) convertie à partir du texte du nœud courant.
Si le texte est vide ou absent : alors la valeur par défaut est retournée.

Génère une erreur si la conversion n'est pas possible.

Les valeurs/formats acceptées/és :

  • Format fixe.
  • Valeurs entre -9 223 372 036 854 775 808 à 9 223 372 036 854 775 807.

Les valeurs de virgule flottante passées comme des arguments seront coupées.


Exemples :

local doc
doc := XML:New()
XML:StartNode(doc, "items")
  XML:AppendNode(doc, "item1", 1234567812345678)
  XML:AppendNode(doc, "item2", "1234567812345678")
  XML:AppendNode(doc, "item3")
XML:EndNode(doc)
XML:First(doc)
XML:FirstChild(doc)
Print(XML:GetNodeInt64(doc))
XML:Next(doc)
Print(XML:GetNodeInt64(doc))
XML:Next(doc)
Print(XML:GetNodeInt64(doc, 8765432187654321))
Print(XML:GetNodeInt64(doc, 87654321.87654))
XML:First(doc)
Print(XML:PrettyFormat(doc))

Résultat :

1234567812345678
1234567812345678
8765432187654321
87654322
<?xml version="1.0"?>
<items>
  <item1>1234567812345678</item1>
  <item2>1234567812345678</item2>
  <item3/>
</items>
