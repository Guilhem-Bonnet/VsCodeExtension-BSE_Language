Fonctions de XML

XML:New()

La fonction crée un objet de type "document XML" et le renvoie.


Paramètres d'entrée :
Aucun.


Résultat :
Un objet de type "document XML".


Exemple :
local doc
doc := GetColorsXml()
Print(XML:ToString(doc))
return

function GetColorsXml()
  local doc
  doc := XML:New()
  XML:StartNode(doc, "colors")
    XML:AppendNode(doc, "color", "red")
    XML:AppendNode(doc, "color", "green")
    XML:AppendNode(doc, "color", "blue")
  XML:EndNode(doc)
return doc

Résultat :

<?xml version="1.0"?>
<colors><color>red</color><color>green</color><color>blue</color></colors>

XML:SetForceLowerCase(Doc, Value)

La fonction force ou non le nommage des attributs et des balises en minuscules (comportement par défaut)


Paramètres d'entrée :

  • doc : le document XML.
  • value (booléen) : nommage forcé en minuscules (True) ou tel que saisi (False)

Résultat :
aucun
