Fonctions de XML

XML:GetNodeCurrency(doc, default)

La fonction renvoie le texte du nœud courant converti en type monétaire.


Paramètres d'entrée :

  • doc : le document XML.
  • default : la valeur par défaut.

Résultat :
La valeur de type monétaire (Currency) convertie à partir du texte du nœud courant.
Si le texte est vide ou absent alors : la valeur par défaut est retournée.

Génère une erreur si la conversion n'est pas possible.


Les valeurs/formats acceptées/és :

  • Format fixe.
  • Format scientifique.

Les valeurs ne sont pas sensitives à la casse de caractères.


Exemples :

local doc
doc := XML:New()
XML:StartNode(doc, "items")
  XML:AppendNode(doc, "item1", 123456.78)
  XML:AppendNode(doc, "item2", "123456.78")
  XML:AppendNode(doc, "item3", "1.2345678E5")
  XML:AppendNode(doc, "item4")
XML:EndNode(doc)
XML:First(doc)
XML:FirstChild(doc)
Print(XML:GetNodeCurrency(doc))
XML:Next(doc)
Print(XML:GetNodeCurrency(doc))
XML:Next(doc)
Print(XML:GetNodeCurrency(doc))
XML:Next(doc)
Print(XML:GetNodeCurrency(doc, 876543.21))
XML:First(doc)
Print(XML:PrettyFormat(doc))

Résultat :

123456,78
123456,78
123456,78
876543,21
<?xml version="1.0"?>
<items>
  <item1>1.23456780000000E+0005</item1>
  <item2>123456.78</item2>
  <item3>1.2345678E5</item3>
  <item4/>
</items>
