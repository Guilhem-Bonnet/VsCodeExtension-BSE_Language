Fonctions de XML

XML:AttrExists(doc, name)

La fonction vérifie si l'attribut du nœud courant existe.


Paramètres d'entrée :

  • doc : le document XML.
  • name : le nom de l'attribut.

Résultat :

  • "True" - l'attribut existe.
  • "False" - l'attribut n'existe pas.

Exemples :

local doc
doc := XML:New()
XML:StartNode(doc, "order")
  Print(XML:AttrExists(doc, "num"))
  XML:SetAttr(doc, "num", "N1234")
  Print(XML:AttrExists(doc, "num"))
  Print(XML:GetAttrStr(doc, "num"))
XML:EndNode(doc)
Print(XML:ToString(doc))

Résultat :

False
True
N1234
<?xml version="1.0"?>
<order num="N1234"/>
