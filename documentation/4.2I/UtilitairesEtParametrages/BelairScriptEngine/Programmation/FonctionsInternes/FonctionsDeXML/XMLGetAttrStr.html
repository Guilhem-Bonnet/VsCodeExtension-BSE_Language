Fonctions de XML

XML:GetAttrStr(doc, name, default)

La fonction renvoie la valeur d'attribut du nœud courant.
Si l'attribut n'existe pas, et si la valeur par défaut n'est pas spécifiée : alors, une erreur est générée.


Paramètres d'entrée :

  • doc : le document XML.
  • name : le nom d'attribut.
  • default : la valeur par défaut (optionnel).

Résultat :
La valeur (chaîne de caractères).


Exemples :

local doc
doc := XML:New()
XML:StartNode(doc, "order")
  XML:SetAttr(doc, "num", "N1234")
  Print(XML:GetAttrStr(doc, "num"))
  Print(XML:GetAttrStr(doc, "created_by", "Unknown"))
XML:EndNode(doc)
Print(XML:ToString(doc))

Résultat :

N1234
Unknown
<?xml version="1.0"?>
<order num="N1234"/>
