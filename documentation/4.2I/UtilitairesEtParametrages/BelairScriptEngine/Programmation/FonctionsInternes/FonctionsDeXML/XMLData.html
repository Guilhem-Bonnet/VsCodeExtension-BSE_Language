Fonctions de XML

XMLData(doc, name, Qry, Columns)

La fonction crée des balises XML correspondant à l'ensemble des données (DataSet) retournées par une requête SQL.


Paramètres d'entrée :

  • doc : le document XML.
  • name : le nom de balise pour chaque ligne de dataset.
  • Qry : l'objet SQL.
  • Columns (optionnel) : une chaîne de caractères contenant les noms de colonnes retenues séparés par "," (virgule) ou ";" (point-virgule).

Résultat :
Aucun.


Exemples :

local doc, qry
qry := SQL:New("SELECT * FROM [tiers] WHERE b_numtierS IN (1000, 1009)")
SQL:Open(qry)
doc := XML:New()
XML:StartNode(doc, "tiers")
  XML:Data(doc, "tier", qry, "b_numtiers;b_typtiers;b_numdpp;b_titre;b_rsociale")
XML:EndNode(doc)
return doc

Le document retourné :

<?xml version="1.0" encoding="utf-8"?>
<tiers>
  <tier>
    <b_numtiers>1000</b_numtiers>
    <b_typtiers>18</b_typtiers>
    <b_numdpp>1</b_numdpp>
    <b_titre>MR</b_titre>
    <b_rsociale>MARTIN Pierre</b_rsociale>
  </tier>
  <tier>
    <b_numtiers>1009</b_numtiers>
    <b_typtiers>1T</b_typtiers>
    <b_numdpp>0</b_numdpp>
    <b_titre>ETS</b_titre>
    <b_rsociale>USINE</b_rsociale>
  </tier>
</tiers>
