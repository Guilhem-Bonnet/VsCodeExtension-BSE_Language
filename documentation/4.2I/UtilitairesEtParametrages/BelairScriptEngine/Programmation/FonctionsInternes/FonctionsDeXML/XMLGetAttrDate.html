Fonctions de XML

XML:GetAttrDate(doc, name, default)

La fonction renvoie la valeur d'attribut du nœud courant convertie en type DateTime.
Si l'attribut n'existe pas, et si la valeur par défaut n'est pas spécifiée : alors, une erreur est générée.


Paramètres d'entrée :

  • doc : le document XML.
  • name : le nom d'attribut.
  • default : la valeur par défaut (optionnel).

Résultat :
La valeur (DateTime).


Exemples :

local doc, created, accessed
doc := XML:New()
XML:StartNode(doc, "order")
  XML:SetAttr(doc, "created", EncodeDate(2017, 11, 7))
  created := XML:GetAttrDate(doc, "created")
  accessed := XML:GetAttrDate(doc, "accessed", Today())
XML:EndNode(doc)
Print(DayOf(created))
Print(MonthOf(created))
Print(YearOf(created))
Print(accessed)
Print(XML:ToString(doc))

Résultat :

7
11
2017
08/11/2017
