Fonctions de XML

XML:Open(content)

La fonction crée un objet de type "document XML" à partir d'XML spécifiés dans une chaîne de caractères.


Paramètres d'entrée :

  • content : la chaîne de caractères contenant l'XML.

Résultat :
Un objet de type "document XML".


Exemples :
local doc, s
s := "<colors><color>red</color><color>green</color><color>blue</color></colors>" 
doc := XML:Open(s)
Print(XML:PrettyFormat(doc))

Résultat :

<colors>
  <color>red</color>
  <color>green</color>
  <color>blue</color>
</colors>
