Fonctions de XML

XML:SaveToFile(doc, filename)

La fonction enregistre le document XML dans un fichier.


Paramètres d'entrée :

  • doc : le document XML.
  • filename : le nom du fichier.

Résultat :
Aucun.


Exemples :
local doc
doc := XML:New()
XML:StartNode(doc, "colors")
  XML:AppendNode(doc, "color", "red")
  XML:AppendNode(doc, "color", "green")
  XML:AppendNode(doc, "color", "blue")
XML:EndNode(doc)
XML:SaveToFile(doc, "test.xml")

Le fichier créé :

<?xml version="1.0" encoding="utf-8"?>
<colors><color>red</color><color>green</color><color>blue</color></colors>
