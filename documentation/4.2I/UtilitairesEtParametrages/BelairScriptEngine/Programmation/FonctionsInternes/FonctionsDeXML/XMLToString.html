Fonctions de XML

XML:ToString(doc)

La fonction renvoie le document XML sous forme de chaîne de caractères.


Paramètres d'entrée :

  • doc : le document XML.

Résultat :
La chaîne de caractères contenant l'XML.


Exemples :
local doc, s
doc := XML:New()
XML:StartNode(doc, "colors")
  XML:AppendNode(doc, "color", "red")
  XML:AppendNode(doc, "color", "green")
  XML:AppendNode(doc, "color", "blue")
XML:EndNode(doc)
s := XML:ToString(doc)
Print(s)

Résultat :

<?xml version="1.0"?>
<colors><color>red</color><color>green</color><color>blue</color></colors>
