Fonctions de XML

XML:ClearFormat(doc, type)

La fonction remet à zéro le formatage des valeurs textes pour un type donné.
La formatage dépendant de "local" est remis à défaut, selon "local".


Paramètres d'entrée :

  • doc : le document XML.
  • type : le nom du type (string) :
    • "date" ;
    • "float".

    • Résultat :
    Aucun.


    Exemples :

    local doc, s
doc :=  XML:New()
XML:StartNode(doc, "values")
  XML:SetFormat(doc, "float", "# ##0.0000")
  XML:SetFormatProperty(doc, "float", "DecimalSeparator", ",")
  XML:SetFormatProperty(doc, "float", "ThousandSeparator", " ")
  XML:AppendNode(doc, "qty1", 1123.456)
  XML:ClearFormat(doc, "float")
  XML:AppendNode(doc, "qty2", 1123.456)
XML:EndNode(doc)
Print(XML:PrettyFormat(doc))

    Résultat :

    <?xml version="1.0"?>
<values>
  <qty1>1 123,4560</qty1>
  <qty2>1.12345600000000E+0003</qty2>
</values>
