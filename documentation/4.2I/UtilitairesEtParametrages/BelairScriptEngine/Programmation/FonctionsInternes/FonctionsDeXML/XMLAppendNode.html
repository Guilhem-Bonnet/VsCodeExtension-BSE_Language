Fonctions de XML

XML:AppendNode(doc, name, value)

La fonction ajoute un nœud enfant sans changer le nœud courant.


Paramètres d'entrée :

  • doc : le document XML.
  • name : le nom du noeud.
  • value : la valeur du noeud (optionnel).

Résultat :
Aucun.


Exemples :

local doc
doc := XML:New()
XML:StartNode(doc, "cars")
  Print(XML:GetNodeName(doc))
  XML:AppendNode(doc, "toyota", "hybrid")
  Print(XML:GetNodeName(doc))
XML:EndNode(doc)
Print(XML:PrettyFormat(doc))

Résultat :

cars
cars
<?xml version="1.0"?>
<cars>
  <toyota>hybrid</toyota>
</cars>
