Fonctions de XML

XML:LoadFromFile(filename)

La fonction crée et initialise le document XML à partir d'un fichier XML.


Paramètres d'entrée :

  • filename : le nom du fichier.

Résultat :
Le document XML.


Exemples :
local doc1, doc2
doc1 := XML:New()
XML:StartNode(doc1, "colors")
  XML:AppendNode(doc1, "color", "red")
  XML:AppendNode(doc1, "color", "green")
  XML:AppendNode(doc1, "color", "blue")
XML:EndNode(doc1)
XML:SaveToFile(doc1, "test.xml")
doc2 := XML:LoadFromFile("test.xml")
Print(XML:PrettyFormat(doc2))

Résultat :

<?xml version="1.0"?>
<colors>
  <color>red</color>
  <color>green</color>
  <color>blue</color>
</colors>
