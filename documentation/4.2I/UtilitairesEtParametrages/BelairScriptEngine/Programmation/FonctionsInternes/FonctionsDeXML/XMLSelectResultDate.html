Fonctions de XML

XML:SelectResultDate(doc, Xpath, default)

La fonction sélectionne le nœud comme XML:SelectResult, puis le convertie en type date et temps comme XML:GetNodeDate.

Paramètres d'entrée :

  • doc : le document XML.
  • Xpath : la requête XPath.
  • default : la valeur par défaut, si la requête ne trouve rien (optionnel).

Exemples :
local doc
doc := XML:New()
XML:StartNode(doc, "items")
  XML:AppendNode(doc, "item1", EncodeDateTime(2017, 11, 9, 14, 35, 48))
  XML:AppendNode(doc, "item2", "2017-11-09T14:35:48.123")
  XML:AppendNode(doc, "item3")
XML:EndNode(doc)
Print(XML:SelectResultDate(doc, "/items/item1"))
Print(XML:SelectResultDate(doc, "/items/item2"))
Print(XML:SelectResultDate(doc, "/items/item3", EncodeDate(2017, 11, 9)))
Print(XML:PrettyFormat(doc))

Résultat :

09/11/2017 14:35:48
09/11/2017 14:35:48
09/11/2017
<?xml version="1.0"?>
<items>
  <item1>2017-11-09T14:35:48.000</item1>
  <item2>2017-11-09T14:35:48.123</item2>
  <item3/>
</items>
