Fonctions de XML

XML:GetNodeName(doc)

La fonction renvoie le nom du nœud courant.


Paramètres d'entrée :

  • doc : le document XML.

Résultat :
Le nom du nœud courant.


Exemples :
local doc
doc := XML:New()
XML:StartNode(doc, "cars")
  Print(XML:GetNodeName(doc) + ": " + XML:GetNodeText(doc))
  XML:StartNode(doc, "toyota")
    XML:SetNodeText(doc, "Prius")
    Print(XML:GetNodeName(doc) + ": " + XML:GetNodeText(doc))
  XML:EndNode(doc)
XML:EndNode(doc)

Résultat :

cars: 
toyota: Prius
