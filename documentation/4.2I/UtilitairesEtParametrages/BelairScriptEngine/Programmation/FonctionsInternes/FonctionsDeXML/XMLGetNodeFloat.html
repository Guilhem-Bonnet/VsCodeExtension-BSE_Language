Fonctions de XML

XML:GetNodeFloat(doc, default)

La fonction renvoie le texte du nœud courant converti en type numérique virgule flottante.


Paramètres d'entrée :

  • doc : le document XML.
  • default : la valeur par défaut.

Résultat :
La valeur de type numérique (double) convertie à partir du texte du nœud courant.
Si le texte est vide ou absent alors : la valeur par défaut est retournée.

Génère une erreur si la conversion n'est pas possible.


Les valeurs/formats acceptées/és

  • Format fixe.
  • Format scientifique.

Les valeurs ne sont pas sensitives à la casse de caractères.


Exemples :

local doc
doc := XML:New()
XML:StartNode(doc, "items")
  XML:AppendNode(doc, "item1", 1234.5678)
  XML:AppendNode(doc, "item2", "1234.5678")
  XML:AppendNode(doc, "item3", "1.2345678E3")
  XML:AppendNode(doc, "item4")
XML:EndNode(doc)
XML:First(doc)
XML:FirstChild(doc)
Print(XML:GetNodeFloat(doc))
XML:Next(doc)
Print(XML:GetNodeFloat(doc))
XML:Next(doc)
Print(XML:GetNodeFloat(doc))
XML:Next(doc)
Print(XML:GetNodeFloat(doc, 8765.4321))
XML:First(doc)
Print(XML:PrettyFormat(doc))

Résultat :

1234,5678
1234,5678
1234,5678
8765,4321
<?xml version="1.0"?>
<items>
  <item1>1.23456780000000E+0003</item1>
  <item2>1234.5678</item2>
  <item3>1.2345678E3</item3>
  <item4/>
</items>
