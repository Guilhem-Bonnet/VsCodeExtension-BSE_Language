Fonctions de XML

XMLAreEqual(doc1, doc2)

La fonction compare deux documents XML.
La fonction est sensitive à la casse de caractères de valeurs et de balises. Pour la comparaison non-sensitive : utiliser XML:AreSame.

Notez que :

Par défaut, les fonctions XML:StartNode et XML:AppendNode créent des balises en minuscule uniquement.


Paramètres d'entrée :

  • doc1 : le premier document XML.
  • doc2 : le deuxième document XML.

Résultat :

  • "True" - les documents sont égaux.
  • "False" - il y a une différence entre ces documents.

Exemples :

local doc1, doc2
doc1 := XML:New()
XML:StartNode(doc1, "items")
  XML:StartNode(doc1, "item1")
    XML:SetAttr(doc1, "attr1", "Attr1 Text")
    XML:SetNodeText(doc1, "Item1 Text")
  XML:EndNode(doc1)
  XML:AppendNode(doc1, "item2", "Item2 Text")
XML:EndNode(doc1)
doc2 := XML:Open("<items><item1 attr1=""Attr1 Text"">Item1 Text</item1><item2>Item2 Text</item2></items>")
Print(XML:AreEqual(doc1, doc2))
doc2 := XML:Open("<items><item1 attr1=""Attr1 Text Changed"">Item1 Text</item1><item2>Item2 Text</item2></items>")
Print(XML:AreEqual(doc1, doc2))

Résultat :

True
False
