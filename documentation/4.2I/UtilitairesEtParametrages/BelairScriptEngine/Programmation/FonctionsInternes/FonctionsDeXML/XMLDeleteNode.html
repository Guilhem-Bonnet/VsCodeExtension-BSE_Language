Fonctions de XML

XML:DeleteNode(doc)

La fonction supprime le nœud courant et le nœud parent devient le nouveau nœud courant.


Paramètres d'entrée :

  • doc : le document XML.

Résultat :
Aucun.


Exemples :

local doc

doc := XML:New()
XML:StartNode(doc, "cars")
  XML:StartNode(doc, "thermique")
    XML:AppendNode(doc, "renault", "Twingo")
    XML:AppendNode(doc, "toyota", "Aygo")
    XML:AppendNode(doc, "Mercedes", "AMG A")
  XML:EndNode(doc)  
  XML:StartNode(doc, "hybrid")
    XML:AppendNode(doc, "renault", "Clio E-Tech")
    XML:AppendNode(doc, "toyota", "Prius")
    XML:AppendNode(doc, "Mercedes", "GLE 350 E")
  XML:EndNode(doc)  
XML:EndNode(doc)

XML:Select(doc, "/cars/hybrid")
print(StrFormat("Current node before delete %s", XML:GetNodeName(doc)))
XML:DeleteNode(doc)
print(StrFormat("Current node after delete %s", XML:GetNodeName(doc)))
Print(XML:Prettyformat(doc))

return 0

Résultat :

Current node before delete hybrid
Current node after delete cars


  
    Twingo
    Aygo
    AMG A
