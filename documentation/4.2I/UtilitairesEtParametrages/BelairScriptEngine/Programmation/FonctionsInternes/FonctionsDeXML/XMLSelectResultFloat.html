Fonctions de XML

XML:SelectResultFloat(doc, Xpath, default)

La fonction sélectionne le nœud comme XML:SelectResult, puis le convertie en type numérique de virgule flottante comme XML:GetNodeFloat.

Paramètres d'entrée :

  • doc : le document XML.
  • Xpath : la requête XPath.
  • default : la valeur par défaut, si la requête ne trouve rien (optionnel).

Exemples :
local doc
doc := XML:New()
XML:StartNode(doc, "items")
  XML:AppendNode(doc, "item1", 1234.5678)
  XML:AppendNode(doc, "item2", "1.2345678E3")
  XML:AppendNode(doc, "item3")
XML:EndNode(doc)
Print(XML:SelectResultFloat(doc, "/items/item1"))
Print(XML:SelectResultFloat(doc, "/items/item2"))
Print(XML:SelectResultFloat(doc, "/items/item3", 8765.4321))
Print(XML:PrettyFormat(doc))

Résultat :

1234,5678
1234,5678
8765,4321
<?xml version="1.0"?>
<items>
  <item1>1.23456780000000E+0003</item1>
  <item2>1.2345678E3</item2>
  <item3/>
</items>
