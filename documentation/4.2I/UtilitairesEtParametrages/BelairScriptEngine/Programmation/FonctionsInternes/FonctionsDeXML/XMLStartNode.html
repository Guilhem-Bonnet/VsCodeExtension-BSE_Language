Fonctions de XML

XML:StartNode(doc, name)

La fonction ajoute un nœud enfant qui sera le nœud courant.


Notez que :

1) Sauf si vous utilisez XML:SetForceLowerCase(doc, false), le nom donné sera converti en minuscule.
2) Conseil : Utilisez les tiret-bas pour séparer les noms composites, i.e. "book_name" au lieu de "BookName", transformé en "bookname".


Paramètres d'entrée :

  • doc : le document XML.
  • name : le nom de nœud.

Résultat :
Aucun.


Exemples :
local doc
doc := XML:New()
XML:StartNode(doc, "cars")
  Print(XML:GetNodeName(doc))
  XML:StartNode(doc, "toyota")
    Print(XML:GetNodeName(doc))
  XML:EndNode(doc)
XML:EndNode(doc)

Résultat :

cars
toyota
