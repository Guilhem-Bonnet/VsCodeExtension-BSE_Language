Fonctions de XML

XML:SetAttr(doc, name, value)

La fonction crée ou affecte l'attribut existant par la valeur donnée.
Selon le type de valeur, elle sera convertie dans une chaîne de caractères au format approprié.


Paramètres d'entrée :

  • doc : le document XML.
  • name : le nom d'attribut.
  • value : la valeur.

Résultat :
Aucun.


Exemples :
local doc
doc := XML:New()
XML:StartNode(doc, "order")
  XML:SetAttr(doc, "date", EncodeDate(2017, 11, 07))
  XML:SetAttr(doc, "completed", true)
  XML:StartNode(doc, "item")
    XML:SetAttr(doc, "name", "Wheat flour")
    XML:SetAttr(doc, "price", 0.70)
    XML:SetAttr(doc, "amount", 150)
  XML:EndNode(doc)
XML:EndNode(doc)
Print(XML:ToString(doc))

Résultat :

<?xml version="1.0"?>
<order date="2017-11-07T00:00:00.000" completed="true"><item name="Wheat flour" price="7.00000000000000E-0001" amount="150"/></order>
