Fonctions de XML

XML:GetAttrBool(doc, name, default)

La fonction renvoie la valeur d'attribut du nœud courant, convertie en type booléen.
Si l'attribut n'existe pas, et si la valeur par défaut n'est pas spécifiée : alors, une erreur est générée.


Paramètres d'entrée :

  • doc : le document XML.
  • name : le nom d'attribut.
  • default : la valeur par défaut (optionnel).

Résultat :
La valeur (booléenne).


Exemples :

local doc
doc := XML:New()
XML:StartNode(doc, "order")
  XML:SetAttr(doc, "completed", true)
  if XML:GetAttrBool(doc, "completed") then
    Print("Completed")
  else
    Print("Not yet")
  Print(XML:GetAttrBool(doc, "modified", false))
XML:EndNode(doc)
Print(XML:ToString(doc))

Résultat :

Completed
False
<?xml version="1.0"?>
<order completed="true"/>
