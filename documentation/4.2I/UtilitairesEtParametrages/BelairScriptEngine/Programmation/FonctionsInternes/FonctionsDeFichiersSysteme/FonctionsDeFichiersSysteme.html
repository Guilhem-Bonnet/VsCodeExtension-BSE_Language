Fonctions de fichiers liées au système d'exploitation

FileCopy(source, dest) ou CopyFile(source, dest)

Copie d'un fichier sur un autre fichier

Paramètres :

source : fichier à copier
dest : fichier à créer par cette copie

FileDelete(filename)

Suppression d'un fichier existant

Paramètres :

filename : fichier à supprimer

FileExists(filename)

Retourne True ou False selon l'existence d'un fichier

Paramètres :

filename : nom du fichier

FileExtractName(filename)

Retourne le nom du fichier sans le chemin d'accès

Paramètres :

filename : nom du fichier

Exemple :

FileExtractName("C:\windows\temp\monfichier.txt")
Résultat : "monfichier.txt"

FileExtractPath(filename)

Retourne le chemin d'accès d'un fichier

Paramètres :

filename : nom du fichier

Exemple :

FileExtractPath("C:\windows\temp\monfichier.txt")
Résultat : "C:\windows\temp"

FileGetTempName(filetype)

Retourne le nom d'un fichier temporaire (aléatoire)

Paramètres :

filetype : type de fichier

Exemple :

FileGetTempName("txt")
Résultat : "12064996274399.txt"

FileRename(filename, newfilename) ou RenameFile(source, dest)

Renomme ou déplace un fichier

Paramètres :

filename : fichier à renommer ou déplacer
newfilename : nouveau nom

PathAddDelim(path)

Ajoute si nécessaire le "\" final à un nom de répertoire

Paramètres :

path : nom du répertoire

Exemple :

PathAddDelim(""C:\windows\temp")
Résultat : ""C:\windows\temp\"

PathDelete(path)

DANGER !!!

Effacement d'un répertoire et de tout son contenu

Paramètres :

path : nom du répertoire

PathDelim()

Retourne le caractère sépaarateur des noms de répertoire

PathExists(path)

Retourne True ou False selon l'existence d'un répertoire

Paramètres :

path : nom complet du répertoire

PathExpand(baipath)

Transforme un nom de répertoire "belair" en nom Windows

Paramètres :

baipath : nom du répertoire

Exemples :

PathExpand("[APPLIC]automate")
Résultat : "D:\belair42\APPDEI\automate"

PathExpand("[SCANNER]")
Résultat : "D:\belair42\APPDEI\SCANNER\"

PathForceDirs(path)

Force la création des répertoires jusqu'à un répertoire donné

Paramètres :

path : nom complet du répertoire final
