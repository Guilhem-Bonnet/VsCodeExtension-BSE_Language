Expressions régulières

RegEx:SetText(regExObject, chaine)

La fonction permet de modifié la chaîne cible d'un objet de type 'RegEx'.

Paramètres :

regExObject : Un objet de type 'RegEx'
chaine : la nouvelle chaine cible (string)

Résultat: aucun

Exemple :


local regExObject 
regExObject := RegEx:New()
RegEx:SetText(regExObject, "Some people, when confronted with a problem, think I know, I'll use regular expressions. Now they have two problems.")
return 0

Résultat : 

Script result:
0
