Expressions régulières

RegEx:Search(regExObject, expression)

La fonction tente de faire correspondre une expression régulière à n'importe quelle partie d'une séquence de caractères

Paramètres :

regExObject : Un objet de type 'RegEx'
expression : Une expression régulière (string)

Résultat: Une variable de type boolean (True - Il y a au moins un match; False sinon)

  DisplayResult(regExObject)
if (not RegEx:Search(regExObject,  "(\w{12,})")) then
  print("Regex (\w{12,}) not found")
