Expressions régulières

RegEx:New(chaine)

La fonction crée un objet de type 'RegEx' et le renvoie.

Paramètre :

chaine : Un objet de type 'RegEx'
expression : la chaîne cible (string)

Résultat: Un objet de type 'RegEx'

Exemple :


local regExObject, regExObject2 
regExObject := RegEx:New("Some people, when confronted with a problem, think I know, I'll use regular expressions. Now they have two problems.")
regExObject2 := RegEx:New()
return 0
