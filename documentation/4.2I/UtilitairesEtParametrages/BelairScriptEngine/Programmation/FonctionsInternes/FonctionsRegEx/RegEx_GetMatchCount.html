Expressions régulières

RegEx:GetMatchCount(regExObject)

La fonction retourne le nombre de match trouvés lors de la dernière recherche via les fonctions RegEx_GetMatchIterator, RegEx_Match et RegEx_Search

Paramètre :

regExObject : Un objet de type 'RegEx'

Résulta: une variable de type Integer

Exemple :


local regExObject, 

regExObject := RegEx:New("Some people, when confronted with a problem, think I know, I'll use regular expressions. Now they have two problems.")
RegEx:Search(regExObject,  ", (.)")
return RegEx:GetMatchCount(regExObject)

Résultat : 

Script result:
3
