Expressions régulières

RegEx:Replace(regExObject, expression, chaine)

La fonction remplace les occurrences d'une expression régulière par un texte de remplacement formaté et retourne le texte modifié.

Paramètre :

regExObject : Un objet de type 'RegEx'
expression : Une expression régulière (string)
chaine : Le texte de remplacement formaté (string)

Résultat: Une variable de type string

Exemple :


local regExObject, result 
regExObject := RegEx:New("Some people, when confronted with a problem, think I know, I'll use regular expressions. Now they have two problems.")
result := RegEx:Replace(regExObject, "(\w{7,})", "[$&]")     
return result

Résultat : 

Script result:
Some people, when [confronted] with a [problem], think I know, I'll use [regular] [expressions]. Now they have two [problems].
