Expressions régulières

RegEx:Match(regExObject, expression)

La fonction tente de faire correspondre une expression régulière à une séquence entière de caractères.

Paramètre :

regExObject : Un objet de type 'RegEx'
expression : Une expression régulière (string)

Résultat: une variable de type boolean (True - si une correspondance existe; False - sion)

Exemple :


local regExObject, 

regExObject := RegEx:New("Some people, when confronted with a problem, think I know, I'll use regular expressions. Now they have two problems.")
if (RegEx:Match(regExObject,  "(Some)(.*)")) then
  DisplayResult(regExObject)
if (not RegEx:Match(regExObject,  "(Some Test)(.*)")) then
  print("Regex (Some Test)(.*) not found")                    
return 0
