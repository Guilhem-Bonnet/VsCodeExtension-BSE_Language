Expressions régulières

RegEx:GetSubMatch(regExObject, index)

La fonction retourne sous forme de tableau les sous match trouvés lors de la dernière recherche via les fonctions RegEx_GetMatchIterator, RegEx_Match et RegEx_Search pour un Match donnée.

Paramètre :

regExObject : Un objet de type 'RegEx'
index : L'index d'un match (integer)

Résultat: Une variable de type Array

Exemple :


local regExObject, 

regExObject := RegEx:New("Some people, when confronted with a problem, think I know, I'll use regular expressions. Now they have two problems.")
if (RegEx:Search(regExObject,  ", (.)")) then
  DisplayResult(regExObject) 
return
