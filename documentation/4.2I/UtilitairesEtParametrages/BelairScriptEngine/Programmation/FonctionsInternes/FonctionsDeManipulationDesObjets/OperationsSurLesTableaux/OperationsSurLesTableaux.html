Fonctions de manipulation des objets


Opérations sur les tableaux


Les tableaux dynamiques font la partie de langage BSE.
De plus, comme les tableaux sont aussi des objets, il est possible de les manipuler par les fonctions "Obj:Get/SetArrayItems" mentionnées ci-dessous.


Exemples

Accès direct aux éléments 
local nums, i
nums := Obj:NewArray()
nums[0] := 11
nums[1] := "String 22" 
nums[2] := 33.44
i := 0
while i < Length(nums) begin
  Print("nums[%s]: %s", i, nums[i])
  i := i + 1
end
SetLength(nums, 2) // resize array to 2
Print(DumpVar(nums))

Résultat :

nums[0]: 11
nums[1]: String 22
nums[2]: 33,44
Array length: 2
[0]: 11
[1]: String 22


Accès aux éléments en utilisant des fonctions 
local nums, i
nums := Obj:NewArray()
Obj:AddArrayItems(nums, 11) // inserted to back
Obj:AddArrayItems(nums, "String 22")
Obj:SetArrayItems(nums, 2, 33.44) // resized and set
i := 0
while i < Length(nums) begin
  Print("nums[%s]: %s", i, Obj:GetArrayItems(nums, i))
  i := i + 1
end
Obj:DelArrayItems(nums, 0) // Delete first array item
Print(DumpVar(nums))

Résultat :

nums[0]: 11
nums[1]: String 22
nums[2]: 33,44
Array length: 2
[0]: String 22
[1]: 33,44
