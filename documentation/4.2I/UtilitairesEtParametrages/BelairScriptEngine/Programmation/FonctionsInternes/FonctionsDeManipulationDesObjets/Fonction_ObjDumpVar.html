Fonctions de manipulation des objets

Obj:DumpVar (synonyme de VarDump)

La fonction renvoie les types et les valeurs des propriétés d'un objet sous forme de chaîne de caractères séparés par CRLF.


Paramètres d'entrée :

  • 1. Une variable de type scalaire, de tableau ou d'objet.


Résultat :
La chaîne de caractères contenant le dump.

Notez que :

La valeur "Unknown" correspond en général à un autre objet que vous pouvez récupérer dans une variable que vous pouvez ensuite explorer par un autre dump.



Exemples :

local entity
entity := Meta:GetEntityByCode("Adh")
Print("---")
Print(Obj:DumpVar(entity))
