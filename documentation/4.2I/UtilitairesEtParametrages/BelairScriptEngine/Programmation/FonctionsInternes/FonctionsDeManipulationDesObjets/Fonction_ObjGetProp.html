Fonctions de manipulation des objets

Obj:GetProp(object, property)

Récupération de la valeur d'une propriété d'un objet

Paramètres d'entrée :

Object : objet contenant la propriété
property : nom de la propriété

Si la propriété n'existe pas, une erreur est retournée: voir Obj:GetPropDef(object, property, default)

Obj:GetPropDef(object, property, default)

Récupération de la valeur d'une propriété d'un objet, ou default si propriété inexistante

Paramètres d'entrée :

Object : objet contenant la propriété
property : nom de la propriété
default : valeur retournée si la propriété n'existe pas
