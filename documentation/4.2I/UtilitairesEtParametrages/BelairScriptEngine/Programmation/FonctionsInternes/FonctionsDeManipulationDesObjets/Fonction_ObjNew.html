Fonctions de manipulation des objets

Obj:New

La fonction crée une instance d'entité sans l'enregistrer dans la BDD. Afin de l'enregistrer : utilisez la fonction Obj:Save().


Paramètres d'entrée :

  • 1. Le type d'entité.


Résultat :
Un objet métier ayant toutes les propriétés (champs) de l'entité non initialisées.


Exemples :

local adh, adh2
adh := Obj:New("Adh")
adh.Contrat := 1
adh.Piece := 3
adh.Adhesion := 3
adh.Entree := EncodeDate(2010, 5, 10)
adh.Sortie := Null()
adh.Risklie := True
adh.Prime := 123456.78
adh.Prime1 := "EUR" 
adh.Tiers := 1000
Print("Adhésions créé")
Print("--------------")
Print(DumpVar(adh))
Obj:Save(adh)
adh2 := Obj:GetByKey("Adh", adh.Contrat, adh.Piece, adh.Adhesion)
Print("Adhésions récupéré")
Print("--------------")
Print(DumpVar(adh2))
