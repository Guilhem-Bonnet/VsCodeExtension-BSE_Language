Fonctions de manipulation des objets


Opérations sur les objets métiers (Business layer)


Vue d'ensemble


Les objets métiers sont définis dans le méta-modèle (entités et champs). Ils sont projetés sur la BDD relationnelle (tables et colonnes).
Dans un script BSE il est possible de manipuler les objets sans les opérations SQL CRUD.
Les fonctions de manipulations CRUD, ainsi que l'extension de langage BSE LINQ (homologue de LINQ de Microsoft), servent à manipuler les objets persistants.

De plus, il est possible de projeter le résultat d'une requête SQL sur l'ensemble des objets.


Limitations


  • Les valeurs correspondant au type BLOB de champ d'entité ne sont pas supportées.
  • La modification de valeur de la clé primaire n'est pas prise en compte.
    Afin de modifier la clé primaire (cas rare) : supprimez puis récréez l'objet.


Opérations CRUD


Récupération des entités

  • Obj:GetByKey.
  • Utiliser LINQ.
  • Obj:SelectSQL : non recommandé, à utiliser avec précaution.

Création des entités

  • Obj:New.

Modification des entités

  • Obj:Save.

Suppression des entités

  • Obj:Delete.

Sérialisation

  • Obj:LoadFromXML.
  • Obj:SaveToXML.
