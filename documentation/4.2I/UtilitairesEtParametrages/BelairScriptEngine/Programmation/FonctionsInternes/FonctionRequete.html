Fonctions Requête


Paramètres

  • Nom de la requête : string (Obligatoire)
  • La liste des paramètres et leur valeur : string (Facultatif)
    • Format -NomDuParamétres :LaValeur ….

Explication du nouveau comportement

Avec ce nouveau comportement il n’y a plus de fenêtre de saisie pour les paramètres. Le seul moyen de définir les valeurs pour la requête est le deuxième paramètre de la fonction. Si un état doit sortir alors le fichier sera save en PDF dans le dossier de travail. Pour récupère le chemin utilisé la fonction PDFFILE.

Exemples :

Requete("DS01", "-CDPFIN:01290")
Print(PDFFILE)

Requete("DS01")
Print(PDFFILE)

Requete("DS01", "")
Print(PDFFILE)
