Fonction Alphanumérique

Fonction Chr(n)

Cette fonction permet de générer un caractère à partir d'un code.
Fonction inverse de Ord()


Voici un exemple : 

local testTab, testTilde
testTab := "Horizontal tab" + Chr(9) + "test"                
testTilde := "Tilde" + Chr(126) + "test" 
Assert(testTab = "Horizontal tab    test", "Tab Caracter not good")
Assert(testTilde = "Tilde~test", "Tilde Caracter not good")
return 0
