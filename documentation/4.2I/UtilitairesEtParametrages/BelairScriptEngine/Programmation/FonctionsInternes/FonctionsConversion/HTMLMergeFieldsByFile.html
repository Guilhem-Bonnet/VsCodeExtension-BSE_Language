Fonctions de HTML

HTML:MergeFieldsByFile(html, ObjBSE)

La fonction exécute la fusion d'un document HTML puis retourne le résultat.


Paramètres d'entrée :

  • Html : Un chaine qui représente le chemin vers le fichier HTML.
  • ObjBSE : Un objet qui représente le champs a fusionner

Note: Le fichier dois être encodé en Unicode (UTF8 ou UTF16)


Résultat :

  • Le chemin vers un fichier temporaire html

    • Si le programme retourne une erreur, celle ci sera retournée par la fonction.


    Exemple :

    local f1, obj, htmlText, htmlFile

htmlText := "<HTML><BODY><P>Test Merge by File <SPAN style=""mso-no-proof: yes"">Field1</SPAN><SPAN style=""mso-no-proof: yes"">Field2</SPAN></P></BODY></HTML>" 
f1 := File:Create("Test.html", "utf16")
File:WriteLine(f1, htmlText)
File:Close(f1)

Obj := Obj:New()
Obj:SetProp(obj, "Field1", "Unit")
Obj:SetProp(obj, "Field2", "Test") 

htmlFile :=  HTML:MergeFieldsByFile("Test.html", obj)

return htmlFile

    Résultat : 

    Script result:
C:\Users\test\Documents\BelAir\16393651270399.html
