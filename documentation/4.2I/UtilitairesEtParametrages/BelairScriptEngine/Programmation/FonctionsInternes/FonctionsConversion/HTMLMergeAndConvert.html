Fonctions de HTML

HTML:MergeAndConvert(html, ObjBSE, headerfile, footerfile, options)

La fonction permet de fusionner une chaine qui représente le contenu d'un HTML, le convertir en PDF puis retourne le résultat.


Paramètres d'entrée :

  • Html : Une chaine qui représente le contenu d'un HTML.
  • ObjBSE : Un objet qui représente le champs a fusionner
  • Headerfile : Chemin vers un fichier HTML représentant l'entête (optionnel, string).
  • Footerfile : Chemin vers un fichier HTML représentant le pied de page (optionnel, string).
  • Options : Options pour l'Edition du PDF (optionnel, string).

Note: Le fichier dois être encodé en Unicode (UTF8 ou UTF16)


Résultat :

  • Le chemin vers le fichier PDF

    • Si le programme retourne une erreur, celle ci sera retournée par la fonction.


    Exemple :

    local f1, obj, htmlText, pdfFile
htmlText := "
    Test Convert HTML TO PDF
    "
f1 := File:Create("Test.html", "utf16")
File:WriteLine(f1, htmlText)
File:Close(f1)
Obj := Obj:New()
Obj:SetProp(obj, "Field1", "Unit")
Obj:SetProp(obj, "Field2", "Test")
pdfFile :=  HTML:MergeAndConvert("Test.html", obj)
return pdfFile

    Résultat : 

    Script result:
C:\Users\test\Documents\BelAir\16592462253930.pdf
