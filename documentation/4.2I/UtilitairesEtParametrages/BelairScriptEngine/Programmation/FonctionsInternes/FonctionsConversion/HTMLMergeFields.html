Fonctions de HTML

HTML:MergeFields(html, ObjBSE, options)

La fonction exécute la fusion d'une chaine qui représente le contenu d'un HTML puis retourne le résultat.


Paramètres d'entrée :

  • Html : Une chaine qui représente le contenu d'un HTML.
  • ObjBSE : Un objet qui représente le champs a fusionner
  • Options : Une chaîne qui permet de récupérer le HTML avec la balsie HTML et BODY : Valeur : Outer ; ou bien justement le contenu du BODY (optionnel, string) : Valeur : Inner

Note: Par défaut le retour représente le Outer


Résultat :

  • Le texte HTML fusionné

    • Si le programme retourne une erreur, celle ci sera retournée par la fonction.


    Exemple :

    local obj, htmlText

htmlText := "<HTML><BODY><P>Test Merge by Text <SPAN style=""mso-no-proof: yes"">Field1</SPAN><SPAN style=""mso-no-proof: yes"">Field2</SPAN></P></BODY></HTML>"
Obj := Obj:New()
Obj:SetProp(Obj, "Field1", "Unit")
Obj:SetProp(Obj, "Field2", "Test")
htmlText :=  HTML:MergeFields(htmlText, obj, "inner")
return htmlText

    Résultat : 

    Script result:
<P>Test Merge by Text <SPAN style="mso-no-proof: yes">Unit</SPAN><SPAN style="mso-no-proof: yes">Test</SPAN></P>
