Fonctions de HTML

HTML:ConvertFileToPdf(html, headerfile, footerfile, options)

La fonction permet de convertir une chaine qui représente le contenu d'un HTML vers un fichier PDF puis retourne le résultat.


Paramètres d'entrée :

  • Html : Une chaine qui représente le contenu d'un HTML.
  • Headerfile : Chemin vers un fichier HTML représentant l'entête (optionnel, string).
  • Footerfile : Chemin vers un fichier HTML représentant le pied de page (optionnel, string).
  • Options : Options pour l'Edition du PDF (optionnel, string).

Note: Le fichier doit être encodé en Unicode (UTF8 ou UTF16)


Résultat :

  • Le chemin vers le fichier PDF

    • Si le programme retourne une erreur, celle ci sera retournée par la fonction.


    Exemple :

    local f1, htmlText, pdfFile

htmlText :=  "<HTML><BODY>Test Convert HTML TO PDF by Text</BODY></HTML>"
f1 := File:Create("Test.html", "utf16")
File:WriteLine(f1, htmlText)
File:Close(f1)

pdfFile :=  HTML:ConvertFileToPdf("Test.html", "", "")

return pdfFile

    Résultat : 

    Script result:
C:\Users\test\Documents\BelAir\16460051091623.pdf
