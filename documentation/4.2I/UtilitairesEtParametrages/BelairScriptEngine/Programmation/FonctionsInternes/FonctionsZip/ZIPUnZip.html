Fonctions Zip

ZIP:UnZip(param1, param2)

La fonction permet de dézipper un fichier zip.
Elle retourne la liste des fichier dézipper


Paramètres d'entrée :

  • param1 : Le chemin d'accès à l'archive à extraire
  • param2 : Le chemin d'accès au répertoire dans lequel placer les fichiers extraits
Résultat

Un tableau constitué de chaîne de caractères.


Exemple : 

local file1, file2, filesList
PathForceDirs("c:\test\")
PathForceDirs("c:\test\test2\")
file1 := File:Create("c:\test\zipTest1.txt")
File:Write(file1, "BELAIR Informatique")
File:Close(file1)
file1 := File:Create("c:\test\test2\zipTest2.txt")
File:Write(file1, "BELAIR Informatique")
File:Close(file1)
Zip:AddTo("c:\test\zipTest1.txt", "c:\test\ziptest.zip")
Zip:AddTo("c:\test\test2\zipTest2.txt", "c:\test\ziptest.zip")   
filesList := Zip:UnZip("c:\test\ziptest.zip", "c:\testUnzip")       
return DumpVar(filesList)

Résultat : 

Array length: 2
  [0]: c:\testUnzip\test\zipTest1.txt
  [1]: c:\testUnzip\test\test2\zipTest2.txt
