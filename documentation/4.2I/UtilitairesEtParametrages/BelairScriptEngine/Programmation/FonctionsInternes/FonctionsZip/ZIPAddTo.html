Fonctions Zip

ZIP:AddTo(param1, param2)

La fonction permet d'ajouter un dossier ou un fichier au fichier zip.
Note :

  • Si le fichier zip existe déjà alors il ajoute sinon il crée le fichier.

Paramètres d'entrée :

  • param1 : Le chemin d'accès au répertoire/ Fichier à archiver
  • param2 : Le chemin de l'archive à créer

Résultat
  • true - Si la fonction a réussi
  • false - Si une erreur est survenue

Exemple : 

local file1, file2
PathForceDirs("c:\test\")
PathForceDirs("c:\test2\")
file1 := File:Create("c:\test\zipTest1.txt")
File:Close(file1)
file2 := File:Create("c:\test2\zipTest2.txt")
File:Close(file2)
Zip:AddTo("c:\test2", "c:\test\ziptest.zip")
Zip:AddTo("c:\test\zipTest1.txt", "c:\test\ziptest.zip")
return 0

Résultat : 

Script result:
0
