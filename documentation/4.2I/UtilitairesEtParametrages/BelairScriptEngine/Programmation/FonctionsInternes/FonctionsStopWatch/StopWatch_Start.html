Fonctions de StopWatch

StopWatch:Start(mockStopWatch)

La fonction démarre la mesure du temps écoulé pour un intervalle.


Paramètres d'entrée :

  • mockStopWatch : Un objet de type 'BaiStopWatch'

Résultat :

  • Aucun

Exemple : 

local mockStopWatch
mockStopWatch := StopWatch:New()
StopWatch:Start(mockStopWatch)
if (StopWatch:IsRunning(mockStopWatch)) then
  Print("BaiStopWatch was started")
else
  Error("BaiStopWatch not started")
return 0

Résultat : 

BaiStopWatch was started

0
