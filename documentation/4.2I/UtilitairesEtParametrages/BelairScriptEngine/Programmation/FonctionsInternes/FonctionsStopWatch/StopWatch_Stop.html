Fonctions de StopWatch

StopWatch:Stop(mockStopWatch)

La fonction cesse de mesurer le temps écoulé pour un intervalle puis retourne le temps écoulé en millisecondes.


Paramètres d'entrée :

  • mockStopWatch : Un objet de type 'BaiStopWatch'

Résultat :

  • Le temps écoulé

Exemple : 

local mockStopWatch, i
mockStopWatch := StopWatch:New()
StopWatch:Start(mockStopWatch)
i := 0
while i < 1000
  i := i + 1
i := StopWatch:Stop(mockStopWatch)
return StrFormat("Temps écoulé %sms", i)

Résultat : 

Script result:
Temps écoulé 3ms
