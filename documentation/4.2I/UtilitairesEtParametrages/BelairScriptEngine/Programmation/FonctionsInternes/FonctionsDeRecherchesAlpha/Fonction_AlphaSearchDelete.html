Fonctions de recherches alpha

AlphaSearchDelete

La fonction supprime l'index de recherches pour un objet.


Paramètres d'entrée :

  • Le type d'objet (code d'entité).
  • Les valeurs de clé (de type entier, jusqu'à 3 valeurs).


Résultat :
Le nombre d'éléments d'index supprimés.


Exemples :

Print(AlphaSearchDelete("TIERS", 1000))

Résultat :

6
