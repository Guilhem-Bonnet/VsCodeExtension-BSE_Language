Fonctions de recherches alpha

AlphaSearchUpdate

La fonction met à jour l'index de recherches pour un objet.

Attention :

Seuls les champs persistants sont pris en compte. Veuillez utiliser AlphaSearchUpdateField pour les champs calculés.

Paramètres d'entrée :

  • Le type d'objet (code d'entité).
  • Les valeurs de clé (de type entier, jusqu'à 3 valeurs).


Résultat :
Le nombre des éléments d'index affectés.


Exemples :

Print(AlphaSearchUpdate("TIERS", 1000))

Résultat :

6
