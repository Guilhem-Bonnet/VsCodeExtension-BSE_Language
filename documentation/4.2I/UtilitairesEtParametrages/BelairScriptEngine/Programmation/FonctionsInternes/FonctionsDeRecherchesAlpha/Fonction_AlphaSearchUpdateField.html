Fonctions de recherches alpha

AlphaSearchUpdateField

La fonction met à jour l'index de recherches pour le champ d'un objet.


Paramètres d'entrée :

  • 1. Le texte.
  • 2. Le type d'objet (code d'entité).
  • 3. Le nom de champ.
  • 4. Les valeurs de clé (de type entier, jusqu'à 3 valeurs).


Résultat :
Le nombre des éléments d'index affectés.


Exemples :

Print(AlphaSearchUpdateField("MARTIN Pierre", "DPP", "b_nompre", 1000, 1))

Résultat :

1
