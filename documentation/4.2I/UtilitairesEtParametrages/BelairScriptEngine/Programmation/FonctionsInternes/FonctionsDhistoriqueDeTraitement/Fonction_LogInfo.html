Fonctions d'historique de traitement (log)

LogInfo(message, arguments)

Ajoute un message d'information dans l'historique.


Paramètres d'entrée :

  • message (string) - le message d'erreur, ou la chaîne de format, comme dans la fonction Print.
  • arguments (optionnel) les arguments à formater.


    • Résultat :
    Aucun.


    Exemples :     

    LogInfo("Processing invoices")
LogInfo("Processing finished at %s", Now())
