Fonctions d'historique de traitement (log)

LogError(code, message, arguments)

Ajoute l'information sur une erreur dans l'historique.


Paramètres d'entrée :

  • code : état (int) - l'état personnalisé, i.e. le code d'erreur.
  • message (string) - le message d'erreur, ou la chaîne de format, comme dans la fonction Print.
  • arguments (optionnel) les arguments à formater.


Résultat :
Aucun.


Exemples : 

LogError(1, "Processing error")
LogError(111, "Range error. Min: %s, max: %s, value: %s", min, max, value)
