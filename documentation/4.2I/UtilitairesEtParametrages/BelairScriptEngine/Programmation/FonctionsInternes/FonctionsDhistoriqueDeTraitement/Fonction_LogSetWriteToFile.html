Fonctions d'historique de traitement (log)

LogSetWriteToFile(valeur)

Active l'écriture des informations dans le log système de Belair ou de BAS.
La valeur est "false" par défaut.


Paramètres d'entrée :

  • valeur (True/False) - activer ou déactiver l'écriture dans le log système.


Résultat :
Aucun.


Exemples : 

LogInfo("Test 1")
LogSetWriteToFile(true)
LogInfo("Hello from script")
LogSetWriteToFile(false)
LogInfo("Test 3")

Résultat :

Dans "Belair.log ou bas_XXX.log".

Bas:5420 [2018-03-20 18:34:48.427 Info] Hello from script
