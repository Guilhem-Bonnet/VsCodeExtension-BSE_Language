Fonctions d'historique de traitement (log)

LogSetMaxBuf(nombre)

Assigne le nombre maximal des enregistrements dans l'historique.
La valeur par défaut est : 25. Les nouveaux enregistrements poussent les anciens en ordre de queue. I.e. le 26ème appel de LogInfo, LogError ou LogWarning supprime le premier enregistrement, etc.


Paramètres d'entrée :

  • nombre (integer): nombre maximal des enregistrements dans l'historique.


Résultat :
Aucun.


Exemples : 

LogSetMaxBuf(2)
LogInfo("Test 1")
LogInfo("Test 2")
LogInfo("Test 3") // "Test 1" supprimé
