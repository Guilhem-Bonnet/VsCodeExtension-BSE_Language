Fonctions JSON

Json:NewArrayValue(doc, value)

La fonction affecte le texte converti de value au JSONArray courant.
Génère une erreur si le parent n'est pas un JSONArray.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'
  • value : ne variable de n'importe quel type primitif (booléen, currency, date, float, integer, int64, UnicodeString).
    La valeur NULL est acceptée

Résultat :
Aucun.


Exemple : 

local doc
doc := Json:New()
Json:NewArray(doc, "colors")
  Json:NewArrayValue(doc, "Red")
  Json:NewArrayValue(doc, "Green")
  Json:NewArrayValue(doc, "Blue")
Json:EndArray(doc)
Json:NewArray(doc, "Entier")
  Json:NewArrayValue(doc, 1)
  Json:NewArrayValue(doc, 2)
  Json:NewArrayValue(doc, 3)
Json:EndArray(doc)           
Json:Close(doc)  
return Json:PrettyFormat(doc)

Résultat : 

{
  "colors": [
    "Red",
    "Green",
    "Blue" 
  ],
  "Entier": [
    1,
    2,
    3
  ]
}
