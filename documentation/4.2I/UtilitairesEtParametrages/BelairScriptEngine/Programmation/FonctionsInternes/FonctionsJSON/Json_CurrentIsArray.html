Fonctions JSON

Json:CurrentIsArray(doc)

Indique si la valeur courante est du type JSONArray.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
La valeur de type booléenne True/False


Exemple : 

local doc
doc := Json:New()          
Json:NewArray(doc, "colors")
  Json:NewObject(doc)
    Json:NewValue(doc, "color", "red")
  Json:EndObject(doc)
Json:EndArray(doc)
Json:Close(doc)    
Json:ValueExists(doc, "colors")
print(Json:CurrentIsArray(doc))  
Json:ValueExist(doc, "colors[0].color")
print(Json:CurrentIsArray(doc))  
return 0

Résultat : 

True
False
