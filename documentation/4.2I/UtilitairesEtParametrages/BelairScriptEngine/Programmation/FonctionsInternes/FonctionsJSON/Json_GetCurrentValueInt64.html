Fonctions JSON

Json:GetCurrentValueInt64(doc)

La fonction renvoie le texte de la valeur courante converti en type entier. (64 bits)


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
Le texte de la valeur courante converti en type entier (int64).
Génère une erreur si la conversion n'est pas possible


Exemple : 

local doc, varTest
doc := Json:New()
Json:NewObject(doc, "object")
  Json:NewValue(doc, "int64", 1234567812345678)
Json:EndObject(doc)
Json:Close(doc)            
Json:ValueExist(doc, "object.int64")
varTest := Json:GetCurrentValueInt64(doc)
print(varTest)
print(VarTypeName(varTest))
return 0

Résultat : 

1234567812345678
Int64
