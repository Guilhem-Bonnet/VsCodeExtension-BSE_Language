Fonctions JSON

Json:GetCurrentValueDate(doc)

La fonction renvoie le texte de la valeur courante convertie en type date et temps.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
Le texte de la valeur courante converti en type date et temps
Génère une erreur si la conversion n'est pas possible


Exemple : 

local doc, varTest
doc := Json:New()
Json:NewObject(doc, "object")
  Json:NewValue(doc, "date", Now())
Json:EndObject(doc)
Json:Close(doc)            
Json:ValueExists(doc, "object.date")
varTest := Json:GetCurrentValueDate(doc)
print(varTest)
print(VarTypeName(varTest))
return 0

Résultat : 

16/09/2019 14:26:21
