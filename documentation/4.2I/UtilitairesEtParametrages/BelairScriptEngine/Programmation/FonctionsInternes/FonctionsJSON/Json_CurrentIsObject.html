Fonctions JSON

Json:CurrentIsObject(doc)

Indique si la valeur courante est du type JSONObject.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
La valeur de type booléenne True/False


Exemple : 

local doc
doc := Json:New()          
Json:NewObject(doc, "colors")
  Json:NewValue(doc, "color", "red")
Json:EndObject(doc)
Json:Close(doc)                             
Json:ValueExist(doc, "colors.color")
print(Json:CurrentIsObject(doc))  
Json:ValueExists(doc, "colors")
print(Json:CurrentIsObject(doc))            
return 0

Résultat : 

False
True
