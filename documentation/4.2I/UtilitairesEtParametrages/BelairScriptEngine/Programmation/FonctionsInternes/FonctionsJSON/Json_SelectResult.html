Fonctions JSON

Json:SelectResult(doc,path,default)

cette fonction cherche la valeur spécifiée par la requête JSONPath comme la fonction Json:ValueExists puis renvoie le texte de la valeur trouvée.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'
  • path : la requête JSONPath
  • default : la valeur par défaut si la requête ne trouve aucun résultat (optionnel)

Résultat :
le texte de la valeur trouvée.

Si la requête JSONPath ne trouve rien et la valeur par défaut n'est pas spécifiée, une erreur est générée.

Exemple : 

local doc
doc := Json:New()
Json:NewObject(doc, "Cars")
  Json:NewArray(doc, "Renault")
    Json:NewObject(doc)
      Json:NewValue(doc, "Model", "clio")
      Json:NewValue(doc, "Price", 10990)
      Json:NewValue(doc, "InStock", false)
    Json:EndObject(doc)
    Json:NewObject(doc)
      Json:NewValue(doc, "Model", "zoe")
      Json:NewValue(doc, "Price", 25700)
      Json:NewValue(doc, "InStock", true)
    Json:EndObject(doc)
  Json:EndArray(doc)
Json:EndObject(doc)
Json:Close(doc)            
print(Json:SelectResult(doc, "cars.renault[0].model"))
print(Json:SelectResult(doc, "cars.renault[1].Price"))
print(Json:SelectResult(doc, "NotAGoodPath", "Value not found"))   
return 0

Résultat : 

clio
25700
Value not found
