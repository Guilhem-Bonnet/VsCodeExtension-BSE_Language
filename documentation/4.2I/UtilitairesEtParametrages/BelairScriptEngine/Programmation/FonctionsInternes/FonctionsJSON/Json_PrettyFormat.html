Fonctions JSON

Json:PrettyFormat(doc)

cette fonction formate le texte d'un document JSON en format lisible par l'humain.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
Le texte (UnicodeString) contenant le JSON formaté


Exemple : 

local doc
doc := Json:New()
Json:NewArray(doc, "colors")
  Json:NewArrayValue(doc, "Red")
  Json:NewArrayValue(doc, "Green")
  Json:NewArrayValue(doc, "Blue")
Json:EndArray(doc)
Json:NewArray(doc, "Entier")
  Json:NewArrayValue(doc, 1)
  Json:NewArrayValue(doc, 2)
  Json:NewArrayValue(doc, 3)
Json:EndArray(doc)
Json:Close(doc)                                    
return Json:PrettyFormat(doc)

Résultat : 

{
  "colors": [
    "Red",
    "Green",
    "Blue" 
  ],
  "Entier": [
    1,
    2,
    3
  ]
}
