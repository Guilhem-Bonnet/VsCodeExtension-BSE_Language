Fonctions JSON

Json:ToString(doc)

cette fonction renvoie le document Json sous forme de chaîne de caractères.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
La chaîne de caractères contenant le JSON.


Exemple : 

local doc
doc := Json:New()
Json:NewObject(doc, "Cars")
  Json:NewArray(doc, "Renault")
    Json:NewObject(doc)
      Json:NewValue(doc, "Model", "zoe")
      Json:NewValue(doc, "Price", 25700.999)
      Json:NewValue(doc, "Currency", "EUR")
    Json:EndObject(doc)
  Json:EndArray(doc)
Json:EndObject(doc)
Json:Close(doc)            
return Json:ToString(doc)

Résultat : 

{"Cars":{"Renault":[{"Model":"zoe","Price":25700.999,"Currency":"EUR"}]}}
