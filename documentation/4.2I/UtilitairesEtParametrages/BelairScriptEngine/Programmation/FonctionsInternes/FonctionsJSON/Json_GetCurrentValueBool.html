Fonctions JSON

Json:GetCurrentValueBool(doc)

La fonction renvoie le texte de la valeur courante convertie en type booléen.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
Le texte de la valeur courante convertie en type booléen
Génère une erreur si la conversion n'est pas possible


Exemple : 

local doc
doc := Json:New()
Json:NewObject(doc, "colors")
  Json:NewValue(doc, "color1", True)
  Json:NewValue(doc, "color2", False)
Json:EndObject(doc)
Json:Close(doc)            
Json:ValueExist(doc, "colors.color1")
print(Json:GetCurrentValueBool(doc))
Json:Next(doc)
print(Json:GetCurrentValueBool(doc))
return 0

Résultat : 

True
False
