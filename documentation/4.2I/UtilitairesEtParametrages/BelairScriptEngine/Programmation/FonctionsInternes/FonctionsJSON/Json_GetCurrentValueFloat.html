Fonctions JSON

Json:GetCurrentValueFloat(doc)

La fonction renvoie le texte de la valeur courante convertie en type numérique virgule flottante.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
Le texte de la valeur courante converti en type numérique virgule flottante (double).
Génère une erreur si la conversion n'est pas possible


Exemple : 

local doc, varTest
doc := Json:New()
Json:NewObject(doc, "object")
  Json:NewValue(doc, "float", 1234.5678)
Json:EndObject(doc)
Json:Close(doc)            
Json:ValueExists(doc, "object.float")
varTest := Json:GetCurrentValueFloat(doc)
print(varTest)
print(VarTypeName(varTest))
return 0

Résultat : 

1234,5678
Double
