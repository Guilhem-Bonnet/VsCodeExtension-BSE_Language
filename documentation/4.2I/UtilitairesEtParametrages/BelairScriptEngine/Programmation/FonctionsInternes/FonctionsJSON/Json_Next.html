Fonctions JSON

Json:Next(doc, keyname)

La fonction rend courante la valeur suivant la valeur courante actuelle.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'
  • keyname : soit un numéro d'index de tableau, soit un nom de clé (optionnel)

Résultat :
Une valeur booléenne True/False. False si la valeur courante est la dernière


Exemple : 

local doc
doc := Json:New()
Json:NewObject(doc, "object")
  Json:NewValue(doc, "varInt", 12)
  Json:NewValue(doc, "varString", "Test")
  Json:NewValue(doc, "varBool", True)
  Json:NewValue(doc, "varDate", Now())
  Json:NewValue(doc, "varNull", NULL)
Json:EndObject(doc)
Json:Close(doc)  
Json:ValueExist(doc, "object")
Json:Recurse(doc)           
print(Json:GetCurrentPath(doc))
Json:Next(doc, "varBool")
while Json:Next(doc)
 print(Json:GetCurrentPath(doc))
return 0

Résultat : 

object.varInt
object.varDate
object.varNull
