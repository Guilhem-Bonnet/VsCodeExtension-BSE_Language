Fonctions JSON

Json:First(doc)

La fonction rend courant le JSONObject racine.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
La valeur booléeene True/False indiquant si le positionnement a réussi.


Exemple : 

local doc
doc := Json:New()
Json:NewObject(doc, "colors")
  Json:NewValue(doc, "color1", "Red")
  Json:NewValue(doc, "color2", "Green")
  Json:NewValue(doc, "color3", "Blue")
Json:EndObject(doc)
Json:Close(doc)            
Json:ValueExists(doc, "colors.color1")
print(Json:First(doc))
return 0

Résultat : 

True
