Fonctions JSON

Json:ValueExists(doc, JSONPath)

cette cherche la valeur spécifiée par la requête JSONPath : la valeur courante devient la valeur trouvée ou la premiere valeur de la collection trouvée.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'
  • JSONPath : la requête JSONPath

Résultat :
La valeur booléenne True/False en fonction du résultat de la recherche.

Si la requête ne trouve rien, le JSONObject racine deviendra la valeur courante.

Exemple : 

local doc
doc := Json:New()
Json:NewObject(doc, "Cars")
  Json:NewArray(doc, "Renault")
    Json:NewObject(doc)
      Json:NewValue(doc, "Model", "zoe")
      Json:NewValue(doc, "Price", 25700.999)
      Json:NewValue(doc, "Currency", "EUR")
    Json:EndObject(doc)
  Json:EndArray(doc)
Json:EndObject(doc)
Json:Close(doc)                
if Json:ValueExists(doc, "Cars.renault[0].Currency") then
 print(Json:GetCurrentPath(doc))                    
if Json:ValueExists(doc, "Cars.renault[0].Model") then
 print(Json:GetCurrentPath(doc))                   
if not Json:ValueExists(doc, "Test") then
 print("Root JSONObject")   
return 0

Résultat : 

Cars.Renault[0].Currency
Cars.Renault[0].Model
Root JSONObject
