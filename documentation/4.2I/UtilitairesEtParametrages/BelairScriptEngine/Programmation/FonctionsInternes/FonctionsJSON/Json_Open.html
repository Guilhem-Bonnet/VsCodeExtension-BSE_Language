Fonctions JSON

Json:open(string)

La fonction crée un objet de type 'document JSON' à partir du JSON spécifié dans une chaîne de caractères.


Paramètres d'entrée :

  • string : la chaîne de caractères contenant le JSON

Résultat :
Un objet de type 'document JSON'


Exemple : 

local doc, varTest
varTest := "{""object"":{""varInt"":12,""varString"":""Test"",""varBool"":true,""varDate"":""2019-09-16T17:44:51.624+02:00"",""varNull"":null}}" 
doc := Json:Open(varTest)
return Json:PrettyFormat(doc)

Résultat : 

{
  "object": {
    "varInt": 12,
    "varString": "Test",
    "varBool": true,
    "varDate": "2019-09-16T17:44:51.624+02:00",
    "varNull": null
  }
}
