Fonctions JSON

Json:NewValue(doc, value)

La fonction affecte le texte converti de value à la valeur courante.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'
  • value : ne variable de n'importe quel type primitif (booléen, currency, date, float, integer, int64, UnicodeString).
    La valeur NULL est acceptée

Résultat :
Aucun.


Exemple : 

local doc
doc := Json:New()
Json:NewObject(doc, "object")
  Json:NewValue(doc, "varInt", 12)
  Json:NewValue(doc, "varString", "Test")
  Json:NewValue(doc, "varBool", True)
  Json:NewValue(doc, "varDate", Now())
  Json:NewValue(doc, "varNull", NULL()) 
Json:EndObject(doc)
Json:Close(doc)            
return Json:PrettyFormat(doc)

Résultat : 

{
  "object": {
    "varInt": 12,
    "varString": "Test",
    "varBool": true,
    "varDate": "2019-09-16T17:32:19.189+02:00",
    "varNull": null
  }
}
