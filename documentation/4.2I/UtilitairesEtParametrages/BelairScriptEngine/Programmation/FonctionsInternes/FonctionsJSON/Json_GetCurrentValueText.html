Fonctions JSON

Json:GetCurrentValueText(doc)

La fonction renvoie le texte de la valeur courante.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
Le texte de la valeur courante.
La valeur est de type UnicodeString.


Exemple : 

local doc
doc := Json:New()
Json:NewObject(doc, "colors")
  Json:NewValue(doc, "color1", "Red")
  Json:NewValue(doc, "color2", "Green")
  Json:NewValue(doc, "color3", "Blue")
Json:EndObject(doc)
Json:Close(doc)            
Json:ValueExists(doc, "colors.color1")
print(Json:GetCurrentValueText(doc))
Json:Next(doc)
print(Json:GetCurrentValueText(doc))
return 0

Résultat : 

Red
Green
