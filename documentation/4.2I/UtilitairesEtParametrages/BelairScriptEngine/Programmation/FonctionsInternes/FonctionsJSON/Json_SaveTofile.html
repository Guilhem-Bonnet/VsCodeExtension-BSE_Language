Fonctions JSON

Json:SaveTofile(doc, filename, prettyformat)

cette fonction enregistre un document JSON dans un fichier.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'
  • filename : un nom de fichier
  • prettyformat : Un booléen afin d'avoir le contenue du fichier en PrettyFormat (optionnel)

Résultat :
Aucun.


Exemple : 

local doc
doc := Json:New()
Json:NewArray(doc, "colors")
  Json:NewArrayValue(doc, "Red")
  Json:NewArrayValue(doc, "Green")
  Json:NewArrayValue(doc, "Blue")
Json:EndArray(doc)
Json:Close(doc)  
Json:SaveToFile(doc, "Test_PrettyFormat.json", true)
Json:SaveToFile(doc, "Test_String.json")                                  
return 0

Résultat :

Fichier Test_PrettyFormat.json 
{
  "colors": [
    "Red",
    "Green",
    "Blue" 
  ]
}
Fichier Test_String.json 
{"colors":["Red","Green","Blue"]}
