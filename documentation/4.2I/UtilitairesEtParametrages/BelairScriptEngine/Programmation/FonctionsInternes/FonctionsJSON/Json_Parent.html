Fonctions JSON

Json:Parent(doc)

cette fonction rend courant le conteneur (JSONObject et JSONArray) parent de la valeur courante.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
la valeur booléenne True/False: True si le positionnement a réussi et False si la valeur courante n'a pas de JSONObject ou JSONArray comme parent.


Exemple : 

local doc
doc := Json:New()
Json:NewArray(doc, "colors")
  Json:NewArrayValue(doc, "Red")
  Json:NewArrayValue(doc, "Green")
  Json:NewArrayValue(doc, "Blue")
Json:EndArray(doc)
Json:NewArray(doc, "Entier")
  Json:NewArrayValue(doc, 1)
  Json:NewArrayValue(doc, 2)
  Json:NewArrayValue(doc, 3)
Json:EndArray(doc)
Json:Close(doc)                                    
Json:ValueExists(doc, "colors")
Json:Recurse(doc)
while Json:Next(doc)
  Print(Json:GetCurrentValueText(doc))
Json:Parent(doc)
Print("Parent --> %s", Json:GetCurrentPath(doc))   
return 0

Résultat : 

Green
Blue
Parent --> colors
