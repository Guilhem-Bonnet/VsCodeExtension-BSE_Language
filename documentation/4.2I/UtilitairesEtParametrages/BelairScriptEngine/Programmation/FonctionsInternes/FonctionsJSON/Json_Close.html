Fonctions JSON

Json:Close(doc)

La fonction met fin à l'écriture de l'objet de type 'document JSON'


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
Aucun.


Exemple : 


local doc
doc := Json:New()
Json:NewObject(doc, "colors")
  Json:NewValue(doc, "color1", "red")
  Json:NewValue(doc, "color2", "green")
  Json:NewValue(doc, "color3", "blue")
Json:EndObject(doc)
Json:Close(doc)      
return Json:PrettyFormat(doc)

Résultat : 

{
  "colors": {
    "color1": "red",
    "color2": "green",
    "color3": "blue" 
  }
}
