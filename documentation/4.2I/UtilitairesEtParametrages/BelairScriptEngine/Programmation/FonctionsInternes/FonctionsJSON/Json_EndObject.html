Fonctions JSON

Json:EndObject(doc)

La fonction met fin à un JSONOBject créé par la fonction Json:NewObject.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
Aucun.


Exemple : 

local doc
doc := Json:New()
Json:NewObject(doc, "colors")
  Json:NewValue(doc, "color1", "Red")
  Json:NewValue(doc, "color2", "Green")
  Json:NewValue(doc, "color3", "Blue")
Json:EndObject(doc)
Json:Close(doc)      
return Json:PrettyFormat(doc)

Résultat : 

{
  "colors": [
    "Red",
    "Green",
    "Blue" 
  ]
}
