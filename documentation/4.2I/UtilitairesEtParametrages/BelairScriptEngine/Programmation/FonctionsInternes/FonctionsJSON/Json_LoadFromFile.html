Fonctions JSON

Json:LoadFromFile(filename)

cette fonction fait ça.


Paramètres d'entrée :

  • filename : un nom de fichier

Résultat :
Un objet de type 'document JSON'


Exemple : 

local doc, doc2
doc := Json:New()
Json:NewObject(doc, "colors")
  Json:NewValue(doc, "color1", "Red")
  Json:NewValue(doc, "color2", "Green")
  Json:NewValue(doc, "color3", "Blue")
Json:EndObject(doc)
Json:Close(doc)            
Json:SaveToFile(doc, "c:\tmp\test.json")
doc2 := Json:LoadFromFile("c:\tmp\test.json")
return Json:PrettyFormat(doc2)

Résultat : 

{
  "colors": {
    "color1": "Red",
    "color2": "Green",
    "color3": "Blue" 
  }
}
