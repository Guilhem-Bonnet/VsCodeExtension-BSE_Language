Fonctions JSON

Json:GetCurrentValueInt(doc)

La fonction renvoie le texte de la valeur courante converti en type entier.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
Le texte de la valeur courante converti en type entier (int32).
Génère une erreur si la conversion n'est pas possible


Exemple : 

local doc, varTest
doc := Json:New()
Json:NewObject(doc, "object")
  Json:NewValue(doc, "int", 12)
Json:EndObject(doc)
Json:Close(doc)            
Json:ValueExists(doc, "object.int")
varTest := Json:GetCurrentValueInt(doc)
print(varTest)
print(VarTypeName(varTest))
return 0

Résultat : 

12
Integer
