Fonctions JSON

Json:GetCurrentValueCurrency(doc)

La fonction renvoie le texte de la valeur courante convertie en type monétaire.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
Le texte de la valeur courante convertie en type monétaire (Currency)
Génère une erreur si la conversion n'est pas possible


Exemple : 

local doc, varTest
doc := Json:New()
Json:NewObject(doc, "object")
  Json:NewValue(doc, "currency", 12.12)
Json:EndObject(doc)
Json:Close(doc)            
Json:ValueExists(doc, "object.currency")
varTest := Json:GetCurrentValueCurrency(doc)
print(varTest)
print(VarTypeName(varTest))
return 0

Résultat : 

12,12
Currency
