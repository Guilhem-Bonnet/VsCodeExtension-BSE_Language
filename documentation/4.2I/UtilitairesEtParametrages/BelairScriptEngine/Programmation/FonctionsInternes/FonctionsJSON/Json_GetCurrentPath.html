Fonctions JSON

Json:GetCurrentPath(doc)

La fonction renvoie le JSONPaths de la valeur courante.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
La valeur de type UnicodeString.
Une chaîne vide est retournée si on est au début ou à la fin du 'document JSON'.


Exemple : 

local doc
doc := Json:New()
Json:NewObject(doc, "colors")
  Json:NewValue(doc, "color1", "Red")
  Json:NewValue(doc, "color2", "Green")
  Json:NewValue(doc, "color3", "Blue")
Json:EndObject(doc)
Json:Close(doc)            
Json:ValueExists(doc, "colors.color1")
print(Json:GetCurrentPath(doc))
Json:First(doc)
print(Json:GetCurrentPath(doc))
return 0

Résultat : 

colors.color1
""
