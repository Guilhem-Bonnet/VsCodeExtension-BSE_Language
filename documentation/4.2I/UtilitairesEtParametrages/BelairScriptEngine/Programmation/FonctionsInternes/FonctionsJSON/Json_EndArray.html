Fonctions JSON

Json:EndArray(doc)

La fonction ferme un JSONArray créé par la fonction Json:NewArray.


Paramètres d'entrée :

  • doc : un objet de type 'document JSON'

Résultat :
Aucun.


Exemple : 

local doc
doc := Json:New()
Json:NewArray(doc, "colors")
  Json:NewArrayValue(doc, "Red")
  Json:NewArrayValue(doc, "Green")
  Json:NewArrayValue(doc, "Blue")
Json:EndArray(doc)
Json:Close(doc)      
return Json:PrettyFormat(doc)

Résultat : 

{
  "colors": [
    "Red",
    "Green",
    "Blue" 
  ]
}
