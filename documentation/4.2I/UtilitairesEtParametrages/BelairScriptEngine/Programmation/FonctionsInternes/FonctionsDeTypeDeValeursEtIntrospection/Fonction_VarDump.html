Fonctions de type de valeurs et introspection

Coalesce(var1,var2, [ ,...99 ])

Cette fonction réplique la fonction SQL COALESCE(v1,v2,[...])
Elle rend la 1ere valeur non nulle de la collection passée en paramètre.
SI toutes les valeurs de la collection sont nulles, la fonction renvoie Null()


Paramètres d'entrée :

  • var1, var2, var3,... : la collection de valeurs (minimum 2, maximum 99).


Exemples : 

montant := coalesce(quit.taxe, 0) 
lieudit := substring(coalesce(tiers.adr3,""),1, 10)

Obj : DumpVar(objet) (synonyme de VarDump)

La fonction renvoie les types et les valeurs des propriétés d'un objet, sous forme de chaîne de caractères séparée par CRLF.


Paramètres d'entrée :

  • objet objet à analyser.


Résultat :
La chaîne de caractères contenant le dump.

Notez que :

La valeur "Unknown" correspond en général à un autre objet, que vous pouvez récupérer dans une variable, que vous pouvez ensuite explorer par un autre dump.


Exemples : 

local entity
entity := Meta:GetEntityByCode("Adh")
Print("---")
Print(Obj:DumpVar(entity))

VarIsBool(variable)

La fonction renvoie True si la variable est de type booléen, False sinon.


Paramètres d'entrée :

  • variable : la variable.

VarIsDate(variable)

La fonction renvoie True si la variable est de type DateTime, False sinon.


Paramètres d'entrée :

  • variable : la variable.

VarIsFloat(variable)

La fonction renvoie True si la variable est de type 'Float', False sinon.


Paramètres d'entrée :

  • variable : la variable.

VarIsInt(variable)

La fonction renvoie True si la variable est de type Entier, False sinon.


Paramètres d'entrée :

  • variable : la variable.

VarIsNum(variable)

La fonction renvoie True si la variable est de type Numérique (extended), False sinon.


Paramètres d'entrée :

  • variable : la variable.

VarIsString(variable)

La fonction renvoie True si la variable est de type chaine (string), False sinon.


Paramètres d'entrée :

  • variable : la variable.
