Fonctions de manipulation de métadonnées

Meta:GetEntities(number)

Renvoie l'entité, selon son numéro dans la liste des entités.


Paramètres d'entrée :

  • number : le numéro de l'entité.


Résultat :

  • L'objet de type "TMetaTable" ou descendant ("TExtendedMetaTable", "TCatalogMetaTable", "TProtectedMetatable").


Exemples : 

local i, j, entity
i := 0
while i < Meta:EntityCount() 
begin
  entity := Meta:GetEntities(i)
  Print("Entity %s (%s). Code: %s, name: %s", i, entity.TypeName, entity.Code, entity.Name)
  i := i + 1
end
