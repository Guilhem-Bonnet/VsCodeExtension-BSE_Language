Fonctions de manipulation de métadonnées

Meta:GetFields(Entity, Number)

Renvoie un champ d'entité, selon son numéro dans la liste des champs.


Paramètres d'entrée :

  • Entity : l'entité.
  • Number : le numéro de champ.


Résultat :

  • L'objet de type "TMetaField".


Exemples : 

local i, j, entity, field
i := 0
while i < Meta:EntityCount() 
begin
  entity := Meta:GetEntities(i)
  Print("Entity %s (%s). Code: %s, name: %s", i, entity.TypeName, entity.Code, entity.Name)
  i := i + 1
  j := 0
  while j < Meta:FieldCount(entity)
  begin
    field := Meta:GetFields(entity, j)
    Print("Field %s (%s). Code: %s, name: %s", j, field.TypeName, field.Code, field.Name)
    j := j + 1
  end
end

Résultat :

Entity 0 (TMetaTable). Code: ADH, name: ADHDEI
Field 0 (TMetaField). Code: Contrat, name: B_CONTRAT
Field 1 (TMetaField). Code: Piece, name: B_PIECE
Field 2 (TMetaField). Code: Adhesion, name: B_ADHESION
Field 3 (TMetaField). Code: Entree, name: B_ENTREE
...
