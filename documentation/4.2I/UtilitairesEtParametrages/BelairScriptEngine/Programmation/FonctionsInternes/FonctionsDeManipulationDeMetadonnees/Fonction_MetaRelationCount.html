Fonctions de manipulation de métadonnées

Meta:RelationCount(Entity)

Renvoie le nombre des relations.


Paramètres d'entrée :
Entity : l'entité. Utilisez Meta:RelationCount() pour obtenir toutes les relations du modèle.


Résultat :

  • Le nombre (integer).


Exemples : 

local entity, relation, i
Print("Relations of ADH")
entity := Meta:GetEntityByCode("Adh")
i := 0
while i < Meta:RelationCount(entity)
begin
  relation := Meta:GetRelations(entity, i)
  Print(relation.Name)
  i := i + 1
end
Print("")
Print("All model relations")
i := 0
while i < Meta:RelationCount()
begin
  relation := Meta:GetRelations(i)
  Print(relation.Name)
  i := i + 1
end
