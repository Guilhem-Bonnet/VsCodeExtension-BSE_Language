Fonctions de manipulation de métadonnées

Meta:FindEntityByCode(code)

Renvoie une entité selon son code.


Paramètres d'entrée :

  • code : code de l'entité.


Résultat :

  • L'objet de type "TMetaTable" ou descendants.
  • "Null" si l'entité n'est pas trouvée.


Exemples : 

local entity
entity := Meta:FindEntityByCode("Adh")
Print("Entity ADH")
Print("---")
Print(Obj:DumpVar(entity))
