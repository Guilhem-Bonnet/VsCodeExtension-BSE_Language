Fonctions de manipulation de métadonnées

Meta:GetLinks(Relation, LinkNumber)

Renvoie un link (lien) de relation, selon son numéro dans la liste.


Paramètres d'entrée :

  • Relation : la relation.
  • LinkNumber : le numéro de link (lien).


Résultat :

  • L'objet de type "TRelationLink".


Exemples : 

local entity, relation, link
entity := Meta:GetEntityByCode("Adh")
relation := Meta:GetRelations(entity, 0)
link := Meta:GetLinks(relation, 0)
Print(Obj:DumpVar(link))
