Fonctions de manipulation de métadonnées

Meta:GetIndexes(Entity, Number)

Renvoie un index d'entité, selon son numéro dans la liste des index.


Paramètres d'entrée :

  • Entity : l'entité.
  • Number : le numéro de l'index.


Résultat :

  • L'objet de type "TMetaIndex".


Exemples : 

local i, j, entity, index
i := 0
while i < Meta:EntityCount() 
begin
  entity := Meta:GetEntities(i)
  Print("Entity %s (%s). Code: %s, name: %s", i, entity.TypeName, entity.Code, entity.Name)
  i := i + 1
  j := 0
  while j < Meta:IndexCount(entity)
  begin
    index := Meta:GetIndexes(entity, j)
    Print("Index %s (%s). Name: %s", j, index.TypeName, index.Name)
    j := j + 1
  end
end

Résultat :

Entity 0 (TMetaTable). Code: ADH, name: ADHDEI
Index 0 (TMetaIndex). Name: IX1_ADH
Index 1 (TMetaIndex). Name: IX2_ADH
...
