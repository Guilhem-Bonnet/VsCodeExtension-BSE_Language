Fonctions de manipulation de métadonnées

Meta:GetIndexFields(Index, FieldNumber)

Renvoie un champ d'index, selon son numéro dans la liste des champs.


Paramètres d'entrée :

  • Index : l'index.
  • FieldNumber : le numéro de champ.


Résultat :

  • L'objet de type "TMetaIndexField".


Exemples : 

local entity, index, field
entity := Meta:GetEntityByCode("Adh")
index := Meta:GetIndexes(entity, 0)
field := Meta:GetIndexFields(index, 0)
Print(Obj:DumpVar(field))
