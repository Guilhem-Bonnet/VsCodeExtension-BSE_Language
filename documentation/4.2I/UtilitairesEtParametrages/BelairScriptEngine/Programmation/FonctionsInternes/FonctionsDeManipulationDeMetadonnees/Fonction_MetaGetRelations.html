Fonctions de manipulation de métadonnées

Meta:GetRelations(Entity, RelationNumber)/(RelationNumber)

Renvoie une relation, selon son numéro dans la liste.


Paramètres d'entrée :

  • Entity : l'entité.
  • RelationNumber : le numéro de relation.

ou :

  • RelationNumber : le numéro de relation.


Résultat :

  • L'objet de type "TRelation".


Exemples : 

local entity, relation1, relation2
entity := Meta:GetEntityByCode("Adh")
relation1 := Meta:GetRelations(entity, 0) // first entity relation
Print(Obj:DumpVar(relation1))
relation2 := Meta:GetRelations(0) // first relation in model 
Print(Obj:DumpVar(relation2))
