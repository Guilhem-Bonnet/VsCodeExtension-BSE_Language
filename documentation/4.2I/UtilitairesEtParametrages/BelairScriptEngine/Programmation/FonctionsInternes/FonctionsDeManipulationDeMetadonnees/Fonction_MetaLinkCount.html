Fonctions de manipulation de métadonnées

Meta:LinkCount(Relation)

Renvoie le nombre des links de relation.


Paramètres d'entrée :

  • Relation : la relation.


Résultat :

  • Le nombre (integer).


Exemples : 

local entity, relation, link, i
entity := Meta:GetEntityByCode("Catal")
relation := Meta:GetRelations(entity, 0)
i := 0
while i < Meta:LinkCount(relation)
begin
  link := Meta:GetLinks(relation, i)
  Print(link.Name)
  i := i + 1
end

Résultat :

b_cie=b_cie
b_prodcie=b_prodcie
