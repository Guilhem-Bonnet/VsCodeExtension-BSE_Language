Fonctions de manipulation de métadonnées

Meta:FindFieldByCode(Entity, EncodeField)

Renvoie un champ d'entité, selon son code.


Paramètres d'entrée :

  • Entity : l'entité.
  • EncodeField : le code de champ.


Résultat :

  • L'objet de type "TMetaField".
  • "Null" si le champ n'est pas trouvé.


Exemples : 

local entity, field
entity := Meta:GetEntityByCode("Adh")
field := Meta:FindFieldByCode(entity, "Contrat")
if not IsNull(field) then
begin
  Print("Entity ADH")
  Print("Field Contrat")
  Print("---")
  Print(Obj:DumpVar(field))
end
