Fonctions du client HTTP

HTTP:SetBody

Défini la valeur du corps de la prochaine requête.


Paramètres d'entrée :

  • 1. objet HTTP
  • 2. chaîne, contenu du body


Résultat :

Aucun.


Exemples :

Note:

Dans cet exemple, on utilise httpbin.org, un site qui permet de vérifier la demande qui lui a été faite.

Le site retourne en json le détail de la requête qui lui a été passée.

Code :

local http, body, json
http := HTTP:New()
HTTP:SetBody(http, "42")
body := HTTP:POST(http, "https://httpbin.org/post") // raise if not 200
json := JSON:Open(body)
print(JSON:SelectResult(json, "data"))

Résultat :

42
