Fonctions du client HTTP

HTTP:SetHeader

Défini la valeur d'un champ header pour la prochaine requête.


Paramètres d'entrée :

  • 1. objet HTTP
  • 2. chaîne, nom du champ
  • 3. chaîne, valeur du champ


Résultat :
Aucun.


Exemples :

Note :

Dans cet exemple, on utilise httpbin.org, un site qui permet de vérifier la demande qui lui a été faite.

Le site retourne en json le détail de la requête qui lui a été passée.

Code :

local Http, Body, ContentType, Json
Http := HTTP:New()
HTTP:SetHeader(Http, "toto", "titi")
Body := HTTP:GET(Http, "https://httpbin.org/get")
Assert(HTTP:ResponseCode(Http) = 200)
json := JSON:Open(Body)
print(JSON:SelectResult(json, "headers.toto")))

Résultat :

titi
