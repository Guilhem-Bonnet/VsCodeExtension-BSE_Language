Fonctions du client HTTP

HTTP:Get

Appelle une URL avec la méthode Get.


Paramètres d'entrée :

  • 1. objet HTTP
  • 2. chaîne, url à appeler


Résultat :

    Le résultat renvoyé par le serveur.
    En cas d'erreur, le résultat est vide.


Exemples :

    Code :
local h, r
h := HTTP:New()
r := HTTP:Get(h, "http://www.perdu.com")
print(r)
    Résultat :
<html><head><title>Vous Etes Perdu ?</title></head><body><h1>Perdu sur l'Internet ?</h1><h2>Pas de panique, on va vous aider</h2><strong><pre>    * <----- vous &ecirc;tes ici</pre></strong></body></html>
