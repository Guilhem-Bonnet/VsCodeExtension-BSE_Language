Fonctions du client HTTP

HTTP:Delete

Appelle une URL avec la méthode DELETE.

Note: il est aussi possible d'utiliser HTTP:SetBody ou HTTP:SetBodyFile pour définir le body de la prochaine requête.


Paramètres d'entrée :

  • 1. objet HTTP


Résultat :

    Renvoi le corps de la réponse.
    En cas d'erreur, le résultat est vide.


Exemples :

Code :

				
Print(local http, body, json
http := HTTP:New()
body := HTTP:DELETE(http, "http://httpbin.org/delete?id=154")
Assert(HTTP:ResponseCode(http) = 200) // raise if not 200
json := JSON:Open(body)
print(JSON:PrettyFormat(json))
