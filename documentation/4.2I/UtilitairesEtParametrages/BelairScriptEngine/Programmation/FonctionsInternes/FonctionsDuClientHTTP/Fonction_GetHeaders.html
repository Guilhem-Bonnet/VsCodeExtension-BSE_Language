Fonctions du client HTTP

HTTP:GetHeaders

Créé un objet dont chaque propriété correspond à un champ header renvoyé par le serveur.


Paramètres d'entrée :

  • 1. objet HTTP


Résultat :

Objet avec la liste des champs headers.


Exemples :

Note:

Dans cet exemple, on utilise httpbin.org, un site qui permet de vérifier la demande qui lui a été faite. Le site retourne en json le détail de la requête qui lui a été passée.

Code :

local Http, Body, Headers
Http := HTTP:New()
Body := HTTP:POST(Http, "https://httpbin.org/post")
Assert(HTTP:ResponseCode(Http) = 200) // raise if not 200
Headers := HTTP:GetHeaders(Http)
print(dumpvar(Headers))
print(StrFormat("Type de serveur= %s", Headers.Server))

Résultat :

  Access-Control-Allow-Origin (UnicodeString): *

Type de serveur= gunicorn/19.9.0
