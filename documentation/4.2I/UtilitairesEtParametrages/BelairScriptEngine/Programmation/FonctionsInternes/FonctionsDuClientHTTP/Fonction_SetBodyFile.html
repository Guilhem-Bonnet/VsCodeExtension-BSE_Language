Fonctions du client HTTP

HTTP:SetBodyFile

Définit la valeur du body de la prochaine requête.


Paramètres d'entrée :

  • 1. objet HTTP
  • 2. chaîne, chemin vers un fichier


Résultat :

Aucun.


Exemples :

Note:

Dans cet exemple, on utilise httpbin.org, un site qui permet de vérifier la demande qui lui a été faite.

Le site retourne en json le détail de la requête qui lui a été passée.

Contenu du fichier Test.Json:

{"test":"UNIT TEST"}

Code :

Local http, body, json
http := HTTP:New()
HTTP:SetBodyFile(http, "c:\tmp\test.json")
body := HTTP:POST(http, "https://httpbin.org/post")
json := JSON:Open(body)
return JSON:SelectResult(json, "json.test")

Résultat :

UNIT TEST
