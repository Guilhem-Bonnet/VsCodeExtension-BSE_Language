Fonctions du client HTTP

HTTP:SaveResponseToFile(http, filename)

enregistrement de la réponse des fonctions http dans un fichier.


Paramètres d'entrée :

  • http objet HTTP
  • filename Fichier à enregistrer


Résultat :

Aucun.


Exemple :

local h
h := HTTP:New()
HTTP:Get(h, "https://www.belair-info.fr/pictos/LOGO-BELAIR-1600px.png")
HTTP:SaveResponseToFile(h, "c:\tmp\belair.png")
