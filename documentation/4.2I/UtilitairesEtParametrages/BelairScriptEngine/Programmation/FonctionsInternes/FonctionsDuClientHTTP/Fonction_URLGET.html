Formules de calcul - Commandes de déclenchement d'une URL

URLGET(http://xxxxxxxx)

Cette fonction doit retourner le même résultat que la fonction de calcul URLGET.

Elle est indispensable dans BSE pour permettre par exemple le traitement d'un flux XML retournée par une URL


Exemple : 
local xmldoc
xmldoc := XML:OPEN(URLGET("https://jimi79.hd.free.fr/belair/dates.php"))
Print(Xml:PrettyFormat(XmlDoc))
