Fonctions du client HTTP

HTTP:Post

Appelle une URL avec la méthode POST.


Paramètres d'entrée :

  • 1. objet HTTP


Résultat :

Renvoi le corps de la réponse.

En cas d'erreur, le résultat est vide.


Exemples :

Code :

local http, body, json
http := HTTP:New()
HTTP:AddField(http, "name", "spencer")
body := HTTP:POST(http, "http://httpbin.org/post")
Assert(HTTP:ResponseCode(http) = 200) // raise if not 200
json := JSON:Open(body)
print(JSON:PrettyFormat(json))
