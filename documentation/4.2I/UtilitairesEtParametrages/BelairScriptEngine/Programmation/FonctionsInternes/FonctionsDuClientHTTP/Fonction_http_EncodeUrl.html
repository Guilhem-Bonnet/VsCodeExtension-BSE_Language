Fonctions du client HTTP

HTTP:EncodeUrl(chaine)

La fonction rend retourne la chaine dans le "codage UR" :https://fr.wikipedia.org/wiki/Encodage-pourcent


Paramètres d'entrée :

chaine la chaîne de caractères à encoder


Résultat :

La chaîne de caractères encodée


Exemples :

Code :

local xml, xmlEncoded    
xml := "<?xml version=""1.0""?><colors><color>red</color><color>green</color><color>blue</color></colors>" 
xmlEncoded := HTTP:EncodeURL(xml)    
print(xmlEncoded)

Résultat :

%3C%3Fxml+version%3D%221.0%22%3F%3E%3Ccolors%3E%3Ccolor%3Ered%3C%2Fcolor%3E%3Ccolor%3Egreen%3C%2Fcolor%3E%3Ccolor%3Eblue%3C%2Fcolor%3E%3C%2Fcolors%3E
