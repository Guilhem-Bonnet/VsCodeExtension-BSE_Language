Fonctions du client HTTP

HTTP:AddField

Définit la valeur d'un champ de formulaire pour la prochaine requête.

Note: il est aussi possible d'utiliser HTTP:SetBody ou HTTP:SetBodyFile pour définir le body de la prochaine requête.


Paramètres d'entrée :

  • 1. objet HTTP
  • 2. chaîne, nom du champ
  • 3. chaîne, valeur du champ


Résultat :
Aucun.


Exemples :

local http, body, json
http := HTTP:New()
HTTP:AddField(http, "name", "spencer")
body := HTTP:POST(http, "http://httpbin.org/post") // raise if not 200
Assert(HTTP:ResponseCode(http) = 200)
json := JSON:Open(body)
print(JSON:SelectResult(json, "form.name"))

Résultat :

spencer
