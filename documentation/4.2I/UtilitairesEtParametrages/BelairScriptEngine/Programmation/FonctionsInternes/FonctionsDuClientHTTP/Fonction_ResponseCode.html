Fonctions du client HTTP

HTTP:ResponseCode

Appelle une URL.


Paramètres d'entrée :

  • 1. objet HTTP


Résultat :

Renvoie le dernier code http reçu.

-1 est renvoyé si aucun appel n'a eu lieu, ou si le dernier a échoué, par exemple si l'url n'existe pas.


Exemples :

Code :

local h, r
h := HTTP:New()
r := HTTP:Get(h, "http://www.perdu.com")
print(HTTP:ResponseCode(h))

Résultat :

200
