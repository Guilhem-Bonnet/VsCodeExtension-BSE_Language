Fonctions mathématiques

Abs(x)

Rend la valeur absolue (positive) de x

Exp(x)

Rend la valeur exponentielle de x

Ln(x)

Rend le logarithme népérien de x (inverse de Exp(x))

LogN(x, base)

Rend le logarithme de x base base

Power(x, base)

Rend la puissance basede x

Root(base, x)

Rend la racine basede x (inverse de Power(x, base))

Round(x)

Rend la valeur "Round" de x (fonctionnalité inconnue !!!!!!!)

Sqrt(x)

Rend la racine carrée de x

Trunc(x)

Rend la valeur entière de x
