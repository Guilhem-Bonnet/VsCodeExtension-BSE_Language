Les fonctions internes


Les fonctions internes font partie du langage BS.

Les noms des fonctions ne sont pas réservés.
Cela veut dire, qu'une variable déclarée "cache" le nom d'une fonction, à l'exception d'un appel explicite avec des parenthèses.


local currentDate
currentDate := 1
Print(currentDate)    // la variable est utilisée.
Print(CurrentDate())  // la fonction est appelée.

Résultat :

1
25/02/2018
