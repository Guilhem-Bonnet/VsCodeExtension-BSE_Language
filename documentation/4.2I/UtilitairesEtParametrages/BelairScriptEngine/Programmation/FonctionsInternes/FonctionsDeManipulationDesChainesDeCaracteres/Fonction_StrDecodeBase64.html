Fonctions de manipulation des chaînes de caractères (strings)

StrDecodeBase64

La fonction renvoie le string décodé au format Base64.


Paramètres d'entrée :

  • 1. Le string à décoder.


Résultat :
La chaîne de caractères décodée.


Exemples : 

local s1, s2
s1 := StrEncodeBase64("Hello Base 64")
Print(s1)
s2 := StrDecodeBase64(s1)
Print(s2)

Résultat :

SGVsbG8gQmFzZSA2NA==
Hello Base 64
