Fonctions de manipulation des chaînes de caractères (strings)

ToStr

Convertit une valeur de n'importe quel type dans une chaîne de caractères.


Paramètres d'entrée :

  • 1. La valeur.


Résultat :
La chaîne.


Exemples : 

Print("Int: " + ToStr(123))
Print("Num: " + ToStr(123.45))
Print("Bool: " + ToStr(true))
Print("Date: " + ToStr(EncodeDate(2018, 3, 15)))
Print("DateTime: " + ToStr(EncodeDateTime(2018, 3, 15, 9, 25, 36)))

Résultat :

Int: 123
Num: 123,45
Bool: True
Date: 15/03/2018
DateTime: 15/03/2018 09:25:36
