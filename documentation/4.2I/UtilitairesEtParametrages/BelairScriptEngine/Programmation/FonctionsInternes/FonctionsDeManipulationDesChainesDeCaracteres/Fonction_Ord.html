Fonctions de manipulation des chaînes de caractères (strings)

Ord()

La fonction renvoie le code de caractère.


Paramètres d'entrée :

  • 1. La chaîne de caractères (uniquement le premier caractère est pris en compte).


Résultat :
Le code (en UTF-16) ou la valeur vide si la chaîne est vide.


Exemples : 

Print(Ord("A"))
Print(Ord(""))
Print(Ord("ABC"))
Print(Ord("Ê"))
Print(Ord("€"))
Print(Ord("Щ"))

Résultat :

65

65
202
8364
1065
