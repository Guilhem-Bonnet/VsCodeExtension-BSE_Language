Lancement de processus Bse vers un programme générique

Exec

La fonction exécute une commande puis retourne le résultat.


Paramètres d'entrée :

  • 1. Le nom du programme
  • 2. Les paramètres d'exécution
  • 3. Une chaîne a transmettre au programme (optionnel).

Résultat :

#le texte de la valeur trouvée
Si le programme retourne une erreur. Le message d'erreur sera retourné par la fonction.

Exemples :

local result, Input
Input := "Test" 
Exec("d:\belair\belair.exe", "/Ad:\belair\test\appxxx.conf")
result := Exec("Java", "-jar MonJar.jar", Input)
return result

Résultat :

Script result:
Unit Test
