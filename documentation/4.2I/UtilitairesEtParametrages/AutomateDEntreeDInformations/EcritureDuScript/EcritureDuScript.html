Ecriture du script

Le script définissant la procédure à appliquer à chaque enregistrement du fichier lu, est constitué d'instructions de format 'classique', dont la liste exhaustive est fournie ci-dessous.

Ces instructions sont de type 'traitement' sur information, et débranchements conditionnels.

Ce programme se réfère simultanément à chacune des informations décrites dans le fichier 'lu', et à la base de données 'Belair'.

Il permet l'utilisation de variables de travail.

Le script est composé à partir des fonctions propres aux formules de calcul utilisables habituellement dans BELAIR, ainsi que de commandes spécifiques à l'automate d'import de données.
