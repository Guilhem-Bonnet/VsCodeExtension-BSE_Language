Messages

Les instructions ECRAN et PRINT permettent à la volonté du programmeur, de suivre le déroulement des traitements effectués.

PRINT("message")

Le message s'affiche dans la fenêtre "Compte-rendu" (habituellement réservée aux anomalies éventuellement rencontrées) et s'écrit dans le Log de traitement.

PRINTLOG("message")

Le message ne s'affiche pas dans la fenêtre "Compte-rendu", il s'écrit seulement dans le Log de traitement.

SAVELOG("fichier")

Sauvegarde le Log de traitement dans un fichier donné (fichier pouvant être donné par la commande de formule de calcul CHOIXFICHIER).

ECRAN("message")

Le message s'affiche dans la fenêtre "Messages" (informations sur le déroulement normal du script) sans inscription dans le Log de traitement.
La syntaxe est : ECRAN("police : " + ?POLI("b_police")) où une chaîne de caractères entre double-quotes est incluse entre les parenthèses.
Dans cet exemple : 'police : ' est une chaîne de caractère quelconque, et ?POLI("b_police") est le nom d'une donnée de la base de données Belair.

Ces instructions de suivi n'interrompent pas le traitement.
Il est possible de faire apparaître ou de tester le numéro d'ordre de l'enregistrement traité par l'instruction RECORD : ECRAN(RECORD)
