Table de stockage d'actions

Table BATCH

Cette table permet d'empiler des listes d'actions diverses à effectuer, par des scripts automate, des formules de calcul, etc...

Le champ B_ACTION permet de nommer l'action.
Les champs B_PARAM1 .... B_PARAM5 sont des zones de texte permettant de stocker différents paramètres, propres à chaque action.

Ensuite, un script peut tourner sur une machine dédiée et déclencher les actions programmées en attente (par exemple, envoi de mails).
