  • Catalogues. : ensemble des tables de catalogue

Caractéristiques d'une table

En cliquant sur une table, la fenêtre de droite affiche les caractéristiques générales de la table.

Nom Valeur Obligatoire
Code Code logique de la table dans Belair OUI
Domaine Domaine fonctionnel
Groupe Groupe fonctionnel
Intitulé Intitulé de la table (titre de l'onglet pour une extension) OUI
Nom Nom physique de la table dans la base de données OUI
Tracée ? Indique si toutes les modifications dans cette table sont tracées (si l'option est activée) OUI
Type de table Indique le type de table :systême, protégée, applicative, extension ou catalogue OUI

Caractéristiques spécifiques aux extensions

Nom Valeur Obligatoire
Extension multiple indique si la relation avec la table maitre est 0-1 ou 0-N OUI
Type d'extension Indique la table principale à laquelle l'extension est attachée OUI
