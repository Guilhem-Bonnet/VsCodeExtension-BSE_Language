Modélisation des données

Vérification du modèle de données

La fonction Modèle | Vérification du modèle vérifie la structure de votre base de données avec le modèle défini [Ctrl]+[F9]

Le résultat de cette vérification est affiché dans les 2 fenêtres du statut en bas à droite

Sauvegarde du modèle de données et modification physique de la base

Toutes les interventions effectuées sur le modèle de données ne sont pas appliquées en temps réel sur la base de données.
Une fois vos travaux terminés, vous devez donc appliquer les modifications du modèle sur votre base

La sauvegarde du modèle de données se fait en sélectionnant la fonction Fichier | Enregistrer [Ctrl]+S

Un contrôle de cohérence est fait, et les modifications apportées au modèle sont alors appliquées à votre base de données.

Le compte-rendu de traitement est consultable dans les fenêtres de statut.
