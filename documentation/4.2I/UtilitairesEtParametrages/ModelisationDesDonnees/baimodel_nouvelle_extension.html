Création d'une nouvelle extension

Démarrage de la fonction de création d'une nouvelle extension

En cliquant sur la ligne 'Extensions' et par le menu principal

Par le menu surgissant, en faisant un clic-droit sur la ligne 'Extensions'

L'interface 'Nouvelle table d'extension apparait:'

Code de table

Saisissez ici un identifiant unique de 3 à 5 lettres, sans espace ni signe.
Le code final de la table sera donc les 2 premieres lettres imposées, selon le type d'extension et l'identifiant saisi ici

Extension multiple

Par défaut, il n'y a qu'un seul enregistrement d'extension associé à l'enregistrement maitre.
SI vous souhaitez que plueisurs enregistrements puissent être associés, cochez cette option:
Dans ce cas, l'onglet contiendra une grille est des boutons de navigation

Validation de la création

Cliquez sur OK pour valider la création de la table.

L'écran suivant apparait, présentant le squelette de la table créée

Nom ajoutez derrière le nom généré le suffixe de votre application (3 lettres). Ce sera le nom physique de la table dans la base.

Les champs de la clé primaire de l'extension sont automatiquement créés en fonction du type d'extension.
Il ne faut pas modifier ces champs

reportez-vous à la page Modification d'une table
