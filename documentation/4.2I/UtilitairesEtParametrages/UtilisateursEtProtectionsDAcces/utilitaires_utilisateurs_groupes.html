Les groupes de limitation

Sélectionnez l'onglet "Groupes de limitation"

Un groupe de limitation est un générique permettant de rassembler plusieurs utilisateurs (par exemple, un groupe portefeuille).

Sur la droite de l'écran, s'affichent les utilisateurs créés précédemment.

Pour associer les utilisateurs aux groupes concernés, sélectionnez le groupe de limtations dans la colonne de gauche et glissez les utilisateurs de la colonne de droite vers la colonne "Utilisateurs concernés".

Il est également possible d'indiquer le ou les groupe(s) de limitation d'un utilisateur donné à partir de l'onglet "Utilisateurs", grâce aux boutons et du pavé 'Limitations d'accès'
Sélectionnez l'utilisateur :

  • Le bouton affiche la liste des groupes existants et permet d'ajouter l'utilisateur au groupe choisi dans la liste.
  • Le bouton enlève l'utilisateur du groupe sélectionné.
