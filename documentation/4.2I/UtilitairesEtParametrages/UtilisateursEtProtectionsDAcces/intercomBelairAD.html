Intercommunication entre Belair et l’Active Directory


Il est maintenant possible de se connecter à Belair en utilisant un compte de l'Active Directory.
Le compte (domaine\compte) doit être associé à un utilisateur Belair existant

Le but étant de laisser toute la gestion des mots de passe à Windows

Explications

Pour vous identifier avec le domaine il faut :

  • Un poste de travail sur le même réseau
  • Un utilisateur BELAIR associé a votre compte dans le domaine (paramétrage dans la gestion des utilisateurs)
  • vérifiez les infos du compte AD
  • depuis Belair, liste des utilisateurs: ajoutez (ou modifiez) le compte sélectionné
  • depuis les détails du compte, saisissez le nom du compte correspondant dans l'Active Directory (Domain Login)

  • Pour se connecter, saisissez votre compte AD (en indiquant le domaine) et le mot de passe associé.
