Protection des écrans d'extension

Cette fonction permet d'occulter un onglet d'extension pour certains groupes d'utilisateurs.

Attention : le paramétrage est prévu à l'inverse des autres protections, c'est-à-dire que seul le(s) groupe(s) d'utilisateur(s) désignés auront accès à l'extension "protégée".

Dans Outils | Formules de calcul | Formules autonomes.

  • Créer une nouvelle formule () dont le code est OBLIGATOIREMENT syntaxé de la façon suivante : PR:XXXXX (où xxxxx est le code de la table d'extension)
    Par exemple, si l'extension porte le code XTTIE, alors le code de la formule de calcul sera PR:XTTIE.
  • ensuite, dans la feuille de calcul, cocher "VRAI/FAUX" (formule booléenne)
  • puis ouvrir le dictionnaire , onglet "Fonction", rubrique "Divers" et glisser la fonction "GROUPE("chaîne")"
  • Remplacer "chaîne" par le(s) code(s) du (des) groupe(s) autorisé(s) à accéder à l'extension.
    (Exemple : =GROUPE("P") - qui renvoie à VRAI si l'utilisateur appartient au groupe de limitation "P".
  • Tous les utilisateurs faisant partie du groupe (P pour notre exemple) auront accès à l'onglet d'extension (XTTIE pour notre exemple).
