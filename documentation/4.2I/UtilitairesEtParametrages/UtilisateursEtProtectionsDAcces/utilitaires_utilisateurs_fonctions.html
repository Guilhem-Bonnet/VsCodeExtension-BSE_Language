Protections de fonctions

Cette fonction est utilisable par le SUPERVISEUR à partir de tous les écrans disposant d'un menu et permet d'interdire facilement à tel ou tel groupe l'accès aux fonctions accessibles par le menu.

Utilisez la fonction Limitation d'accès pour activer le mode "Limitation d'accès" (une coche apparaît alors en face de cette fonction dans le menu).

Tant que ce mode est activé, le choix d'un élément du menu ne déclenche pas la fonction correspondante mais affiche un écran vous permettant d'indiquer les groupes de limitations pour lesquels la fonction est interdite. Les utilisateurs faisant partie d'au moins un des groupes ainsi définis ne pourront pas utiliser cette fonction. En résumé, un utilisateur peut utiliser une fonction s'il ne fait partie d'aucun groupe de limitations pour lequel l'accès est refusé.

Utilisez à nouveau la fonction Limitation d'accès pour désactiver le mode "Limitation d'accès" (la coche disparaît) ; le mode est automatiquement désactivé à chaque changement d'écran.

Les utilisateurs ne voient pas dans les menus les fonctions auxquels ils n'ont pas accès.

Cette fonctionnalité est étendue également aux fonctions du portefeuille. Pour en protéger certaines utiliser la fonction Limitation d'accès du portefeuille et procéder de la même façon.
