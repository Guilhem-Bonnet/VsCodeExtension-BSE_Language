Principe de fonctionnement

Le programme Ecoute est basé sur deux composants : l'intégration de fichier et la copie de base de données.
Le programme Ecoute se lance depuis le répertoire d'installation de BELAIR (par défault 'C:\Belair4').

La première composante, intégration de fichier, permet de traiter des fichiers par un script BSI, via l'automate d'import. L'intégration de fichier peut être utilisée par exemple pour enregistrer des dossiers saisis par internet.

La seconde composante, copie de base, permet de faire un backup/restore d'une base A vers une base B. La copie de base peut être utilisée par exemple pour avoir une base temporaire disponible en lecture qui ne puisse pas altérer la base de production.

Fichier INI

Le programme est paramétré grâce à un fichier INI. Le fichier INI à utiliser est paramétré lors de l'appel du programme Ecoute, à l'aide de la ligne de commande suivante :

Ecoute.EXE /ini:[chemin_vers_le_fichier_ini]

Exemple :
Ecoute.EXE /ini:c:\belair3\appdev\config.ini

Contenu du fichier INI

Le fichier INI est découpé en trois sections :

  • une section [Copie], utilisée pour la copie de la base
  • une section [Memo], utilisée pour envoyer des alertes à un utilisateur en cas d'erreur
  • une section [Ecoute], utilisée pour l'intégration de fichier.
