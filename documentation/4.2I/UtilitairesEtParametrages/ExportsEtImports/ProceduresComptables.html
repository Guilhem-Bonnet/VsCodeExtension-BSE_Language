Les procédures comptables

Les opérations suivantes s'effectuent à partir de la comptabilité :


Exportation

A partir de l'application l'origine, il existe deux façons de procéder :

  • Il est possible d'exporter toutes les procédures comptables :


  • Sélectionner la procédure à exporter. (Pour sélectionner plusieurs procédures, utiliser les fonctions de sélection de WINDOWS : touche Ctrl + clic).


Le nom du fichier d'export est TRFKPR, cliquer dans (les fichiers d'export sont stockés implicitement dans le répertoire "travail" du poste mais il est possible d'indiquer un autre chemin de réception).
Lorsque le fichier est constitué, un message apparaît : "Export terminé". Cliquer sur
Plusieurs fichiers sont alors créés : TRFKPR.PX, TRFKPR.MB, TRFKPR.db.

Importation

A partir de l'application d'arrivée :


Le fichier TRFKPR est proposé dans le répertoire "travail" du poste, cliquer dans (si les fichiers ont été stockés à un autre endroit, il faut ouvrir le répertoire les contenant avant de cliquer sur ).
Le contenu du fichier apparaît dans une fenêtre vous permettant de sélectionner la (ou les) procédures(s) à importer. Cocher "Avec" ou "Sans" remplacement des pré-existantes.
Lorsque cette sélection est effectuée, cliquer sur un des boutons ou selon d'option choisie.
Lorsque l'importation est terminée, un message apparaît "Import terminé". Cliquer sur
