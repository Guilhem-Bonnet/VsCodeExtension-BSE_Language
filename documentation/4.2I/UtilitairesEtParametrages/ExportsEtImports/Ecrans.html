Les écrans

Les opérations suivantes s'effectuent dans Outils | Paramètres | Modélisation de l'application.

Exportation

A partir de l'application d'origine, demander Fichier | Ecrans de saisie. Il est possible d'exporter tous les écrans en demandant Fichier puis exporter Tous les écrans.
OU
Sélectionner l'écran à exporter.
Demander Fichier puis Exporter écran.
Le nom du fichier d'export est TRFECRAN, cliquer dans Ouvrir (les fichiers d'export sont stockés implicitement dans le répertoire "travail" du poste mais il est possible d'indiquer un autre chemin de réception). Lorsque le fichier est constitué, un message apparaît : "Export terminé". Cliquer sur OK.

Plusieurs fichiers sont alors créés : TRFECRAN.MB, TRFECRAN.db.

Importation

A partir de l'application d'arrivée, demander Fichier | Ecrans de saisie
puis éventuellement, Fichier | Importer écran(s)
Le fichier TRFECRAN est proposé dans le répertoire "travail" du poste, cliquer dans Ouvrir (si les fichiers ont été stockés à un autre endroit, il faut ouvrir le répertoire les contenant avant de cliquer sur Ouvrir). Le contenu du fichier apparaît dans une fenêtre vous permettant de sélectionner l'écran (ou les écrans). Cocher "Avec" ou "Sans" remplacement des pré-existantes.
Lorsque cette sélection est effectuée, cliquer sur un des boutons Importer les sélectionnées ou Tout Importer selon d'option choisie.

Lorsque l'importation est terminée, un message apparaît "Import terminé". Cliquer sur OK
