    (le fichier créé prend en compte tous les tarifs de tous les produits).


Le nom du fichier d'export est TRFTAR, cliquer dans (les fichiers d'export sont stockés implicitement dans le répertoire "travail" du poste mais il est possible d'indiquer un autre chemin de réception).
Lorsque le fichier est constitué, un message apparaît : "Export terminé". Cliquer sur
Plusieurs fichiers sont alors créés : TRFTAR.PX, TRFTAR.MB, TRFTAR.db.

Importation

A partir de l'application d'arrivée, demander
Le fichier TRFTAR est proposé dans le répertoire "travail" du poste, cliquer dans (si les fichiers ont été stockés à un autre endroit, il faut ouvrir le répertoire les contenant avant de cliquer sur ).
Le contenu du fichier apparaît dans une fenêtre vous permettant de sélectionner le (ou les) tarif(s) à importer. Cocher "Avec" ou "Sans" remplacement des pré-existants.
Lorsque cette sélection est effectuée, cliquer sur un des boutons ou selon l'option choisie.
Lorsque l'importation est terminée, un message apparaît "Import terminé". Cliquer sur
