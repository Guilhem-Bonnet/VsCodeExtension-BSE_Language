Les lettres

Les opérations suivantes s'effectuent à partir de la gestion de textes


Exportation :

A partir de l'application d'origine, il existe deux façons de procéder :

  • Il est possible d'exporter toutes les lettres en demandant :


  • Ouvrir le classeur contenant la (ou les) lettre(s) devant être exportée(s).
    Sélectionner la lettre à exporter en cliquant dessus. (Pour sélectionner plusieurs lettres, utiliser les fonctions de sélection de WINDOWS : touche Ctrl + clic)

Le nom du fichier d'export est TRFLET, cliquer dans (les fichiers d'export sont stockés implicitement dans le répertoire "travail" du poste mais il est possible d'indiquer un autre chemin de réception). Lorsque le fichier est constitué, un message apparaît : "Export terminé". Cliquer sur

Plusieurs fichiers sont alors créés : TRFLET.db, TRFLET.MB, TRFLET.PX et autant de .doc que de lettre(s) exportée(s).


Importation :

A partir de l'application d'arrivée, demander :

Le fichier TRFLET est proposé dans le répertoire "travail" du poste, cliquer dans (si les fichiers ont été stockés à un autre endroit, il faut ouvrir le répertoire les contenant avant de cliquer sur ).

Le contenu du fichier apparaît dans une fenêtre vous permettant de sélectionner la (ou les) lettre(s) à importer.
Cocher "Avec" ou "Sans" remplacement des préexistantes.
Lorsque cette sélection est effectuée, cliquer à votre choix sur un des boutons :

OU
