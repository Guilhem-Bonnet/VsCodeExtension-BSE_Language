Exportation

A partir de l'application d'origine, ouvrir la table à exporter.


Le nom du fichier d'export est TRFTAB, cliquer dans (les fichiers d'export sont stockés implicitement dans le répertoire "travail" du poste mais il est possible d'indiquer un autre chemin de réception).
Lorsque le fichier est constitué, un message apparaît : "Export terminé". Cliquer sur
Plusieurs fichiers sont alors créés : TRFTAB.PX, TRFTAB.db.
