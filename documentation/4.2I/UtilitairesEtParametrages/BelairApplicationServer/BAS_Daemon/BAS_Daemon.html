BAS Daemon

L'application BAS Daemon est un service Windows qui gère le ou les instances de BAS (1 instance par solution).

Installation et démarrage


Installer le service :

BasDaemon /install

Désinstaller le service :

BasDaemon /unistall


Démarrez le service depuis le panneau de configuration (voir "services").

Configuration

Le fichier de configuration BasDaemon.conf doit être placé dans le même répertoire que l'exécutable.
Ce fichier décrit la ou les solutions (applications) servies par les instances de BAS.

[Apps]
App1=D:\belair\APPDEI\appdei.conf
App2=E:\APPTTT\appttt.conf

IMPORTANT :

Les ports de BAS définis dans le fichier de configuration de chaque application doivent tous être différents.
I.e. 8080 pour App1, 8081 pour App2 etc.

Démarrage

  • Depuis la liste des services, trouvez le service BasDaemonService puis démarrez.
  • Depuis la ligne de commande : 

    net start BasDaemonService

net stop BasDaemonService

net restart BasDaemonService

Toutes ces opérations de configuration et de paramétrage peuvent être effectuées via la console d'adminstration (BasAdmin)
