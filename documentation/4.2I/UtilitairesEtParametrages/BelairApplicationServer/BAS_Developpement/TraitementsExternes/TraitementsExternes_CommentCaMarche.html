Traitements externes : comment ça marche ?


Schéma de fonctionnement


Le traitement est effectué par le programme BasExtPrs.exe, qui se trouve dans le répertoire BelAir (installation "serveur" ou "mono-poste" est nécessaire).

Les détails de fonctionnement

  • BAS doit être en exploitation, le fichier de configuration doit avoir ces paramètres.
  • Le service de programmation de temps (scheduler) est disponible.
  • Le traitement est démarré par scheduler, 1 ou plusieurs fois.

Surveiller les traitements

Démarrer BelAir.
Vérifier le cours dans l'écran de statut des traitements (Belair : menu Outils - Liste de traitements en cours).
