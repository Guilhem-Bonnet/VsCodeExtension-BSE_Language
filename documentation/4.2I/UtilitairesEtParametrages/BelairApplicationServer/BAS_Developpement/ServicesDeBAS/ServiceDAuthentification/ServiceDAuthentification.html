Service d'authentification BAS

Le service IBasAuthService a pour buts :

  • Tester la methode Echo ;
  • Se connecter et ouvrir la session ;
  • Fermer la session ;
  • Récupérer les informations de serveurs et d'applications.

Voir aussi La connexion et l'accès au services BAS.


API d'accès

CheckSession()

Vérifie si la session est active et, si non, génère une exception avec le code spécifique (voir en bas de page).

void CheckSession(BasSecurityContext sc)

Paramètres :

  • sc : l'objet contexte de sécurité obtenu par l'appel de "OpenSession()".

CloseSession()

Ferme la session, préalablement ouverte par OpenSession().

void CloseSession(BasSecurityContext sc)

Paramètres :

  • sc : l'objet contexte de sécurité.

Echo()

Renvoie le message.

string Echo(string message)

Paramètres :

  • message : le message à retourner.

Résultat :

  • Le message retourné.

GetAppInfo()

Renvoie les informations sur l'application.

BasAppInfo GetAppInfo(BasSecurityContext sc)

Paramètres :

  • message : le message à retourner.

Résultat :

  • L'objet "BasAppInfo" dont la structure est la suivante :
    • string ConfFileName : le chemin et le nom de ficher de configuration ;
    • string Name : le nom de l'application, i.e. "DEI" pour celle de développement ;
    • string Title : l'intitulé de l'application.

GetSysInfo()

Renvoie les informations système.

BasSysInfo GetSysInfo()

Résultat :

  • L'objet "BasSysInfo" dont la structure est la suivante :
    • string ApplicationName : le nom de l'application, i.e. "DEI" pour celle de développement ;
    • string OSVersionName : la version du système d'exploitation, i.e. "Windows 10 64 bits ..." ;
    • BasVersionInfo VersionInfo : l'objet contenant l'information de la version BAS :
      • int Major : un numéro majeur ;
      • int Minor : un numéro mineur ;
      • char Release : une lettre de rélease entre 'A' et 'Z' ;
      • int Revision : un numéro de la révision ;
      • string Version : une chaîne version complète i.e. "4.1B.123456".

OpenSession()

Ouvre une session BAS pour un utilisateur spécifié, ou bien génère une exception avec le code spécifique (voir en bas de page).
La session ouverte sera fermée dans 20 minutes en cas d'inactivité.

BasSecurityContext OpenSession(string logon, string password)

Paramètres :

  • logon : le nom de connexion.
  • password : le mot de passe.

Résultat :

  • L'objet "BasSecurityContext" à conserver et à utiliser pour tous les appels de BAS.
