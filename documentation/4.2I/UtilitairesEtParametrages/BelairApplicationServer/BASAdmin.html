BASAdmin : console d'administration du BAS

BasAdmin est la console graphique d'administration du BAS

Cet utilitaire permet d'effectuer les opérations suivantes

  • Installation / désinstallation du service Bas (BasDaemon)
  • Démarrage / Arrêt / Redémarrage du service
  • Installation d'une base Belair
  • Désinstallation d'une base Belair
  • Installation / désinstallation du service complémentaire BaiimpPool (BaiimpPool)
  • Démarrage / Arrêt / Redémarrage du service
  • Installation d'une base Belair
  • Désinstallation d'une base Belair
  • Installation / désinstallation du service complémentaire BaiGelDesAvoirs
  • Configuration du service BaiGelDesAvoirs
Installation et démarrage

BasAdmin.exe est installé dans le répertoire d'installation des programmes Belair (uniquement en mode serveur ou mono-poste)

BasAdmin.exe doit être exécuté sur la machine hébergeant le serveur BAS, en mode administrateur

Fonctions d'administration BasDaemon

L'onglet BasDeamon présente les fonctions de configuration du service BasDeamon.


Sélectionnez le fichier de paramétrage du service Bas (fichier BasDaemon.conf qui se trouve normalement dans le même répertoire).
Vous pouvez ensuite installer, désinstaller le service; le démarrer, l'arrêter et le relancer.

Installation d'une base Belair


L'onglet Installer une base permet d'ajouter une base Belair dans la liste des applications servies par BAS sur la machine

  • Sélectionnez le fichier de configuration de la base Belair (fichier Appxxx.conf)
  • Choisissez le port associé à cette base
    Le même port ne peut pas être associé à deux applications Belair
  • Cliquez sur Enregistrer
Désinstallation d'une base Belair


L'onglet Desinstaller une base permet d'enlever une base Belair de la liste des applications servies par BAS sur la machine

  • Sélectionnez dans la liste l'application à désinstaller
  • Cliquez sur Supprimer
Démarrer / Arrêter / Relancer le service

3 boutons permettent de démarrer, d'arrêter et de relancer le service Basdeamon

Fonctions d'administration BaiimpPool

L'onglet BaiimpPool présente les fonctions de configuration du service BaiimpPool.


Sélectionnez le fichier de paramétrage du service BaiimpPool (fichier baiimpService.conf qui se trouve normalement dans le même répertoire).
Vous pouvez ensuite installer, désinstaller le service; le démarrer, l'arrêter et le relancer.

Installation d'une base Belair


L'onglet Installer une base permet d'ajouter une base Belair dans la liste des applications servies par BaiimpPool sur la machine

  • Sélectionnez le fichier de configuration de la base Belair (fichier Appxxx.conf)
  • Saisissez les paramètres de l'utilisateur Belair utilisé (nom et mot de passe)
  • Saisissez le nombre d'instance de l'automate (Baiimp.exe) à activer sur cette base
  • Cliquez sur Enregistrer
Désinstallation d'une base Belair


L'onglet Desinstaller une base permet d'enlever une base Belair de la liste des applications servies par BaiimpPool sur la machine

  • Sélectionnez dans la liste l'application à désinstaller
  • Cliquez sur Supprimer
Démarrer / Arrêter / Relancer le service

3 boutons permettent de démarrer, d'arrêter et de relancer le service BaiimpPool

Fonctions d'administration BaiGelDesAvoirs

L'onglet BaiGelDesAvoirs présente les fonctions de configuration du service BaiGelDesAvoirs.


Onglet Configuration
L'algorithme de comparaison entre une entité belair (personne morale, personne physique) et les entrées du registre du gel des avoirs se base sur un poids, affecté à chaque critère pouvant être pris en compte.
Le poids total (le niveau de pertinence) d'une recherche est donc un résultat mathématique combinant les poids des critères correspondant entre l'entité dans Belair et l'entrée dans le registre.

La modification de ces poids peut donc changer le résultat des recherches préalablement faites.

Chacun des onglets ci-dessous permet de modifier le poids de chaque critère pour chaque entité.





Le dernier onglet fixe le seuil minimum d'alerte.
Le critère principal de la comparaison se fait sur le nom indiqué dans le registre, comparé aux différentes informations compatibles dans Belair (nom, raison sociale, alias...).
Un algorithme de similitarité orthographique donne un coefficient (entre 0 et 1).
Si le résultat de l'algorithme est inférieur au seuil mimimal d'alerte paramétré, l'entrée de registre est dans l'évaluation de la pertinence de la comparaison.

Enregistrement de la configuration du service BaiGelDesAvoirs:


Le service doit être redémarré pour que les modifications soient prises en compte.
