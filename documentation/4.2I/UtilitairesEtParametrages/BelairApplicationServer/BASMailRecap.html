Envoi de mail récapitulatif de fin de traitement BAS

Le Service BAS doit être actif

Depuis l'outil BasAdmin.exe

  • Depuis l'onglet Administration BasDaemon : Pointez vers le fichier BasDaemon.conf
  • Depuis Installer une Base : Pointez vers le fichier de configuration Belair, Indiquez le nom du serveur, Indiquez le port de communication
  • Depuis l'onglet Administration BasDaemon : Cliquez sur Installation de BasDaemon puis Démarrer le service

Depuis les paramètres d'application, onglet Paramètres de messagerie

Au niveau du paramètrage SMTP, cliquez sur Configurer

  • Nom : SMTP
  • Email : Email du compte SMTP
  • Répondre à : Email du compte SMTP
  • Cochez Utiliser la connexion personnalidée
  • Serveur : Indiquez le Serveur SMTP
  • Port : Indiquez le port
  • Sécurité : Choisir le type de sécurité
  • Méthode d'authentification : Choisir mot de passe
  • Login + MDP : Indiquez vos paramètres de compte SMTP

Testez les paramètres pour obtenir le message : Connexion au serveur OK

Catalogue BJO à renseigner

Accessible depuis le menu Outils / Catalogues : Sélection BJO - Paramètrage Email Récapitulatif de fin de traitement BAS

  • Ajouter le correspondant avec le +
  • Sous Catalogue : Indiquez JOB
  • Classe du job : Indiquez le nom du job
  • Cochez Utiliser la connexion personnalidée

En pratique

Depuis Outils / Paramètres / Paramètres de traitement BAS :

Permet de choisir une action depuis la liste déroulante (Nom de la tache)

Exemple : TRBEPROCESSING

Renseigner un nom au niveau du Libellé de la tache

Saisir les valeurs utilisées

ReceiveLogByEmail doit être à TRUE


Enregistrer le traitement au format XML (exemple TRBEPROCESSING.xml) permet l'automatisation par une ligne de commande.

Depuis Powershell : changez le dossier courant vers le repertoir Belair4

Lancez la commande : .\BasClientCMD.exe /IC:/Belair4/APPXXX/TRBEPROCESSING.xml

Lance le traitement


Depuis Bureautique / Paramétrage BSL :

Permet de choisir une action BSE

Exemple : SYS_ECHO

Il est possible de lancer directement l'action avec le bouton LANCER : il faudra alors renseigner les valeurs puis valider par le bouton Lancer

Lance le traitement

L'email est bien arrivé dans la boite de réception du destinataire


Enregistrez l'action avec le bouton Extraction (exemple : SYS_ECHO.XML) afin d'automatiser le traitement par une ligne de commande.

Depuis Powershell : changez le dossier courant vers le repertoir Belair4

Lancez la commande : .\BasClientCMD.exe /IC:/Belair4/APPXXX/SYS_ECHO.xml

Lance le traitement

L'email est bien arrivé dans la boite de réception du destinataire
