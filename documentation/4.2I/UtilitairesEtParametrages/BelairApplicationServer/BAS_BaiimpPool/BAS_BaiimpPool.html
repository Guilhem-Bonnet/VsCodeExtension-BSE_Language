BaiimpPool

L'application BaiimpPool est un service Windows qui gère le "pool" des instances de l'automate (Baiimp) pouvant gérer des appels synchrones dans les scripts BSE via les fonctions Automate(..) et AutomatebyBsi(..).

Configuration

Le fichier de configuration BaiImpService.conf doit être placé dans le même répertoire que l'exécutable.
Ce fichier décrit la ou les solutions (applications) servies par les instances de BaiimpPool.

[Apps]
[App1]
MaxProcessCount=5
Base=d:\Belair42\AppDei\AppDei.conf
User=SUPERVISEUR
Password=""
[App2]
MaxProcessCount=2
Base=d:\Belair42\AppBis\AppBis.mssql.conf
User=SUPERVISEUR
Password=""

Note : il est possible d'avoir plusieurs applications.

Installation et démarrage


Installer le service :

….\ BaiimpPool.exe /install

Désinstaller le service :

….\ BaiimpPool.exe /unistall


Démarrez le service depuis le panneau de configuration (voir "services").

lancer le service

Première méthode

  • Clic droit dans la barre Windows en bas
  • Clic sur gestionnaire des taches
  • Aller dans l'onglet tout a droite (Services)
  • Choisir le service baiimppoolService
  • Cliquer droit dessus est sélectionner « Démarrer »

Deuxième méthode

  • Appuyer sur les touches WIN + R
  • Taper Cmd
  • Sc start baiimppoolService (attention il faut être administrateur)

Toutes ces opérations de configuration et de paramétrage peuvent être effectuées via la console d'adminstration (BasAdmin)
