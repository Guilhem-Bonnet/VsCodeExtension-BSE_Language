Création d'une boucle

Permet notamment d'additionner des valeurs d'un champ répondant à des conditions identiques définies dans le SQL

SQL:NEW(TOTO) // Création du SQL
SQL:OPEN(TOTO) // Ouverture du SQL
SI RESULT = 0 ;>FINTAB // S'il n'y a plus de valeur répondant aux conditions du SQL, on va à la ligne contenant le label FINTAB, sinon on continue sur la ligne suivante
:BOUCLE // Déclaration du label BOUCLE
SQL:NEXT(TOTO) // On passe à la valeur suivante trouvée par le SQL et on reprend le traitement pour cette valeur
SI RESULT > 0 ;>BOUCLE // Si cette valeur existe, on retourne à la ligne comportant le label BOUCLE
:FINTAB // Déclaration du label FINTAB
SQL:CLOSE(TOTO) // Arrêt de la requête
SQL:FREE(TOTO) // Suppression de la requête

On introduit entre ces lignes, les formules de calcul que l'on souhaite appliquer aux valeurs sélectionnées par le SQL en fonction du résultat que l'on souhaite obtenir. Ces nouvelles lignes seront intégrées dans le SQL mais ne feront pas partie du SQL (donc non précédées d'un *) et sont constituées de la même façon que dans une formule de calcul classique (se reporter à la documentation correspondante).
