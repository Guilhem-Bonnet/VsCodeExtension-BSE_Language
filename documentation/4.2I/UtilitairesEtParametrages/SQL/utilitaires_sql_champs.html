Champs et paramètres

Comment lire un champ contenu dans une requête ?

Pour les tables de BELAIR

?table(nomchamp)

?TIERS(b_numtiers)

Pour les requêtes

?SQLnom(nomchamp)

?SQLTOTO(b_numtiers)

Comment passer des paramètres ?

En plaçant les paramètres dans des variables intermédiaires et en mettant :lx dans la requête.
