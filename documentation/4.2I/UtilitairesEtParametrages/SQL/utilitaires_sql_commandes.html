Commandes : lignes précédées de *

SELECT

Sélection des champs utilisés dans la requête, les champs doivent être séparés par une virgule.

Remarque : Toujours associé à la commande

FROM [table]

Sélection de la table dont sont issus les champs, le nom de la table doit être indiqué entre crochets [table]

* SELECT b_numtiers, b_rsociale FROM [TIERS]
Sélectionne les champs issus de la table TIERS donnant le numéro de tiers et le nom-prénom

AS

Permet de renommer un champ.

* SELECT b_numtiers AS numero FROM [TIERS]
Les valeurs sélectionnées sont stockées sous le nom 'numero'

UPDATE [table]

Sélection de la table dans laquelle on souhaite modifier des valeurs

Remarque : Toujours associé à la commande

SET

Permet d'associer une nouvelle valeur à un champ

* UPDATE [POLI] * SET b_cietaxes = 1
Attribue la valeur 1 au champ b_cietaxes de la table POLI

WHERE

Permet d'imposer une condition à la sélection

* SELECT b_numtiers FROM [TIERS] * WHERE b_rsociale = "MARTIN Pierre"
Sélectionne le numéro de tiers de MARTIN Pierre, ou les numéros, si il existe plusieurs MARTIN Pierre saisis en portefeuille

AND

Permet de joindre plusieurs condition WHERE

* SELECT b_numtiers, b_rsociale FROM [TIERS] * WHERE b_rsociale = "MARTIN Pierre" AND b_codp = "78000"

DISTINCT

Permet de ne pas sélectionner des doublons

* SELECT DISTINCT b_codp, b_ville FROM [TIERS]

ORDER BY

Permet de classer selon un champ par ordre croissant de la valeur de ce champ (ordre alphabétique si champ alpha)

* SELECT DISTINCT b_codp, b_ville FROM [TIERS] * ORDER BY b_codp
