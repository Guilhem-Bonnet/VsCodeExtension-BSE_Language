Qu'est-ce que SQL?

Le SQL est un langage informatique d'interrogation de bases de données. Il permet de manipuler, dans BELAIR, des requêtes 'autonomes', afin de palier à l'absence de certaines fonctions dans les formules de calcul, pour l'import de données (automate), en gestion de texte, en requête et pour les procédures comptables. Le SQL étant un langage informatique universel, vous trouverez, dans le commerce, des livres traitant de ce sujet. Toutefois, chaque base de donnée comportent ses spécificités vis-à-vis des commandes SQL.
