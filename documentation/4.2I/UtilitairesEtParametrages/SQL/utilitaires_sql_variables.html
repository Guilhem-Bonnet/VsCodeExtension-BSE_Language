Lignes variables *$=

Il est possible de stocker dans une variable alphabétique une ligne de commande SQL qui sera interprétée lors de son exécution.

IA$:=""
SI ?RDPP("B_NOM")="MARTIN";IA$:="Where b_numtiers=1000"

SQL:NEW(TEST)
*SELECT * FROM [RIB]
*$=IA
SQL:OPEN(TEST)

Suivant la valeur contenue dans ?RDPP("B_NOM"), le SQL prendra l'une des 2 formes suivantes lors de son éxécution.

SELECT * FROM [RIB]

ou

SELECT * FROM [RIB] WHERE B_NUMTIERS=1000
