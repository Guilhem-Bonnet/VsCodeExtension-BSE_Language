Fonctions de gestion de fichiers

BASE64DECODE(chaine)

Décode et retourne une chaine codée en base64.

BASE64ENCODE(chaine)

retourne une chaine encodée en base64.

OPEN("command")

Cette fonction permet d'ouvrir un fichier (programme, document, ...)

  • Elle réplique le comportement qu'a Windows, lorsqu'on fait Win + R
    Exemple :     
    Open("https://docs.microsoft.com/en-us/windows/win32/api/shellapi/nf-shellapi-shellexecutew")
ou
Open("C:\Users\dimitri\Documents\BelAir\Belair.log")

FICHIER_COPIER(source,destination)

Copie le fichier "source" vers le chemin "destination".
Retour: 1 si l'opération s'est correctement déroulée et 0 en cas d'erreur.

FICHIER_DEPLACER(source,destination)

Déplace le fichier "source" vers "destination".
Retour: 1 si l'opération s'est correctement déroulée et 0 en cas d'erreur.

FICHER_EXISTE(fichier)

Renvoie 0 si le fichier n'existe pas ou 1 s'il existe.

FICHIER_IMPRIMER(fichier)

Imprime "fichier".

FICHIER_RENOMMER(source,nouveau nom)

Renomme "fichier" avec le "nouveau nom".
Retour: 1 si l'opération s'est correctement déroulée et 0 en cas d'erreur.

FICHIER_SUPPRIMER(fichier)

Supprime "fichier".
Retour: 1 si l'opération s'est correctement déroulée et 0 en cas d'erreur.

Pour toutes ces fonctions, les chemins relatifs Belair ([APPLIC], [SCANNER,] etc...) sont automatiquement trnasformés en leur équivalent Windows.
Attention :
les fonctions FICHIER_RENOMMER et FICHIER_DEPLACER ne marchent que sur le même disque.
Dans le cas contraire, utiliser FICHIER_COPIER puis FICHIER_SUPPRIMER.

LITFICHIER(fichier)

Met le contenu du fichier dans une chaine alphabétique.

Exemple d'utilisation :
IA$ := litfichier("c:\tmp\a.txt")
ID := nblignes(IA)
IB := 0
IC := 0
IE := 0
:loop1
si IC >= ID;>cont1
IB$ := extline(IA, IC)
IB := IB
IC := IC + 1
IE := IE + IB
>loop1
:cont1
print(IE)

LITFICHIER64(fichier)

Code en BASE64 et retourne le contenu du fichier fichier.

NBLIGNES(chaine)

Retourne le nombre de lignes contenues dans chaine.

EXTLINE(chaine, index)

Retourne la ligne index contenue dans chaine.

PATHEXPAND(filename)

Transforme un chemin d'accès 'Belair' en son équivalent Windows.

Paramètre :

filename : chemin d'accès vers le répertoire ou le fichier

Exemples :

PATHEXPAND("[ARCHIVES]\201811\") => D:\belair42\APPDEI\ARCHIVES\201811\
PATHEXPAND("[ARCHIVES]\201901\") => D:\belair42\APPDEI\ARCHIVES\201901\
PATHEXPAND("[SCANNER]\201903") => D:\belair42\APPDEI\SCANNER\201903
PATHEXPAND("[SCANNER]\201903\doc1234578.PDF") => D:\belair42\APPDEI\SCANNER\201903\doc1234578.PDF
