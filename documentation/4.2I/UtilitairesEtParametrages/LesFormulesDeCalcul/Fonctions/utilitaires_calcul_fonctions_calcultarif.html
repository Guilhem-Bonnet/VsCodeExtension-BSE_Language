Calcul d'une prime de base

CALCULTARIF([codegar])

Cette fonction de calcul numérique permet de recalculer la prime de base d'un risque assuré


Paramètre d'entrée :

  • codegar : code de la garantie (optionnel)
Si codegar est vide, toutes les garanties sont recalculées: la fonction rend la prime TOTALE
Sinon, le (re)calcul ne se fait que sur la garantie codegar et la fonction rend le montant de la prime totale de cette garantie

Exemple

IT := CALCULTARIF() // recalcul de toute la prime de base
IT := CALCULTARIF("RC") // recalcul uniquement de la garantie RC
