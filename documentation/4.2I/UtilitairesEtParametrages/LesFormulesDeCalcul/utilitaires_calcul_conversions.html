Conversions

Conversion d'une valeur numérique en valeur alphabétique

$(valeur, "référence) incomplet

Paramètres :

valeur : valeur de type numérique

Exemple :

$(100 + 10) donne "110"

Conversion d'une valeur alphabétique en valeur numérique

#("valeur") incomplet

Cette instruction permet par exemple d'insérer dans un calcul numérique la valeur d'une information déclarée en type alphabétique (ou majuscule) mais contenant une valeur numérique.

Paramètres :

valeur : Valeur de type alphabétique
