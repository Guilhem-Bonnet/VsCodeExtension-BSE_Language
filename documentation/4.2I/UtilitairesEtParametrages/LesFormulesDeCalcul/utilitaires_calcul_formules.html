Appel d'une formule liée à un champ

[formule]

Ces formules sont des formules de calcul comme les autres, avec un nom particulier, qui peuvent donc être appelées par n'importe quelle autre instruction.
Valeur par défaut : $code:champ
 Test si saisie : #code:champ

Exemple :

IA$ := {"$RISA:B_APPEL") // valeur par défaut du champ RISA("B_appel")
