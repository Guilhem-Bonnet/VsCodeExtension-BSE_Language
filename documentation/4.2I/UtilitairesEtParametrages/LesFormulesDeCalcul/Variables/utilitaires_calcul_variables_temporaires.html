Variables temporaires


Les variables temporaires permettent de stocker des calculs intermédiaires, dans le but de simplifier les formules de calculs

ICode type := valeur incomplet

Affectation d'une valeur à une variable temporaire.

Paramètres :

Code : Lettre 'A' à 'Z', c'est l'identifiant de la variable temporaire (vous disposez donc en permanence de 26 variables)

Type : Type de variable

  • $ : variable alphabétique
  • N : variable numérique (non obligatoire)
  • X : variable booléenne
  • D : variable 'date'

L'absence d'indicateur de 'Type' fera que la variable sera considérée par défaut comme numérique

Valeur : Valeur à affecter (doit être du même type que la variable)

Exemple :

IAD:=?CONT("b_fincours") - 30 // date = 30 jours avant la fin de période en cours

Attention : Une variable temporaire garde sa valeur, tant qu'une autre valeur ne lui est pas affectée. Ceci permet notamment de passer une valeur stockée dans une variable temporaire d'une formule de calcul à une autre.

Utilisation de la valeur d'une variable temporaire.

Paramètres :

Code : code la variable à récupérer (lettre 'A' à 'Z')

Le 'Type' de variable n'est plus à préciser car elle contient maintenant une valeur de format défini.

Exemple :

>IAD:=?CONT("b_fincours") - 30
SI IA // la formule doit être ici de format alphabétique car il s'agit du format du résultat ="alerte
Attention : Les variables sont GLOBALES et accessibles par tous les traitements déclenchés par script automate ou formule de calcul.
Les variables ne sont PAS REINITIALISEES à chaque utilisation dans un script ou une formule de calcul.
