Variables contextuelles

INumero

Les variables contextuelles fonctionnent sur le même principe que les variables temporaires mais leur utilisation est limitée à certaines fonctionnalités du progiciel (en général dans les procédures comptables, lors d'opérations spécifiques tels les envois de prélèvement automatiques).
La documentation de ces fonctions indique alors le rôle de chacune des variables disponibles.

Il est impossible d'affecter une valeur à une variable contextuelle.

Paramètres :

numero : numéro de la variable contextuelle à récupérer (chiffre de '1' à '9')
