Variables liées aux encaissements

  • SRA : Somme restant à affecter

  • ENC_MON : Montant encaissé / affecté

  • ENC_REF : Référence du paiement

  • ENC_DAT : Date du paiement

  • ENC_EFF : Date d'effet d'encaissement

  • ENC_DEV : Devise de l'encaissement en cours

  • ENC_TYP : Type d'encaissement utilisé

  • ENC_TIE : Tiers Payeur

  • ENC_ENTITE : Permet de récupérer l'entité comptable en cours, lors d'une opération comptable

Exemple :

DEVSOM(ENC_MON,ENC_DEV,"EUR") traduira le montant de l'encaissement effectué en euros.
