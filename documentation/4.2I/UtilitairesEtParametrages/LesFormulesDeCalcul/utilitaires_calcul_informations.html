Information d'une table :

?TABLE("information") incomplet

TABLE : Code de la table BELAIR contenant l'information (exemple TIERS, CONT, etc...)

Paramètres :

information : Valeur alphabétique donnant le nom du champ

Exemple :

?TIERS("b_rsociale") champ b_rsociale de la table TIERS

?("information") incomplet

Permet d'accéder à un champ de la table en cours (la table de l'information que l'on calcule ou que l'on vérifie)

Paramètres :

information : Valeur alphabétique donnant le nom du champ (constante entre " ", référence à un calcul alphabétique, etc...)

CHAMP et CHAMPOLD (accessible uniquement dans les formules de vérification de saisie) incomplet

CHAMP : Valeur du champ venant d'être saisie
CHAMPOLD : Ancienne valeur du champ avant sa nouvelle saisie

Manipulation d'une information de table par les variables incomplet

Le nom de la table est remplacé par une variable ;

Exemple :
IB$ := "B_RISQUE"
?RVEH(IB) := IZ // équivalent de ?RVEH("B_RISQUE")

Le nom du champ est remplacé par une variable ;

Exemple :

IA$ := "RVEH"
?$IA("B_RISQUE") := IZ // équivalent de ?RVEH("B_RISQUE")

Le nom du champ et celui de la table sont remplacés par des variables ;

Exemple :

IA$ := "RVEH"
IB$ := "B_RISQUE"
?$IA(IB) := IZ // équivalent de ?RVEH("B_RISQUE")
