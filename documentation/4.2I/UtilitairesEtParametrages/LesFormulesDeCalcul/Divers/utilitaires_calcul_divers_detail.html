Détail d'une personne physique

DETAILDPP("?","?")

Retourne une chaîne avec les champs passés en 2ième paramètre.

Paramètres permettant de prendre en compte plusieurs personnes sur un contrat

FILEFIRST("RDPP")

Se positionne sur le premier enregistrement de la table RDPP (risque personnes physiques).

FILENEXT("RDPP")

Se positionne sur l'enregistrement suivant.

Exemple :

La prime doit être de 10,00 euros pour chaque assuré ayant moins de 25 ans et 15,00 euros au delà.
La prime totale à percevoir doit représenter le cumul des montants.

IA:=0 // total
SI FILEFIRST("RDPP")="VRAI";>SUIVANT;>FIN // lecture du 1er ; si aucun = fin
:SUIVANT
SI ?RDPP("b_age")// cumul dans IA
SI FILENEXT("RDPP")="VRAI";>SUIVANT;>FIN // suivant ; si plus aucun = fin
:FIN
=IA // résultat final
