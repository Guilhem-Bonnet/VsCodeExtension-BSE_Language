Formules de calcul - Interactions avec l'utilisateur

Les fonctions de cette page permettent d'afficher un message (temporaire ou à confirmer), de saisir une valeur, de choisir un valeur dans un menu ou de choisir une action en réponse à un message.

ALERTE("message")

Affiche le message pendant 3.5 secondes.

ALERTE("message", "durée")

Affiche le message pendant xx secondes correspondant à "durée".

SAISIE("question")

Affiche le message en attendant une réponse de l'utilisateur devant être stockée dans une variable et pouvant être utilisée dans une formule de calcul.

Exemple :

IA$:= SAISIE("Voulez-vous continuer (O/N)")
SI IA = "O";>CONTINU
SI IA = "N";=""

MENU("titre","texte de saisie","codes","valeurs")

Création d'un menu déroulant permettant de récupérer un "code" devant être stockée dans une variable.

Paramètres :

"titre" : titre de la fenêtre qui s'ouvre pour la saisie.

"texte" : intitulé de la saisie demandée.

"codes" : les différents codes doivent être séparés par le signe ; (point-virgule).
C'est le code qui sera stocké dans la variable.

"valeurs" : les différentes valeurs doivent être séparées par le signe ; (point-virgule).
Ce sont les valeurs qui seront proposées à l'affichage dans le menu déroulant.


A chaque "valeur" doit correspondre un "code" donné.

Si on ajoute un point-virgule au début ou à la fin, cela permet d'avoir un choix de réponde "à vide".

Exemple de formule :

IA$:= MENU("connexe","Deuxième prénom ?","A;B;C;","Jean;René;Pierre;") =IA

CATALOGUE("catalogue","sous-catalogue")

Déclenche l'ouverture du catalogue de risques indiqué en paramètres.

SHOWMESSAGE(message)

Affiche un message à l'utilisateur dans une simple boite de dialogue (bouton "OK")

Paramètres :

message : Le message à afficher

Exemple

ShowMessage("Voici un simple message")

MESSAGEDLG(message,boutons,TypeDialogue)

Affiche un message à l'utilisateur dans une boite de dialogue et demande un choix (liste de boutons).
le nom du bouton sélectionné est rendu par la formule (en MAJUSCULE).

Paramètres :

message : Le message à afficher
boutons : Liste des boutons, séparés par un ";" (voir liste ci-dessous)
TypeDialogue : Type de la boite de dialogue; choix parmi "Information" / "Confirmation" / "Warning" / "Error" (optionnel, par défaut "Information")
L'icone correspondant au type de dialogue sera affiché dans la fenetre.

Exemples

IA$ := MessageDlg("Voici la nouvelle fonction MessageDLG", "Close", "Information")
print("Voici la nouvelle fonction MessageDLG ?" + IA)

IA$ := MessageDlg("Voulez-vous vraiment tester MessageDLG ?", "Yes;No", "Confirmation")
print("Voulez-vous vraiment tester MessageDLG ?" + IA)

IA$ := MessageDlg("Vous avez cassé la fonction MessageDLG !!!", "Abort", "Error")
print("Vous avez cassé la fonction MessageDLG !!!" + IA)

IA$ := MessageDlg("Attention voici la nouvelle fonction MessageDLG", "OK", "Warning")
print("Attention voici la nouvelle fonction MessageDLG" + IA)

IA$ := MessageDlg("Voici tous les boutons possibles", "Yes; No; OK; Cancel; Abort; Retry; Ignore; All; NoToAll; YesToAll; Close")
print("Vous avez cliqué sur " + IA)

No Non
