Formules de calcul - Commandes de déclenchement d'une URL

URLGET(http://xxxxxxxx)

Cette fonction permet de déclencher une URL et récupere le résultat sous forme chaine.

En pratique, on peut ainsi appeler un scipt php conçu pour rendre un résultat simple, comme je le fais par exemple pour récuperer la clé d'une application


Exemple : 
IA$ := URLGET("https://httpbin.org/get")
Print(IA)

Pour des raisons de sécurité, à partir du build 21509, c'est le protocole TLS 1.2 qui est utilisé à la place du protocole TLS 1.0 par la fonction URLGET pour le https.
