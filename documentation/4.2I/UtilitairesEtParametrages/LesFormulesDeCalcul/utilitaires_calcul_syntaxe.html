Règles syntaxiques des explications de ce manuel

Mot en gras : constante ou mot réservé

{ } : paramètre optionnel

: (en italique) terme

Syntaxe générale d'une instruction

{Condition } ; action 1 ; {action 2}

Condition : instruction booléenne rendant une valeur VRAI ou FAUX

Action 1 : action exécutée si Condition = VRAI ou si Condition non renseignée

Action 2 : action exécutée si Condition = FAUX

  • à ne renseigner que si Condition est renseignée
  • si absent et si Condition = FAUX, l'exécution passe à la ligne suivante
