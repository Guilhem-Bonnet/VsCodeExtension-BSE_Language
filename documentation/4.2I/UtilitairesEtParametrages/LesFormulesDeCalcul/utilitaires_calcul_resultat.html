Rendre le résultat

=valeurincomplet

valeur doit être du même type (ou format) que la formule
Cette instruction interrompt le déroulement de la formule et rend valeur comme résultat.
Toutes les formules doivent contenir au moins une instruction de ce type ; sinon, la formule rendra un résultat vide.

Paramètres :

valeur : Valeur rendue par la formule

Exemple :

=1000
=?TIERS("b_age") + 10

Cas particulier pour les formules booléennes (condition de saisie, vérification de valeur saisie) :

Les formules booléennes doivent rendre un résultat égal à VRAI ou FAUX.
Pour rendre VRAI, mettre une instruction =VRAI
Pour rendre FAUX, mettre une instruction =FAUX

Le résultat peut également être celui d'une condition.

Exemple :

L'instruction suivante affecte comme résultat d'une formule booléenne VRAI pour toutes les personnes physiques dont le titre est "MME" :

=?TIERS("b_titre") = "MME"
