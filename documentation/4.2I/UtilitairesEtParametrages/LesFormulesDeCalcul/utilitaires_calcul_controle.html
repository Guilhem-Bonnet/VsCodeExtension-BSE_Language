Contrôle du déroulement d'une formule

Pose d'un label

:LABEL incomplet

Les labels servent à mettre des repères dans la formule pour pouvoir effectuer des débranchements (GOTO).
Une ligne portant un label ne doit contenir que ce label.

Paramètres :

LABEL : Nom du label (en majusctule)

Débranchement vers un label (GOTO)

>LABEL incomplet

Un débranchement permet d'interrompre le traitement séquentiel d'une formule ; le traitement reprend à la ligne suivant celle contenant le label indiqué.

Paramètres :

LABEL : Nom du label (en majusctule)
