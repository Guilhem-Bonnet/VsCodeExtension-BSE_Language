Calcul d'une ligne contenue dans une variable

[!instruction] et {!instruction}incomplet

Il est maintenant possible de demander le calcul d'une ligne contenue dans une variable.

[!instruction] : rend le résultat du calcul de l'instruction si celui-ci est au format numérique

{!instruction} : rend le résultat du calcul de l'instruction si celui-ci est au format alpha

Ceci est utile lorsque l'instruction n'est pas forcément connue par avance mais doit être interprétée ou constituée en dynamique.

Exemple :

IA$ := ?CONT("B_TOTANN")
SI ?CONT("B_FRAC") = "S" ; IA$ := IA + "/2"t;t;t;t;t;t;
SI ?CONT("B_FRAC") = "T" ; IA$ := IA + "/4"
SI ?CONT("B_FRAC") = "M" ; IA$ := IA + "/12"
IX := [!IA]

L'intérêt principal est en fait de pouvoir interpréter une instruction de calcul contenue quelque part dans la base (par exemple, les champs de fusion en gestion de texte)
