Création d'un écran d'extension

Les extensions sont des données supplémentaires pouvant être ajoutées aux données principales.
Une extension est donc une table 'annexe' associée à une table principale.

Types d'extension disponibles

Type Table maitre associée Code de l'extension
Personne assurée RDPP XPaaa
Quittance QUIT (bientôt...) XQaaa
Risque automobile RVEH XVaaa
Risque divers RDIV XDaaa
Risque immobilier RIMM XIaaa
Sinistres SIN XSaaa
Tiers TIERS XTaaa
Tous les contrats CONT XCaaa
Tous risques RISA XRaaa
Toutes pièces PIEC XAaaa

La création d'un écran d'extension nécessite deux phases

  • la création de la table et de ses champs module 'Modélisation des données (*)
  • la création de l'écran de saisie associé à cette table module 'Modélisation des écrans
cliquer ICI pour accéder au module Modélisation des données
(*) module optionnel permettant la création de nouvelles tables d'extension ou l'ajout de champs dans les extensions standard
