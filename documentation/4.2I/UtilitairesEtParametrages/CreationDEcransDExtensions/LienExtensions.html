Lien des extensions aux écrans du portefeuille

Lien aux tiers

Des extensions Tiers peuvent être déclarées obligatoires pour un type de tiers:
Ces extensions seront alors automatiquement attachés à chaque nouveau tiers de ce type.

Menu Gestion | Tiers | Type de Tiers

Sélectionner le type de tiers concerné puis ouvrir l'onglet "Extensions"

Sélectionner l'extension dans la partie "Disponible(s)" et cliquer sur le bouton ajouter.

Les extensions non définies en obligatoires peuvent néanmoins être ajoutées sur un tiers existant.
A partir du tiers, demander la fonction de menu Fichier | Extensions
Sélectionner l'extension dans la partie "Disponible(s)" et cliquer sur le bouton ajouter.
Les extensions sélectionnées s'afficheront en onglet supplémentaire sur le tiers.


Lien aux contrats, sinistres

Extensions obligatoires pour un produit (contrat et sinistre)

Des extensions contrat et sinistre peuvent être déclarées obligatoires pour un produit: ces extensions seront alors automatiquement attachés à chaque nouveau contrat de ce produit, et pour chaque sinistre ouvert sur ces contrats.

Menu Gestion | Contrats | Produits

Sélectionner l'onglet "Produits distribués" et le produit concerné

Menu Edition | Extensions obligatoires du produit.

Sélectionner l'extension dans la partie "Disponible(s)" et cliquer sur le bouton ajouter.
Les extensions sélectionnées s'afficheront en onglet supplémentaire sur les contrats (sinistres) concernés.

Les extensions non définies en obligatoires peuvent néanmoins être ajoutées sur un contrat ou un sinistre existant.

  • A partir du contrat, demander la fonction de menu Fichier | Extensions
  • A partir du sinistre, demander la fonction de menu Fichier | Modification extensions

Extensions d'adhésions pour un type d'écran

Des extensions peuvent être associées aux risques des contrats des produits d'un type d'écran.

Menu Gestion | Contrats | Produits

Sélectionner l'onglet "Types d'écrans" et le type d'écran concerné

La partie droite affiche les caractéristiques techniques du risque assuré et les extensions disponibles, selon les caractéristiques techniques du risque assuré (par exemple, si le type d'écran ne prévoit pas la présence d'un véhicule, les extensions 'véhicule' ne sont pas disponibles).

Il existe 4 options:

  • Non : extension non disponible (elle ne pourra pas être ajoutée manuellement)
  • Fac : extension facultative (elle pourra pas être ajoutée manuellement)
  • Obl : extension obligatoire (elle sera ajoutée automatiquement)
  • Lié : extension obligatoire liée (pas d'onglet supplémentaire, les données sont intégrées dans l'écran de saisie principal du risque
Pour changer le type d'association de l'extension, cliquer sur son nom.
Vous ne pouvez pas déclarer ici une extension liée. Cette opération se fait par le module 'modélisation des écrans'

Les extensions facultatives peuvent être ajoutées au coup par coup sur une adhésion.
demander la fonction de menu Adhésion | Extensions.
Sélectionner l'extension dans la partie "Disponible(s)" et cliquer sur le bouton ajouter.
Les extensions sélectionnées s'afficheront en onglet supplémentaire sur le détail du risque.
